# スライド作成プロジェクト

## 概要
プレゼンテーション用スライドの作成・管理を行うプロジェクトです。

## プロジェクト構造
```
planning/ - 要件定義、設計書、プロジェクト管理文書
testing/ - テスト用スクリプト、データ、ログ
implementation/ - 最終実装物、設定ファイル、デプロイメント
misc/ - 作業ファイル、参考資料、進捗保存
slide_materials/ - スライド作成に使用する画像、アイコン、テンプレートなど
completed_slides/ - 完成したスライドファイルの保存先
```

## 主要成果物
- [ ] 要件定義書 (`planning/client_design/`)
