この度は、市民向けAI講座の企画をご検討いただき、誠にありがとうございます。前回ご提案いたしました内容につきまして、日付が古いとのご指摘をいただき、大変失礼いたしました。2026年1月12日現在の最新のAI市場動向を徹底的に再調査し、講座内容を全面的にアップデートいたしましたので、改めてご提案させていただきます。

2026年、AI活用は新たなステージに突入しました。もはや単一のツールを使いこなす時代は終わり、**複数のAIを適材適所で組み合わせる「ワークフロー思考」**こそが、生産性向上の鍵となっています。本講座は、その最先端の実践知を、市民の皆様が明日から仕事で使える形に凝縮したものです。

## 2. 講座コンセプト：AIを「個」から「チーム」へ

本講座が目指すのは、参加者がAIを単なる「便利な道具」としてではなく、**思考を拡張し、面倒な作業を代行してくれる「優秀なAIチーム」として活用できるようになること**です。

| 従来のAI活用（As-Is） | 本講座が目指すAI活用（To-Be） |
|:---|:---|
| 1つのAI（例：ChatGPT）に全ての作業を依頼する | **複数のAIの得意分野を組み合わせ、ワークフローを構築する** |
| AIの回答をそのまま使う（思考停止） | AIの提案を叩き台に、人間が最終的な意思決定を行う |
| 資料作成などを「少し楽にする」レベル | **資料作成や情報収集を「ほぼ自動化」するレベル** |
| ツールの機能を知っている（知識） | **ツールを組み合わせて課題を解決できる（実践知）** |

## 3. 2026年1月最新トレンド分析と講座への反映

今回の講座設計にあたり、2026年1月現在の最新トレンドを再調査しました。特に注目すべきは以下の3点であり、これらを講座コンテンツに色濃く反映させています。

### トレンド①：ChatGPT一強時代の終焉と「Geminiエコシステム」の台頭

- **現状**: 2026年1月、ついにChatGPTのシェアは64.5%まで低下し、Googleの**Gemini**が21.5%まで急成長。この背景には、GeminiがAndroid、Google Workspace、Google検索といった巨大なエコシステムに統合され、ユーザーが意識することなく利用する「アンビエントAI」が現実のものとなったことがあります。
- **講座への反映**: 「ChatGPTだけ使っていれば良い」という考え方をアップデート。「**統合性のGemini vs 信頼性のClaude vs 創造性のChatGPT**」という2026年の新たな勢力図を提示し、タスクに応じた戦略的なAI選択能力をハンズオンで養います。

### トレンド②：AIスライド作成ツールの進化と「ワークフロー」の確立

- **現状**: **NotebookLM**の高品質なスライド生成能力は健在ですが、プロンプトによるデザインの安定化が新たな課題となっています。一方で、Googleスライドと直接連携する「**まじん式プロンプト**」や、Gemini自体の「**Canvas機能**」も進化し、用途に応じた使い分けが重要になっています。
- **講座への反映**: 3つの主要なスライド作成手法（NotebookLM、まじん式、Gemini Canvas）を全てハンズオンで体験。さらに、「NotebookLMは編集できない」という定説を覆す**Canva連携の裏技**までをセットで提供し、完全なワークフローを習得します。

### トレンド③：「AIエージェント」化の本格的な幕開け

- **現状**: 2026年は「AIワークフロー元年」とも呼ばれ、AIが自律的にタスクをこなす「AIエージェント」が実用段階に入りました。**Claude Code 2.1**に搭載された「**Skills機能**」は、定型的な開発タスクを自動化する強力なツールとして注目されています。
- **講座への反映**: 講座の後半で、こうした最先端のビジネス自動化事例を紹介。参加者が「自分の仕事もAIエージェントに任せられるかもしれない」という未来像を描けるように導きます。
