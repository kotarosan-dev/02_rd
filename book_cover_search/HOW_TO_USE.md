# 書影検索システム - 使い方ガイド

## 基本的な使い方

### 方法1: Claudeにファイルを渡して一緒に検索を進める（推奨）

1. **初期情報の整理**
   - `書影検索_情報整理シート.md` を開いて、覚えている情報を記入
   - 記入後、このファイルをClaudeに渡す

2. **質問リストの記入**
   - `書影検索_質問リスト.md` を開いて、Phase 1の基本情報を記入
   - 記入後、このファイルもClaudeに渡す

3. **Claudeとの対話**
   ```
   例: 「book_cover_search/書影検索_情報整理シート.md と 
        book_cover_search/書影検索_質問リスト.md を見て、
        書影から本を探すのを手伝ってください」
   ```

4. **検索の進行**
   - Claudeが `書影検索_ワークフロー.md` に従って検索を進める
   - 候補が見つかったら、`書影検索_フィードバック収集フォーム.md` に記入
   - フィードバックをClaudeに渡すと、次の候補を提示

5. **履歴の記録**
   - `書影検索_検索履歴管理シート.md` に検討した候補を記録
   - Claudeがこのファイルを参照して重複を避ける

### 方法2: 自分で検索を進める

1. **`書影検索_ワークフロー.md` を参照**
   - 全体の進行手順を確認

2. **各フェーズに従って作業**
   - Phase 1: 情報整理シートに記入
   - Phase 2: 参照表紙を特定して比較
   - Phase 3: 画像検索ガイドに従って検索
   - Phase 4: 候補を評価してフィードバック収集
   - Phase 5: 再検索と絞り込み

3. **検索履歴を記録**
   - 検討した候補を検索履歴管理シートに記録
   - 除外条件と重点項目を更新

## ファイルの役割

### 必須ファイル（Claudeに渡す）

1. **`書影検索_情報整理シート.md`**
   - 探している本の基本情報
   - 表紙の視覚的特徴
   - 参照表紙との比較

2. **`書影検索_質問リスト.md`**
   - 構造化された質問フォーム
   - 段階的な情報収集

3. **`書影検索_検索履歴管理シート.md`**
   - 検討済み候補の記録
   - 除外条件と重点項目

### 参照ファイル（Claudeが参照）

4. **`書影検索_ワークフロー.md`**
   - 全体の進行手順
   - Claudeがこの手順に従って検索

5. **`書影検索_画像検索ガイド.md`**
   - 画像検索の方法
   - 推奨キーワード

6. **`書影検索_候補絞り込みロジック.md`**
   - 評価方法とスコアリング
   - 候補の絞り込み方法

7. **`書影検索_フィードバック収集フォーム.md`**
   - 構造化されたフィードバックフォーム
   - 記入後、Claudeに渡す

## 推奨ワークフロー

### Step 1: 準備
1. `書影検索_情報整理シート.md` に記入
2. `書影検索_質問リスト.md` のPhase 1を記入

### Step 2: Claudeとの対話開始
```
「book_cover_search/書影検索_情報整理シート.md と
  book_cover_search/書影検索_質問リスト.md を見て、
  書影から本を探すのを手伝ってください。
  
  ワークフローに従って進めてください。」
```

### Step 3: 候補の提示
- Claudeが3-5件の候補を提示
- 各候補の一致点・相違点を明記

### Step 4: フィードバック
- `書影検索_フィードバック収集フォーム.md` に記入
- ファイルをClaudeに渡す

### Step 5: 再検索
- Claudeがフィードバックを分析
- 検索履歴管理シートを更新
- 新規候補を提示

### Step 6: 繰り返し
- Step 4-5を繰り返す
- 本が見つかるまで継続

## コツ

### 1. 一度に多くのファイルを渡さない
- 最初は情報整理シートと質問リストだけ
- 必要に応じて他のファイルも渡す

### 2. ワークフローを明示する
- 「ワークフローに従って進めてください」と伝える
- Claudeが `書影検索_ワークフロー.md` を参照

### 3. 検索履歴を共有する
- 検索履歴管理シートを定期的にClaudeに渡す
- 重複検索を避ける

### 4. フィードバックを構造化する
- フィードバック収集フォームを使用
- 曖昧な表現を避ける

## トラブルシューティング

### Claudeがファイルを読み込めない場合
- ファイルパスを正確に指定
- 相対パス: `book_cover_search/書影検索_情報整理シート.md`
- 絶対パス: `C:\Users\user\Desktop\kotarosan\03_Internal\02_R&D\book_cover_search\書影検索_情報整理シート.md`

### 候補が見つからない場合
- 検索範囲を広げる
- 視覚的特徴の条件を緩める
- 代替アプローチを検討

### 検索が散漫になる場合
- ワークフローを参照して進行を確認
- 検索履歴管理シートで重複を確認
