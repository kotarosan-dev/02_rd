# 書影検索システム - 使用方法ガイド

## 概要

書影（表紙画像）から本を探す際の精度向上と、効率的な進行方法を実現するためのツールセットです。

## 使い方

### Claudeにファイルを渡して一緒に検索を進める（推奨）

1. **初期情報の整理**
   - `書影検索_情報整理シート.md` に記入
   - `書影検索_質問リスト.md` のPhase 1を記入

2. **Claudeとの対話開始**
   ```
   「book_cover_search/書影検索_情報整理シート.md と
    book_cover_search/書影検索_質問リスト.md を見て、
    書影から本を探すのを手伝ってください。
    
    ワークフローに従って進めてください。」
   ```

3. **検索の進行**
   - Claudeが `書影検索_ワークフロー.md` に従って検索を進める
   - 候補が見つかったら、`書影検索_フィードバック収集フォーム.md` に記入
   - フィードバックをClaudeに渡すと、次の候補を提示

詳細な使い方は **`HOW_TO_USE.md`** を参照してください。

## ファイル構成

このフォルダ内のファイル構成：

### 1. 情報整理・管理
- **`書影検索_情報整理シート.md`**: 探している本の基本情報と表紙の視覚的特徴を構造化して記録
- **`書影検索_検索履歴管理シート.md`**: 検討した候補と検索履歴を記録し、重複を避ける

### 2. 情報収集
- **`書影検索_質問リスト.md`**: 構造化された質問フォームで段階的に情報を収集

### 3. 検索支援
- **`書影検索_画像検索ガイド.md`**: 画像検索ツールの使い方と推奨キーワード

### 4. 評価・絞り込み
- **`書影検索_候補絞り込みロジック.md`**: 候補を3-5件に絞り込む評価方法とスコアリング
- **`書影検索_フィードバック収集フォーム.md`**: 構造化されたフィードバック収集フォーム

### 5. ワークフロー
- **`書影検索_ワークフロー.md`**: 全体の進行手順とチェックリスト

### 6. ガイド
- **`書影検索_README.md`**: このファイル（使用方法ガイド）

## 主要な改善点

### 1. 構造化された情報収集
- 曖昧な記憶を体系的に整理
- 重要な情報と瑣末な情報を区別

### 2. 画像検索の優先活用
- テキスト検索より画像検索を優先
- Google画像検索の逆引き検索を活用

### 3. 効率的なフィードバックループ
- 3-5件に絞り込んで提示
- 構造化されたフィードバック収集

### 4. 検索履歴の記録
- 重複検索を避ける
- 効果的なキーワードを特定

### 5. 段階的絞り込み
- 優先順位を明確化
- 出版社・レーベル → 時期 → デザイナー → タイトル

## 使用例

### 例: SF小説の表紙を探す場合

1. **情報整理シートに記入**
   - 白いシルエットの人物、両腕を広げている
   - 青/黒の宇宙的背景、複数の惑星
   - 2010年以前、ハヤカワ・創元以外

2. **参照表紙を特定**
   - 「永遠の終り」が近い
   - 「タウ・ゼロ」の雰囲気が近い

3. **画像検索を実行**
   - 「永遠の終り」の表紙画像でGoogle画像検索
   - 類似画像から候補を抽出

4. **候補を評価**
   - スコアリングで3-5件に絞り込み
   - 各候補の一致点・相違点を明記

5. **フィードバックを収集**
   - 構造化されたフォームで評価
   - 検索履歴に記録

6. **再検索**
   - フィードバックを分析
   - 新規候補を検索

## 注意事項

### 1. 記憶の曖昧さ
- 記憶は時間とともに変化する可能性がある
- 複数の候補を並行して検討する

### 2. 表紙の版による違い
- 同じ書籍でも版によって表紙が異なる場合がある
- 複数の版を確認する

### 3. 検索範囲の広さ
- 候補が多すぎる場合は、出版社・レーベルで絞り込む
- 画像検索を優先して視覚的特徴で絞り込む

### 4. 時間の目安
- 1サイクル: 約2-3時間
- 全体: 3-5サイクルで特定できる可能性が高い（6-15時間）

## トラブルシューティング

### 候補が見つからない場合
1. 検索範囲を広げる（時期、出版社）
2. 視覚的特徴の条件を緩める
3. 代替アプローチを検討（タイトル・内容からの逆引き）

### 候補が多すぎる場合
1. 出版社・レーベルで絞り込む
2. 刊行時期で絞り込む
3. 画像検索で視覚的特徴で絞り込む

### 候補が少なすぎる場合
1. 検索範囲を広げる
2. 視覚的特徴の条件を緩める
3. 参照表紙を追加する
