# 新・空き家問題 -2030年に向けての大変化

**著者**: 牧野知弘
**出版**: 祥伝社新書（2025年1月）

---

## 1. 一言で言うと

2030年前後に首都圏で大量相続が発生し、空き家激増と住宅マーケット大変化という「天国と地獄」が同時に訪れる——空き家問題は地方だけでなく大都市圏こそ深刻化する時代が来る。

---

## 2. 主要なポイント

### ポイント1: 空き家900万戸の衝撃——実は首都圏が空き家天国
- 2024年時点で全国の空き家は900万戸（7軒に1軒が空き家）
- **東京都の空き家数は88万6500戸で全国ダントツ**
- 世田谷区だけで5万8850戸の空き家が存在
- 空き家の約半数（443万戸）は賃貸用空き家、個人住宅空き家は385万戸

### ポイント2: マンション空き住戸は隠れた大問題
- 共同住宅の空き住戸は502万戸（空き家全体の56%以上）
- マンション空き住戸率は10.2%に達する
- 戸建てと異なり管理費・修繕積立金が毎月発生（放置のペナルティ）
- 外国人所有マンションの空き住戸化、管理不全マンションの増加

### ポイント3: 2030年首都圏大量相続——「空き家第2世代」問題
- 首都圏には65歳以上の高齢者が905万人、うち480万人が後期高齢者
- 高齢者単独世帯は20年間で2.5倍に激増
- 団塊世代（約155万人）が2025年までに全員後期高齢者入り
- **二次相続では配偶者特別控除が使えず、相続税負担が急増**
- 東京都の相続税課税割合は18.7%、世田谷区は33.6%

### ポイント4: 2030年以降、首都圏の家は買いやすくなる
- 今後15年で首都圏で675万〜750万人の相続が発生
- 相続対象物件の3割がマーケットに出るだけで年間8万戸の供給圧力
- 賃貸戸建て住宅という新しい選択肢の登場
- Z世代にとって「家をローンで買う」はコスパ悪い発想になる可能性

### ポイント5: 法改正で空き家放置のペナルティ強化
- **2024年4月〜相続登記が義務化**（違反は10万円以下の過料）
- 管理不全空き家は固定資産税の減免措置が不適用に（実質4.2倍増）
- 相続土地国庫帰属制度の創設（負担金を支払えば国に返納可能）
- 特定空き家は即解体撤去対象に

---

## 3. 実践への示唆（明日から使える3つのアクション）

### アクション1: 親の財産を早めに把握し、家族会議を開く
- 親が元気なうちに不動産を含めた財産内容を確認する
- 実家の未来図（残す/売却/賃貸）について親の意向を聞く
- 認知症になる前に任意後見人の指名や家族信託を検討する

### アクション2: 空き家の出口戦略をデシジョンツリーで整理する
空き家の選択肢は5つしかない：
1. **残す→空き家管理**（期限を決めて問題先送りにしない）
2. **残す→有効利用**（グループホーム、シェアハウス、民泊等）
3. **残す→賃貸・売却**（空き家バンク活用、隣地所有者への声がけ）
4. **更地化→利活用**（駐車場、市民農園、賃貸アパート）
5. **更地化→売却・貸地**（相続土地国庫帰属制度の活用）

### アクション3: 不動産購入は「5年先のエリア状況」を見据える
- 購入エリアの高齢者単独世帯比率をチェックする
- 今後の相続発生見込みと空き家増加を予測する
- 売却・買い替え前提なら中長期のマーケット変化を考慮する

## 4. 印象に残った引用

> 「マンションが買えない！」という悲鳴に近い声が各所から聞こえてきます...しかし見方を変えると、これまで人生で得られるはずの収入のほとんどを住宅購入に注ぎ込んでいた日本人にとって、家なんてどこにでもある単なる消費財となる時代がすぐそこまで来ていると言い換えることができます。

> 「え？家ってローンを組んで買うものだったの？コスパ悪すぎ」——Z世代が社会の中心に躍り出た時、きっとこう嘯くことになるのです。

> 空き家を邪魔者として嫌悪するのではなく、管理していくという発想です...目の前の空き家の存在に解体だ！撤去だ！と目くじらを立てる前に、住民全体で問題の解決にあたる街プラウドの存在が、実は空き家問題解決の根底にあるのです。

---

## 5. こんな人におすすめ

- **親の実家を相続する可能性がある40〜60代**：相続対策と空き家化防止の実践ガイドとして
- **マンション購入を検討している若いカップル**：2030年以降のマーケット変化を知るために
- **不動産投資家・業界関係者**：首都圏大量相続がもたらす市場インパクトの理解に
- **地方自治体・まちづくり関係者**：コンパクトシティ政策と空き家対策の統合的視点に
- **おひとりさま（単身者）**：相続・財産処分・遺贈など終活の具体的指針として

## 補足：本書のキーワード

| 用語 | 説明 |
|------|------|
| 空き家第1世代 | 地方の実家を相続したが放置している人たち |
| 空き家第2世代 | 大都市郊外の実家（ニュータウン等）を相続する人たち |
| 二次相続 | 両親のうち残った一方が亡くなる相続（配偶者特別控除が使えない） |
| 管理不全空き家 | 特定空き家になる恐れのある空き家（固定資産税減免不適用） |
| 相続土地国庫帰属制度 | 相続した不要な土地を国に返納できる制度（2023年施行） |
| 街プラウド | 建物ではなく街・地域に対する愛着と誇り |
