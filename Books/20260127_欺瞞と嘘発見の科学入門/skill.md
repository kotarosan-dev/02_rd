# Evaluative Reasoning & Deception Analysis Framework

クリス・N・H・ストリート『欺瞞と嘘発見の科学入門』（福村出版、2025）に基づく、評価的推論と科学的嘘発見分析のフレームワーク。

## 概要

本フレームワークは、嘘発見に関する一般的な神話を科学的証拠に基づいて解体し、「評価的推論」（クリティカル・シンキング）の能力を用いて真偽判断を行う手法を体系化する。ポピュラーサイエンスにありがちな「嘘を見抜く7つのサイン」といった単純化を避け、この分野の複雑さと不確実性を認識した上で判断を下す。

## 使用場面（When to Use）

- 人事面接や採用判断において信頼性を評価するとき
- 交渉や商談において相手の主張を検証したいとき
- 科学的主張やエビデンスの信憑性を評価するとき
- 組織内のコンプライアンス調査や内部監査を行うとき
- 報道や情報の真偽を判断するとき
- 法執行・司法における取り調べを設計・評価するとき

## コアコンセプト

### 1. 評価的推論（Evaluative Reasoning）とは

> 著者が評価的推論という用語を好むのは、経験上、学生はクリティカル・シンキングを提示された議論の欠点を見つけなければならないという意味と混同しがちだからである。

**評価的推論の定義:**
- 議論に根拠があり、一貫性があり、裏付けとなるデータや理論があるかを評価する
- 単に欠点を探すのではなく、支持する証拠も公平に検討する
- 信念を更新する過程（ベイズ的思考）を形式化したもの

### 2. 真実バイアス（Truth Bias）

人間は相手の発言を真実と仮定する傾向がある：

```
研究知見:
- 1日に嘘をつく回数: 平均2回
- 嘘の頻度の分布: 大きく偏っている
  - 約40-50%の人は24時間まったく嘘をつかない
  - 約5%の人が全体の約半数の嘘をついている
- 真偽判断の正確性: 偶然（50%）よりわずかに優れる程度

→ 人間は「真実を前提とする」ことで社会的信頼を構築している
```

### 3. 嘘発見の神話と現実

| 一般的信念 | 科学的証拠 |
|------------|------------|
| 嘘をつくと視線を逸らす | 信頼できる指標ではない |
| 身体がそわそわする | 真実を語る人も緊張すれば同様 |
| 声が高くなる | 一貫した証拠なし |
| ポリグラフは嘘を検出する | 「情動的覚醒」を測定しており嘘そのものではない |
| 微表情で嘘がわかる | 証拠は懐疑的に見られている |

## フレームワーク/プロセス

### Phase 1: 主張の評価

```
1. 主張を明確化する
   ↓
2. 隠れた前提を特定する
   ↓
3. 反証可能性を検討する
   ↓
4. 証拠の質を評価する
   ↓
5. 代替説明を検討する
```

**反証可能性チェック:**
```
質問: この説明が誤っていることを示すデータは何か？

良い説明:
- 40-60%の正確性を予測する説明
  → 39%以下または61%以上で反証可能

悪い説明:
- 0-100%の正確性を予測する説明
  → すべてのデータと一致し、何も予測しない

最悪の説明:
- あらゆるデータを「言い訳」で一致させる説明
  → 反証不可能＝非科学的
```

### Phase 2: 嘘をつく心理過程の理解

#### ADCAT理論（Activation-Decision-Construction-Action Theory）

```
A: 活性（Activation）
   - 真実は自動的に活性化される
   - 嘘をつくには真実を抑制する必要がある
   ↓
D: 決定（Decision）
   - 嘘をつくか真実を話すかの「期待値」を計算
   - 期待値 = [結果の価値] × [結果の確率]
   ↓
C: 構成（Construction）
   - 長期記憶から素材を引き出して嘘を構成
   - 「埋め込み型の嘘」: 真実の中に嘘を埋め込む
   ↓
A: 実行（Action）
   - 自己表出を管理しながら嘘を実行
   - 嘘をつく人は印象管理により注力
```

#### 認知負荷の観点

```
従来の見解:
  嘘をつくこと > 真実を話すこと（認知的負荷）

しかし、以下の場合は逆転しうる:
- 嘘をつくインセンティブが非常に高いとき
- 嘘が十分にリハーサルされているとき
- 嘘が単純で真実が複雑なとき
- 真実を思い出すより嘘の方がアクセスしやすいとき
```

### Phase 3: 真偽判断の実践

#### ステップ1: バイアスの認識

```
自分の判断に影響するバイアス:
□ 確証バイアス: 自分の信念を支持する情報を好む
□ 真実バイアス: 相手の発言を真実と仮定しがち
□ 後知恵バイアス: 結果を知ってから「わかっていた」と思う
□ サンクコスト効果: 過去の投資が判断を歪める
```

#### ステップ2: 証拠の構造化

```
【主張】
  ↓
【証拠の種類】
  ├── 直接証拠（主張を直接支持するもの）
  ├── 間接証拠（推論を必要とするもの）
  └── 反証（主張と矛盾するもの）
  ↓
【証拠の質】
  ├── 信頼性（情報源の信頼度）
  ├── 妥当性（測定方法の適切さ）
  └── 再現性（他の研究での確認）
```

#### ステップ3: 代替仮説の検討

```
主張A: 「この人は嘘をついている」

代替仮説を列挙:
- 緊張しているだけ
- 文化的に異なる非言語表現
- 記憶の不確かさ
- 誤解や誤報告
- プライバシーへの懸念

各仮説の事前確率と尤度を評価
```

### Phase 4: 意思決定

#### ベイズ的更新

```
事後確率 ∝ 事前確率 × 尤度

例:
- 事前確率: 基準率（一般的にどれくらいの人が嘘をつくか）
- 尤度: 観察された行動が嘘/真実の場合にどれくらい起こりうるか
- 事後確率: 更新された判断

注意: 「嘘のサイン」の尤度比は低い
→ 単一の行動から嘘を判断することは統計的に困難
```

## 具体的な実行手順

### 情報収集面接の設計

```
従来のアプローチ（問題あり）:
- クローズド質問の連発
- 威圧的な雰囲気
- 「嘘のサイン」を探す

推奨アプローチ:
1. オープン質問を使う
   「何があったか教えてください」

2. 認知負荷を高める質問
   「逆の時系列で説明してください」

3. 予期しない質問
   「その場所の匂いを覚えていますか？」

4. 戦略的証拠使用（SUE技法）
   - 最初に自由回答を得る
   - その後、持っている証拠と照合
   - 矛盾点について説明を求める
```

### 主張評価チェックリスト

```
□ 主張は明確に定義されているか
□ 反証可能な予測をしているか
□ 証拠は独立した情報源から得られているか
□ 代替説明は検討されているか
□ 効果量は実用的に意味があるか
□ サンプルサイズは十分か
□ 結果は再現されているか
□ 利益相反はないか
□ 一般化可能な条件か
```

## 入力/出力

### 入力
- 評価対象の主張や発言
- 利用可能な証拠や情報
- 文脈情報（関係性、利害関係、状況）

### 出力
- 主張の信憑性評価（確率的判断）
- 追加情報収集の推奨事項
- 判断の根拠の明示

## 嘘をつく動機の分類

| 次元1 | 次元2 | 例 |
|-------|-------|-----|
| 自己利益 | 利益獲得 | 昇進のための誇張 |
| 自己利益 | 損失回避 | 失敗の隠蔽 |
| 他者利益 | 利益獲得 | 相手を励ますための誇張 |
| 他者利益 | 損失回避 | 相手を傷つけないための隠蔽 |

**文化的差異:**
- 個人主義文化: 自己の誠実さを守る傾向
- 集団主義文化: 人間関係維持のために嘘を許容する傾向

## 注意点

1. **単一の「嘘のサイン」に依拠しない**
   - メタ分析によれば、ほとんどの行動指標は信頼性が低い
   - 同じ行動が嘘にも真実にも現れうる

2. **文脈の重要性**
   - 人は長期的関係がある相手の嘘をよりよく見抜ける可能性
   - 実験室と現実世界では嘘発見の条件が異なる

3. **真偽判断の限界を認識する**
   - 人間の真偽判断は偶然よりわずかに優れる程度
   - 「嘘発見の専門家」も一般人と大差ない

4. **過信を避ける**
   - 自分の判断力への過信が最も危険
   - 不確実性を認め、確率的に考える

5. **倫理的配慮**
   - 嘘発見技術の濫用リスク
   - 偽陽性（無実の人を嘘つきと判断）の深刻さ

## 実践への示唆

### ビジネスパーソン向け
- 採用面接で「嘘を見抜く」ことへの過信を戒める
- 構造化された検証プロセスを設計する
- 非言語行動の解釈には慎重を期す

### 法執行・司法関係者向け
- 単一の「嘘のサイン」に依拠した判断を避ける
- 認知面接法など情報収集法の訓練を受ける
- ポリグラフの結果を過信せず、他の証拠と併せて評価する

### 研究者向け
- 生態学的妥当性の高い実験デザインを追求する
- 個人差・文化差・文脈依存性を考慮した研究を行う
- 嘘発見の「失敗」から学ぶ姿勢を持つ
