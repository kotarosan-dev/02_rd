# パーセプション・マーケティングフレームワーク

## 基本原則

1. **パーセプションは現実**: 「Perception is reality」──客観的な認識こそが現実である
2. **ブランドとパーセプションの対極構造**: ブランドは「企業がどう思われたいか」、パーセプションは「世の中がどう見るか」
3. **5つの要素による形成**: パーセプションは「事象」「リテラシー」「グループ」「タイミング」「コントラスト」の5要素で形成される
4. **客観性の重視**: 企業は自らを客観視し、消費者視点での認識を把握する必要がある

## Instructions

### フェーズ1: パーセプションの現状把握

1. **現状のパーセプションを測定する**: 消費者が商品・ブランドをどう認識しているかを調査
   - 定性調査：消費者インタビュー、フォーカスグループ
   - 定量調査：アンケート調査、ブランド定点調査
   - ソーシャルリスニング：SNS上の言及を分析

2. **ブランドとパーセプションのギャップを特定する**: 企業が目指すブランドイメージと、実際のパーセプションの差を明確化
   - ギャップが大きい領域を優先的に改善
   - オーセンティシティ（ブランドとパーセプションの重なり）を拡大

3. **パーセプション形成の5要素を分析する**: 現在のパーセプションがどの要素によって形成されているかを特定
   - 事象：どのような事実や出来事が認識の起点となっているか
   - リテラシー：対象層の経験や価値観、文化的背景
   - グループ：どのカテゴリーやグループに属していると認識されているか
   - タイミング：社会の潮流や時流の影響
   - コントラスト：競合や対比対象との関係

**完了条件**: 現状のパーセプションが明確に把握され、ブランドとのギャップが特定され、5要素の分析が完了している。

### フェーズ2: パーセプション戦略の選択

1. **五段活用から適切な戦略を選択する**: 目的に応じて「つくる」「かえる」「まもる」「はかる」「いかす」のいずれかを選択
   - **つくる**: 新市場創造、新カテゴリー創出
   - **かえる**: 既存ブランドの活性化、顧客層拡大
   - **まもる**: 危機管理、好ましい認識の維持
   - **はかる**: 継続的な計測と分析
   - **いかす**: 社内変革、イノベーション領域への応用

2. **パーセプションフロー・モデルを設計する**: 消費者のパーセプション変化の流れを可視化
   - 4P（製品、価格、流通、販売促進）を「知覚刺激」として位置づけ
   - パブリシティー→パーセプションチェンジ→ビヘイビアチェンジの流れを設計

3. **PRのピラミッドを活用する**: 戦略PRを初期の知覚刺激として位置づけ
   - 下段：パブリシティー（認知）の獲得
   - 中段：パーセプションチェンジ（認識変容）の実現
   - 上段：ビヘイビアチェンジ（行動変容）の達成

**完了条件**: 目的に応じた戦略が選択され、パーセプションフロー・モデルが設計され、PRのピラミッドが活用計画に組み込まれている。

### フェーズ3: パーセプション形成の実行

1. **事象を創出する**: パーセプション形成の起点となる具体的な事実や出来事を意図的に創出
   - 実証実験、調査データの公表
   - メディアが取り上げやすい「センセーショナルな内容」の設計
   - 第三者機関や専門家の協力を得る

2. **リテラシーのギャップを活用する**: 対象層のリテラシー（経験や価値観）のギャップを利用
   - 海外の事例や文化を導入
   - 専門知識や新しい視点を提供
   - 対象層の潜在的なニーズを発見し、新しいリテラシーを提示

3. **コントラストを生み出す**: 相対的な比較によって新しいパーセプションを形成
   - 競合との明確な差別化
   - 従来の常識との対比
   - 古いものと新しいものの対比

4. **タイミングを読む**: 社会の潮流や時流を読み、適切なタイミングで施策を実行
   - 社会課題やトレンドとの接続
   - 季節性やイベントとの連動
   - 長期的な視点での計画立案

**完了条件**: 事象が創出され、リテラシーのギャップが活用され、コントラストが生み出され、適切なタイミングで施策が実行されている。

### フェーズ4: パーセプションの計測と改善

1. **継続的な計測を実施する**: パーセプションの変化を定期的に測定
   - 時系列での変化を追跡
   - 複数の調査手法を組み合わせる
   - ソーシャルリスニングとの連動

2. **PDCAサイクルを回す**: 計測結果に基づいて戦略を改善
   - パーセプションが目詰まりを起こしていないか確認
   - 成功した場合は広告予算を追加してブースト
   - 失敗した場合は戦略を見直し

3. **オーセンティシティを維持する**: ブランドとパーセプションの整合性を確保
   - 企業の行動とパーセプションの一致
   - ステークホルダー間のパーセプションギャップの解消
   - 炎上リスクの回避

**完了条件**: 継続的な計測が実施され、PDCAサイクルが回り、オーセンティシティが維持されている。

## Examples

### パーセプションを「つくる」事例

- **アリエール**: 「通常の洗濯では菌は残っている」というパーセプションを創出し、除菌洗剤市場を創造
- **atama plus**: 「基礎学力の習得はAIによって効率化できる」というパーセプションを形成し、3年で1900教室に導入
- **資生堂「uno」**: 「第一印象はつくれる」というパーセプションを創出し、男性用メイク市場を開拓
- **Sansan**: 「名刺は企業の資産であり、人脈は共有できる」というパーセプションを形成し、名刺管理サービス市場を寡占

### パーセプションを「かえる」事例

- **Zoom**: ビジネス用途から個人用途へパーセプションを拡張し、利用を急拡大
- **ジレット**: 「最高の男」のパーセプションチェンジで成功
- **ワークマン**: 荒天再現の過酷イベントで、実用性の高いパーセプションを強化

### パーセプションを「まもる」事例

- **大戸屋**: 広報戦略で「手作りvs工場」の対立構造を描き、パーセプションを死守
- **吉野家**: 「うまい、やすい、はやい」の価値を飲食を超えたパーセプションとして維持

## Common Mistakes

- **認知とパーセプションの混同**: 認知度の向上だけでパーセプションが変わると思い込む
- **主観的な視点の維持**: 企業側の視点だけで戦略を立て、消費者視点を欠く
- **事象の不在**: パーセプションを変えたいが、具体的な事象を創出しない
- **タイミングの誤読**: 社会の潮流を読まずに施策を実行する
- **計測の軽視**: パーセプションの変化を計測せず、効果を検証しない
- **オーセンティシティの欠如**: ブランドとパーセプションのギャップを放置し、炎上リスクを高める
- **短期的な視点**: パーセプション形成は中長期的な視点が必要なのに、短期的な成果を求めすぎる
- **PRのピラミッドの順序無視**: パブリシティーを飛ばして、いきなり広告に予算を投下する
