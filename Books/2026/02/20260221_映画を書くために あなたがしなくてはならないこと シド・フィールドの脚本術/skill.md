# シド・フィールド脚本術フレームワーク

## 概要

シド・フィールド著『映画を書くためにあなたがしなくてはならないこと』および『素晴らしい映画を書くためにあなたに必要なワークブック』に基づく、物語構造の設計・執筆フレームワーク。「パラダイム」と呼ばれる三幕構成モデルを軸に、キャラクター構築からシーン設計、推敲に至るまで、脚本執筆の全工程を体系的にカバーする。

## 使用場面（When to Use）

- 映画脚本の企画・執筆
- テレビドラマ・ウェブドラマのシナリオ構成
- 小説・短編小説のプロット設計
- ゲームシナリオ・インタラクティブストーリーの構造設計
- ビジネスプレゼンテーションのストーリー構成
- 企画書・提案書の物語的構成
- コンテンツマーケティングにおけるナラティブ設計
- ドキュメンタリーの構成設計

## フレームワーク/プロセス

### Phase 1: パラダイム（構造の設計）

```
|<--- 第一幕 --->|<-------- 第二幕 ---------->|<--- 第三幕 --->|
|   セットアップ  |    コンフロンテーション      |  レゾリューション |
|    (設定)      |      (対立・葛藤)           |    (解決)      |
|    約1/4       |        約1/2               |    約1/4       |
|               PP-I                  MP                  PP-II              |
|                ^                    ^                    ^                  |
|         プロットポイントI      ミッドポイント      プロットポイントII         |
```

**三幕の機能:**
1. **第一幕（セットアップ）**: 主人公、状況、ドラマティック・ニード（劇的欲求）を提示する
2. **第二幕（コンフロンテーション）**: 主人公が障害に直面し、葛藤を経験する
3. **第三幕（レゾリューション）**: 物語を結末に導き、問いに答える

**構造的転換点:**
- **プロットポイントI**: 物語の方向を大きく変え、第二幕へ推進する出来事
- **ミッドポイント**: 第二幕の中央で物語の前半と後半を構造的に分割する出来事
- **プロットポイントII**: クライマックスへの流れを決定づけ、第三幕へ推進する出来事

### Phase 2: 主題（テーマ）の確立

物語の核心を一文で表現する:
- **主題 = Action + Character（行動 + 人物）**
- 例: 「元ボクサーが自らの尊厳を取り戻す物語」（ロッキー）
- 例: 「私立探偵が権力の腐敗の真相を追う物語」（チャイナタウン）

### Phase 3: キャラクター構築

#### 3.1 キャラクターの二つの側面

```
内面的な生活          外面的な生活
(Interior)           (Exterior)
・生い立ち            ・物語開始時の状況
・家族関係            ・職業
・トラウマ            ・人間関係
・価値観              ・行動パターン
・恐れ                ・台詞の特徴
```

#### 3.2 キャラクター伝記の作成

以下の要素を詳細に設定する:
1. **誕生から物語開始まで**: 出生地、家庭環境、教育、重要な出来事
2. **ドラマティック・ニード**: 主人公が物語の中で何を望んでいるか
3. **態度（Attitude）**: 世界に対する基本的な姿勢
4. **変化（Transformation）**: 物語を通じてどう変わるか

#### 3.3 キャラクター設計の原則

- **「行動は人物である」（Action is Character）**: 人物は行動によって定義される
- 台詞ではなく行動で人物を描く
- 障害の大きさが人物の深みを決める
- 人物の「必要（need）」と「欲求（want）」を区別する

### Phase 4: 四ページのあらすじ

パラダイムに基づき、以下の構造で物語の概要を4ページにまとめる:
- **1ページ目**: 第一幕（セットアップ）の概要
- **2-3ページ目**: 第二幕（コンフロンテーション）の概要
- **4ページ目**: 第三幕（レゾリューション）の概要

### Phase 5: シーン設計

#### 5.1 シーンの基本要素

各シーンは以下を含む:
1. **目的**: このシーンは物語のどこに位置し、何を達成するか
2. **場所と時間**: どこで、いつ起きるか
3. **コンフリクト**: 何が対立しているか
4. **変化**: シーンの開始時と終了時で何が変わるか

#### 5.2 最初の10ページの設計

冒頭10ページで以下を確立する:
- [ ] 主人公は誰か
- [ ] 何についての物語か
- [ ] ドラマティックな状況は何か
- [ ] 物語のトーンは何か
- [ ] 読者/観客を引きつけるフックがあるか

### Phase 6: 執筆と推敲

#### 6.1 執筆の手順

1. エンディングを決める
2. オープニングを決める
3. プロットポイントIを決める
4. プロットポイントIIを決める
5. ミッドポイントを決める
6. 各幕の内部構造を肉付けする
7. シーンごとに執筆する

#### 6.2 推敲のチェックリスト

- [ ] 三幕構成が機能しているか
- [ ] プロットポイントが物語を効果的に転換させているか
- [ ] 主人公のドラマティック・ニードが明確か
- [ ] 行動がキャラクターを表現しているか
- [ ] 最初の10ページが読者を引きつけるか
- [ ] 各シーンが物語を前進させているか
- [ ] 台詞が自然で、キャラクターに固有か
- [ ] エンディングが主題に応えているか

## 具体的な実行手順

### ステップ1: 主題の定義
- 一文で物語の核心を表現する
- 「誰が」「何を」「どうする」の要素を含める

### ステップ2: エンディングの決定
- 結末を先に決めることで、物語全体の方向性を定める
- 「どう終わるか」は「何を語りたいか」と直結する

### ステップ3: キャラクター伝記の作成
- 主要キャラクター（少なくとも主人公）の伝記を書く
- 物語開始前の人生を詳細に設定する

### ステップ4: 四ページのあらすじ
- パラダイムに沿って物語の概要を4ページにまとめる
- 三幕の配分を意識する

### ステップ5: シーンカードの作成
- 各シーンを1枚のカード（またはメモ）に要約する
- カードを並べ替えて最適な順序を見つける

### ステップ6: 第一稿の執筆
- 完璧を求めず、まず全体を書き上げる
- 「書くことについて書く」のではなく、「映像で語る」

### ステップ7: 推敲
- 構造レベル、シーンレベル、台詞レベルの三段階で推敲する
- 各推敲では一つのレイヤーに集中する

## 入力/出力

### 入力
- 物語のアイデアやテーマ
- キャラクターの着想
- 伝えたいメッセージや感情

### 出力
- 構造化された脚本（映画: 約120ページ）
- プロット設計書
- キャラクター設計書
- シーン構成表
- 四ページのあらすじ

## 応用: ビジネスプレゼンテーションへの転用

```
第一幕（セットアップ）     → 課題の提示、現状分析
プロットポイントI          → 課題の深刻さを示すデータ/事例
第二幕（コンフロンテーション） → 解決策の検討、比較分析
ミッドポイント              → 核心的なインサイトの提示
プロットポイントII          → 具体的な提案への転換
第三幕（レゾリューション）   → 提案の詳細、期待される成果、行動の呼びかけ
```

## 注意点

- **パラダイムは「型」であって「枠」ではない**: 構造を理解したうえで、意図的に逸脱することは創造的な行為である。まず型を学び、その後で崩す
- **すべての映画がパラダイムに従うわけではない**: 特にヨーロッパやアジアのアートフィルムは異なる構造原理に基づくことが多い
- **構造は物語の「骨格」であって「全体」ではない**: 構造が優れていても、キャラクター、台詞、映像的想像力が伴わなければ良い脚本にはならない
- **「書くこと」は「書くこと」によってのみ学べる**: 理論の学習は必要条件であるが、十分条件ではない。実際に書く実践が不可欠である
- **本フレームワークはシド・フィールドの脚本術三部作（特に第1作と第2作）に基づく**: より深い学習のためには原著の通読を推奨する
