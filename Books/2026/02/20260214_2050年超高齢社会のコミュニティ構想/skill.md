# 超高齢社会の地域コミュニティ設計フレームワーク

## 出典
若林靖永・樋口恵子 編『2050年 超高齢社会のコミュニティ構想』岩波書店、2015年

## フレームワーク概要
超高齢・少子・人口減少・単身社会において、地域住民が穏やかに暮らし続けるための地域コミュニティの設計手法。「確かな未来」（人口動態）を起点に、バックキャスティングで地域の「こうありたい姿」を描き、その実現主体と仕組みを構築する。

## 構成要素

### 1. 動かぬ与件の特定
地域の未来を規定する変動不可能な条件を洗い出す。

| 与件 | 内容 |
|------|------|
| 超高齢化の継続 | 70歳以上が人口の3割超 |
| 単身世帯の多数派化 | 全世帯の42%が単身 |
| 元気な高齢者の存在 | 70歳代の90%が自立可能 |
| 空き家・空き施設の激増 | 居住地の2割が無居住化 |
| 従来の家族像の瓦解 | 標準世帯が18%に縮小 |
| 大介護時代の到来 | 認知症800万人時代 |

### 2. 地域単位の設定：小学校区
設計単位として小学校区を選択する根拠は3条件の同時充足。

- **徒歩圏のアクセス性**：子育て世代、高齢者、学童が無理なく歩いてアクセスできる
- **地域アイデンティティ**：住民がリアリティを持てる地縁の単位
- **ビジネスの最低採算性**：人口約5000人の商圏として成立

### 3. 地域拠点の3機能設計（「集いの館」モデル）

```
+--------------------------------------------------+
|           集いの館（合計90坪）                      |
|                                                    |
|  +------------+  +------------+  +---------------+ |
|  | お店       |  | よろず     |  | フリー        | |
|  | (30坪)     |  | 相談デスク |  | スペース      | |
|  |            |  |            |  | (60坪)        | |
|  | 食料品     |  | 暮らし全般 |  | 交流・見守り  | |
|  | 日用医薬品 |  | の相談窓口 |  | 学童保育      | |
|  | 配食       |  | 専門機関   |  | 子育て支援    | |
|  |            |  | への接続   |  | 多世代交流    | |
|  +------------+  +------------+  +---------------+ |
+--------------------------------------------------+
      運営主体：元気な高齢者チーム（毎日17名体制）
```

#### 機能1：お店（物販機能）
- コンビニ型30坪、当日消費の食品と日用医薬品
- 宅配ステーション機能
- 年365日、8:00-19:00営業
- 担当：毎日9名（午前4名・午後5名）

#### 機能2：よろず相談デスク（相談機能）
- ワンストップで暮らし全般の相談に対応
- 対象領域：医療、介護、子育て、見守り、家事支援、保険、住宅、教育、雇用、葬祭等
- 専門機関（地域包括支援センター、社会福祉法人、NPO等）への接続
- 担当：毎日6名（午前・午後各3名）

#### 機能3：フリースペース（交流機能）
- 老若男女が出入り自由で過ごせる場
- 簡易調理設備、食卓テーブル完備
- 小学校区の実情に応じた空間配分（固定モデルなし）
- 担当：毎日2名（午前・午後各1名）

### 4. 担い手モデル：元気な高齢者の社会参画設計

参画を促すインセンティブ設計のポイント：
- 柔軟な勤務時間（本人の都合で選択可能）
- 役割の選択性（好きな仕事を担当）
- 少額報酬の支給
- 友人関係の拡大
- 世代間交流の機会
- 健康状態の確認
- 共食の場の提供

必要人員試算：
- 1館あたり毎日17名 x 365日 = 年間延べ6,205人日
- 全国15,000館 = 年間9,307.5万人日
- 70歳代の元気な高齢者約1,360万人が一人年7日参加で充足

### 5. ネットワーク設計：ハブ&スポークモデル

「集いの館」をハブとし、以下の組織・機関をスポークとしてネットワーク化する。

- 生活協同組合（購買、医療、福祉、共済、住宅等）
- 地域包括支援センター
- 社会福祉法人
- NPO・ボランティア団体
- ワーカーズコレクティブ
- 民間企業（コンビニ、ドラッグストア、外食・中食企業等）
- 行政機関（市区町村）
- 医療機関
- 教育機関

### 6. 収益モデル

| 収益源 | 内容 |
|--------|------|
| 物販利益 | 「お店」の商品販売による収益 |
| 紹介手数料 | よろず相談デスクから専門事業者への顧客紹介 |
| 代理店収入 | 介護・医療・金融等のサービスの代理契約 |
| 配達手数料 | ネット通販商品の受取拠点としての機能 |
| 施設利用料 | フリースペースでのイベント・教室開催 |

## 適用手順

1. **与件分析**：対象地域の人口動態、世帯構造、高齢化率、空き家率等を把握
2. **単位設定**：小学校区を基本単位とし、地域特性に応じて調整
3. **資源棚卸し**：活用可能な空き施設、元気な高齢者の人数、既存組織の機能を調査
4. **ボトムアップ設計**：地域住民（特に高齢者）との対話を通じて3機能の空間配分を決定
5. **ネットワーク構築**：ハブ&スポークの接続先となる組織・機関との連携関係を構築
6. **試行運用**：プロトタイプを運用し、住民のニーズと運営の実態を照合
7. **横展開**：成功モデルを他の小学校区へ水平展開

## 活用場面

- 自治体の地域包括ケアシステム設計
- 生活協同組合の地域戦略策定
- NPO・社会的企業の事業企画
- 空き家・空き施設の利活用計画
- 企業の地域貢献・CSV戦略
- コミュニティビジネスの事業計画立案
