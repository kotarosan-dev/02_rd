# Evolutionary Social Psychology Framework

ウィリアム・フォン・ヒッペル『われわれはなぜ嘘つきで自信過剰でお人好しなのか――進化心理学で読み解く、人類の驚くべき戦略』（ハーパーコリンズ・ジャパン、2019）に基づく、進化的視点からの社会的行動分析フレームワーク。

## 概要

本フレームワークは、人間の社会的行動（協力、競争、欺瞞、リーダーシップ、幸福追求）を、サバンナでの生存適応の産物として体系的に分析する手法を提供する。「なぜそのように振る舞うのか」を進化的機能から理解することで、個人・組織・社会レベルでの行動予測と介入設計に活用できる。

## 使用場面（When to Use）

- 組織内のリーダーシップ構造を設計・評価するとき
- チーム内の協力と競争のバランスを最適化したいとき
- マーケティングやプロダクト設計で人間心理の深層を理解したいとき
- 社会的比較や自信過剰がもたらすバイアスを分析するとき
- 従業員の幸福度・エンゲージメント施策を設計するとき
- 交渉や説得の戦略を進化心理学的に設計するとき

## コアコンセプト

### 1. 社会的跳躍（The Social Leap）

> 森を離れたのは人類のほうではなく、森がわたしたちから離れていった。

人間の心理を形作った根本的イベントは、約600万~700万年前の熱帯雨林からサバンナへの移行である。この環境変化が、以下の連鎖的適応を引き起こした：

```
熱帯雨林の消失
  ↓
サバンナでの捕食圧増大
  ↓
集団協力の必要性
  ↓
社会的知性の進化
  ↓
言語・文化・イノベーションの発生
  ↓
農業革命 → 階層社会の形成
```

### 2. 協力と欺瞞の共進化

人間の協力能力と欺瞞能力は表裏一体として進化した：

| 適応的機能 | 協力面 | 欺瞞面 |
|-----------|--------|--------|
| 集団防衛 | 石投げの協調行動 | 脅威の誇張による結束強化 |
| 情報共有 | 知識の伝達・共同注意 | 都市伝説・感情的誇張 |
| 配偶者獲得 | 互恵的利他行動 | 自己高揚・自信過剰 |
| 社会的地位 | 能力の正直な信号 | 身の丈を超えた信号の発信 |

**自己欺瞞の適応的機能：**
- 自分を「実際より良く」信じ込むことで説得力が増す
- 自信過剰は「正直な信号」として機能しうる
- 自己欺瞞が先行し、他者への欺瞞が後続する

### 3. 社会的相対性理論

> 重要なのは、わたし本人がどれほど利口で魅力的なのかではなく、まわりの男性たちよりも利口で魅力的であるという点だ。

人間の満足度・幸福感は絶対的水準ではなく相対的地位で決まる：

```
性淘汰の原理
  ↓
相対的地位の重要性
  ↓
社会的比較の常態化
  ↓
「快楽適応の罠」（ヘドニック・トレッドミル）
  ↓
絶対的な豊かさが幸福に直結しない
```

### 4. ゾウ型 vs ヒヒ型リーダーシップ

| 特性 | ゾウ型リーダー | ヒヒ型リーダー |
|------|--------------|--------------|
| 動機 | 集団奉仕型 | 自己奉仕型 |
| 選出基準 | 知恵・経験 | 攻撃性・支配力 |
| 資源配分 | 平等・相互的 | 独占・偏在 |
| 地位の特権 | ほぼなし | 大きい（報酬・繁殖等） |
| 脅威への反応 | 集団利益を維持 | 自己保身に走る |
| 出現条件 | 資源独占不可能な環境 | 資源独占可能な環境 |
| 人間社会の例 | ハッザ族、分散型組織 | ヤノマミ族、階層型企業 |

**リーダーシップ類型を規定する構造的条件：**
```
資源の独占可能性が低い → ゾウ型リーダーシップ
  例：遊牧社会、フラット組織

資源の独占可能性が高い → ヒヒ型リーダーシップ
  例：農耕社会、階層的企業

婚姻形態:
  一夫一婦制 → ゾウ型を促進
  一夫多妻制 → ヒヒ型を促進

報酬構造:
  長期業績連動 → ゾウ型を促進
  固定高額報酬 → ヒヒ型を促進
```

## フレームワーク/プロセス

### Phase 1: 行動の進化的機能分析

対象となる行動パターンの進化的起源を特定する：

```
1. 行動の記述
   「何が起きているか」を価値判断なしに記述
   ↓
2. 進化的環境での機能推定
   「祖先の環境でこの行動はどんな適応的機能を果たしたか」
   ↓
3. 現代環境との不一致（ミスマッチ）の特定
   「祖先の環境と現代環境で何が変わったか」
   ↓
4. 行動の再評価
   「この行動は現代でも適応的か、それともミスマッチか」
```

### Phase 2: 社会的ダイナミクスの診断

組織やチームの社会的構造を進化的枠組みで診断する：

```
チェック項目:
□ 資源配分は独占型か分散型か
□ リーダーの報酬は集団の業績と連動しているか
□ メンバーの移動性は高いか低いか
□ 「不正者検出」のメカニズムが機能しているか
□ 社会的比較が過度に促進されていないか
□ 集団間競争と集団内競争のバランスはどうか
```

### Phase 3: 介入設計

進化心理学的知見に基づいた介入策を設計する：

```
問題: ヒヒ型リーダーシップの蔓延
介入策:
  1. リーダー報酬を長期業績に連動させる
  2. 内部昇進を重視し、候補者の行動履歴を評価する
  3. 分野別の専門リーダーシップを認める
  4. 不平等の構造的要因を低減する

問題: 快楽適応による従業員の不満
介入策:
  1. 物質的報酬より体験型報酬を重視する
  2. 相対評価ではなく絶対評価の要素を増やす
  3. 社会的つながりの機会を設計に組み込む
  4. 貢献の可視化を地位ではなく意味で行う

問題: 自信過剰によるリスク管理の失敗
介入策:
  1. 意思決定における「悪魔の代弁者」の制度化
  2. 過去の予測精度の定量的フィードバック
  3. 集団的意思決定プロセスの導入
  4. 自信過剰が適応的な場面と非適応的な場面の区別
```

## 具体的な実行手順

### 組織のリーダーシップ類型診断

```
1. 報酬構造の分析
   - リーダーと平均従業員の報酬比率を算出
   - 報酬の業績連動度を評価
   ↓
2. 資源配分パターンの調査
   - 情報・権限・機会の集中度を測定
   - 「分割統治」行動の有無を確認
   ↓
3. リーダー選出プロセスの評価
   - 内部昇進vs外部採用の比率
   - 選出基準（専門性vs支配性）の確認
   ↓
4. 類型の判定と介入方針の策定
   - ゾウ型/ヒヒ型/混合型の判定
   - 構造的条件の改善策を設計
```

### 幸福度向上のための進化的チェックリスト

```
社会的つながり:
□ 心の内を共有できる相手がいるか
□ 集団への所属感があるか
□ 高齢期の孤独リスクに対処しているか

相対性の罠への対処:
□ 社会的比較を意識的に管理しているか
□ モノよりコト（経験）に投資しているか
□ 地位の追求と幸福の追求を区別しているか

進化的命令への対応:
□ 協力と互恵性の機会があるか
□ 専門性を活かした貢献ができているか
□ 次世代への投資（子育て・教育・指導）ができているか
```

## 入力/出力

### 入力
- 分析対象の社会的行動や組織構造
- 環境条件（資源配分、移動性、競争構造）
- 個人・集団の行動パターンデータ

### 出力
- 行動の進化的機能の分析結果
- リーダーシップ類型の診断
- 進化心理学に基づく介入提案
- 進化的ミスマッチの特定と対処策

## 注意点

1. **遺伝決定論に陥らない**: 進化は傾向を与えるが決定はしない。遺伝子は環境と相互作用して発現する。近視の例のように、同じ遺伝的素因でも環境によって発現は大きく異なる

2. **適応主義の過剰適用を避ける**: すべての行動に「適応的理由」を見出そうとするのは方法論的に危うい。仮説と実証を区別する

3. **文化的多様性への配慮**: 進化的傾向は普遍的でも、その表現形は文化によって大きく異なる。ハッザ族とヤノマミ族の差が示すように、同じ進化的素因が異なる社会構造の下で全く異なる行動を生む

4. **現代環境とのミスマッチに注意**: 進化的に適応的だった行動が現代環境で不適応になる場合がある。自信過剰は小集団では有利でも、大規模組織では破壊的になりうる

5. **倫理的配慮**: 「進化的に自然」であることは「道徳的に正しい」ことを意味しない（自然主義的誤謬）。進化心理学は行動の説明を提供するが、行動の正当化には使えない
