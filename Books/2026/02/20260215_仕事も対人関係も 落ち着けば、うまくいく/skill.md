# 落ち着きのフレームワーク ―― 平常心で実力を発揮する技術

## 概要

精神科医・和田秀樹が提唱する、緊張・不安・焦りをコントロールし、平常心を保って本来の実力を発揮するための体系的メソッド。認知行動療法、森田療法、行動経済学の知見を統合し、「心の仕組みの理解」「認知の修正」「自信の構築」「割り切りの習慣化」という四段階で感情マネジメントを実践する。

## 使用場面（When to Use）

- 重要なプレゼンテーション・商談・面接の前に緊張が高まるとき
- 苦手な相手とのコミュニケーションで気後れするとき
- 完璧主義や周囲の評価への過敏さで自分を追い込んでいるとき
- 想定外の出来事に直面して動揺しているとき
- 失敗への恐怖でチャレンジを避けてしまうとき
- 日常的にストレスや不安感が強く、パフォーマンスが低下しているとき

## フレームワーク/プロセス

### Phase 1: 心の仕組みを診断する（自己認知）

緊張の根源にある自分の思考パターンを特定する。

**チェックすべき5つの思考習慣:**

| 思考パターン | 特徴 | 自問 |
|---|---|---|
| 目先思考 | 今のことしか見えず長期視点がない | 「この問題は人生全体でどれほど重要か？」 |
| かくあるべし思考 | 「こうすべき」と決めつけて柔軟性がない | 「他の選択肢や道はないか？」 |
| 二分割思考 | 白か黒か、成功か失敗かの二択で判断する | 「中間の可能性はないか？」 |
| 勝ち負け思考 | すべてを勝敗で捉える | 「勝ち負け以外の評価軸はないか？」 |
| 自意識過剰 | 周囲の目を過度に気にする | 「実際に人はどれほど自分を見ているか？」 |

### Phase 2: 慌てない心をつくる（認知の修正 ── 三段階ステップ）

**ステップ1: 誤った先入観を変える**
- 想定外の範囲を狭める（起こりうる事態の9割は事前に想定可能）
- 「いい楽観主義」を身につける（楽観＋準備の両立）
- 失敗を織り込み済みにする（最悪のシナリオに対処法を用意）

**ステップ2: 誤った考え方を変える**
- 「緊張は悪いことではない」と捉え直す（いい緊張の活用）
- 「考えたくないこと」をあえて考える（回避ではなく直面）
- 「理屈通りにいかない」前提で行動する
- 頭の中だけで整理せず、書き出して可視化する
- 苦手なことは「スキップする方法」を考える

**ステップ3: 誤った行動を変える**
- 10割を目指さない（7〜8割で合格とする）
- 迷ったら多くの意見を聞く
- 失敗したら別の方法を試す
- うまくいかなかったときの「逃げ場」を用意しておく

### Phase 3: 自信を育てる（森田療法的アプローチ）

**原則: 変えられるものを変え、変えられないものは手放す**

- 得意なことに集中し、不得意なことは放置する
- 一度でうまくいくと考えない（試行錯誤を前提にする）
- 「まぁ、いいか」とスルーせず、改善策を考え続ける
- 「取り柄」を一つ磨き、それを自分の売り物にする
- 「場数」を踏むことで慣れを獲得する
- プロセスではなく「結果」を評価軸にする

### Phase 4: 割り切りを習慣化する（日常の感情設計）

**人間関係の割り切り:**
- 利害関係がない人は気にしない
- みんなに好かれようと思わない
- 無理して緊張を隠さない

**発想の割り切り:**
- 今は負けても次に勝てばいい
- 「今を楽しむ」ことを優先する
- 自分を大きく見せようとしない
- 自分一人で抱え込まない
- 「参照点」を意識的に低く設定する

**日常生活の割り切り:**
- 睡眠不足を深刻に考えすぎない
- 食べたいものを食べる（栄養バランスは意識しつつ）
- 我慢せず「いい思い」を積み重ねる

## 具体的な実行手順

### 緊急時（本番直前の緊張対処）

1. 深呼吸を3回行い、身体の緊張を緩める
2. 「最悪の場合何が起きるか？」を具体的に書き出す
3. 「次のチャンスは得られるか？」を自問する → Yesなら「失敗しても大丈夫」と確認する
4. 自分に合ったリラックス法（音楽、香り、ストレッチなど）を実行する

### 日常的な習慣形成

1. **毎朝**: 今日の「参照点」を意識する（期待値を適切に設定する）
2. **業務中**: 「かくあるべし」「二分割」「勝ち負け」思考に陥っていないか内省する
3. **困難な場面**: 「変えられるもの」と「変えられないもの」を分類し、前者に集中する
4. **失敗時**: 「次のチャンスが得られるか？」を判断基準にリカバリー策を立てる
5. **一日の終わり**: 小さな成功や楽しかった出来事を振り返り、参照点を調整する

## 入力/出力

### 入力
- 緊張・不安・焦りを感じている具体的な状況
- 自分の思考パターンの自己診断結果（Phase 1のチェック）
- 「変えられるもの」と「変えられないもの」のリスト

### 出力
- 緊張の原因分析と対処戦略
- 自分の認知の偏りの特定と修正方針
- 具体的な行動計画（準備・リハーサル・逃げ場の設計）
- 参照点の再設定による期待値の適正化

## 注意点

- 本フレームワークは予防と軽度の改善を目的としている。過緊張や不安症と診断されるレベルの症状がある場合は、専門の精神科医やカウンセラーへの相談が不可欠である
- 「考え方を変えれば解決する」という過度な個人還元に陥らないよう注意する。職場環境や人間関係の構造的問題が原因の場合は、環境の改善も並行して検討する必要がある
- 「弱点を放置する」「やりたくないことはやらない」といった助言は、自分の立場や責任を踏まえて適用範囲を見極める必要がある
- 参照点を下げることは「向上心を捨てる」ことではない。努力は継続しつつ、結果に対する期待値を現実的に設定するという意味である
