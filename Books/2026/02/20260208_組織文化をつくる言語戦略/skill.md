# 多国籍組織の言語戦略フレームワーク

## 概要

企業における言語は経営課題である。言語教育を個人のスキル獲得に留めず、組織文化の形成と経営へのインパクトを生み出す人事施策として設計する。CEFRを日英共通の評価指標とし、言語データの蓄積と活用によって組織の状態を可視化し、意思決定に活かす。

## 使用場面（When to Use）

- 多国籍組織で言語・コミュニケーションの課題が顕在化しているとき
- 日本語研修・英語研修の成果が現場のパフォーマンスにつながっていないとき
- 社内公用語の導入を検討しているが判断に迷うとき
- 外国籍社員の採用要件における言語スキルの設定方法を見直したいとき
- 言語教育を外部ベンダーに丸投げしており、自社にフィットしていないと感じるとき
- AI翻訳の普及に伴い、言語教育の方向性を再考する必要があるとき
- D&I推進において言語面のインクルージョンを強化したいとき

## 3つの柱：言語教育プログラムの設計

### 柱1: 非母語話者向け言語教育（日本語・英語）

**従来型（文型中心型）の問題点**:
- 文型の積み上げ式では「会話ができる」状態に至らない
- 教室での学習と現場でのコミュニケーションの乖離
- 画一的なカリキュラムが多様な職場環境に対応できない

**行動中心アプローチ（CEFR準拠）**:
1. 「言語を使って何ができるか」を起点にカリキュラムを設計
2. 自社の業務で必要な課題遂行能力を特定
3. 課題遂行の達成度をCEFR基準で評価
4. スピーキングテスト等で運用力を直接測定

### 柱2: 母語話者教育（やさしいコミュニケーション）

**基本原則**: コミュニケーションの負担は双方が負う

**プログラム内容**:
1. 自分の言語・コミュニケーションスタイルをメタ認知する
2. 「やさしい日本語」の原則を学ぶ（一文一義、短文化、専門用語の言い換え）
3. プレイン・ランゲージの国際的概念を理解する
4. 相手の言語レベルに応じた調整能力を育成する
5. 非言語コミュニケーション（声のトーン、表情、ジェスチャー）の活用

**克服すべき心理的障壁**:
- 「日本語の難しさへの信仰」: 日本語の難解さを無意識に誇りとする傾向
- 言語的優位性の立場の優位性へのすり替え
- 「やさしい日本語」を使うことへの心理的抵抗

### 柱3: Meeting Halfway（歩み寄り）

**概念**: 双方が歩み寄ることでコミュニケーションの質を最大化する

**実践方法**:
- 非母語話者: 学習言語での課題遂行能力を高める
- 母語話者: 相手のレベルに合わせた言語調整を行う
- 双方: 文化的背景や価値観の違いを前提としたコミュニケーションを実践する

## CEFR活用による言語能力の可視化

### 導入のメリット

1. **共通指標**: 日英で同じ物差しを使うことで、異なる言語の話者間の協働可能性を判断できる
2. **行動基準**: 「何ができるか」と「どのような制限があるか」の両面を把握できる
3. **組織分析**: 部門・チーム単位での言語レベル分布を可視化し、課題を特定できる
4. **経営判断**: 言語データを人事データとクロスさせることで、施策の効果を測定できる

### レベル別の行動特性（組織での振る舞い）

| CEFRレベル | 組織内での典型的な行動 |
|---|---|
| A1-A2 | 会議での議論についていくのが困難、発言がほとんどできない |
| B1 | 自分の言語能力不足による困り事に言及することが多い |
| B2 | チーム内で困っている人をどう支援するかを考える、会議のキーパーソンとなる |
| C1-C2 | 母語に近いレベルで組織に貢献、言語調整を意識的に実践 |

### データの活用方法

1. 個人の言語プロファイル（例: 日本語C1・英語B1）を把握
2. チーム・部門単位のレベル分布を分析
3. レベル分布と実際の課題（ヒアリング結果）の整合性を検証
4. エンゲージメントサーベイ、マネージャー評価との相関を分析
5. 言語レベルの推移と行動変容の追跡

## 社内公用語に対する戦略的判断

**メルカリが社内公用語を定めない理由**:
- 全員英語化は発言の減少と心理的安全性の低下を招く
- 言語レベルの分布上、英語だけの会議が成立しないケースが多い
- 一律の言語政策ではなく、チームの言語プロファイルに基づく柔軟な対応が有効

**判断プロセス**:
1. チームの言語プロファイル（CEFR分布）を確認する
2. 会議が成立するかどうかをデータに基づき判断する
3. 必要に応じて言語教育チームに相談し、プランを策定する
4. 結果を追跡し、フィードバックを行う

## サイトスペシフィックな言語教育の設計

**原則**: 自社の文脈に紐づいた教材・プログラムこそが効果を生む

**設計手順**:
1. 経営陣の困り事と将来のビジョンをヒアリングする
2. 現場で実際にどんな場面で言語が使われているかを調査する
3. 自社特有の語彙・表現を収集しコーパス化する
4. 現場の文脈に即した教材・トレーニングを開発する
5. 成果の追跡と改善のサイクルを回す

**事例**: メルカリでは、Slackでやりとりされる注目の話題を音声化し、シャドーイング教材として活用するツールを開発した。

## 言語教育を経営にインパクトさせるための2つの可視化

### 可視化1: 変化の可視化

言語教育による個人のスキル成長とマインドセットの変化が、実際の行動変容につながっているかどうかを追跡する。

**測定方法**:
- CEFR レベルの推移
- 行動変容の自己評価アンケート
- 他者（上司・同僚）からの行動評価

### 可視化2: 組織インパクトの可視化

言語レベルの変化・行動変容と、組織成果との定量的な関係性を探る。

**掛け合わせるデータ**:
- 言語レベルデータ（CEFR） x エンゲージメントサーベイ
- 言語レベルデータ x マネージャー評価
- 言語レベルデータ x Slack等のコミュニケーションログ分析
- 言語レベルデータ x 離職率・定着率

## 注意点

- 言語教育の成果が表れるには時間がかかる。短期的なROIだけで判断しない
- 経営陣は漠然とした方向性は持っていても具体的な答えは持っていない。言語教育の専門家が解像度を上げる役割を担う
- AI翻訳は情報伝達の正確性を向上させるが、「伝えたい熱量」や価値観の違いへの対応は依然として人間の課題
- 言語の問題として片付けられていた課題の本質は、コミュニケーションの質と「違いに対する向き合い方」にある
- 外部ベンダーへの丸投げではなく、自社に専門家を配置し、サイトスペシフィックな施策を設計・運用することが最も効果的
