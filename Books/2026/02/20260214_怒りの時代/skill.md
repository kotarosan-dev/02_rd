# ルサンチマン構造分析フレームワーク

## 出典
パンカジ・ミシュラ『怒りの時代――世界を覆い続ける憤怒の近現代史』（草思社、2021年）

## 概要
社会的・政治的な怒りの噴出を、近代化に内在する構造的矛盾として分析するためのフレームワーク。テロリズム、ポピュリズム、排外主義などの現象を、個別の病理としてではなく、「模倣の欲望」と「ルサンチマン」の力学として体系的に理解する。

## フレームワーク構成

### 1. 模倣の欲望（Mimetic Desire）の分析

| 分析項目 | 問い | 具体例 |
|----------|------|--------|
| 模倣対象 | 誰の・何を模倣しようとしているか | 西洋の物質的繁栄、制度、ライフスタイル |
| 模倣動機 | なぜ模倣が始まったか | 軍事的敗北、経済格差の認識、万博等の衝撃 |
| 模倣の代償 | 模倣の過程で何が失われたか | 伝統文化、共同体の紐帯、精神的基盤 |
| 模倣の限界 | なぜ完全な模倣が不可能か | 資源不足、歴史的蓄積の欠如、制度の移植困難 |

### 2. ルサンチマンの発生メカニズム

```
[近代化の理念的約束] → 自由・平等・富の普遍的実現
        ↓
[模倣の実行] → 制度・技術・価値観の導入
        ↓
[期待と現実の乖離] → 約束された繁栄が少数者にのみ帰属
        ↓
[屈辱感の蓄積] → 取り残された者の実存的怒り
        ↓
[ルサンチマンの結晶化] → 嫉妬 + 屈辱 + 無力感の複合感情
        ↓
[敵の特定] → エリート、移民、異教徒等のスケープゴート化
        ↓
[暴力的表出] → テロリズム、暴動、排外主義的政策
```

### 3. 「微差のナルシシズム」チェックリスト

対立する二者間に以下の特徴が見られるか確認する:

- [ ] 実際には相互に深く模倣し合っている
- [ ] 微小な差異を誇大に主張している
- [ ] 敵対者と自己の類似性を否認している
- [ ] 固有の文化的アイデンティティを「発明」している
- [ ] 敵の手法（メディア戦略、組織形態等）を借用している

### 4. 社会的怒りの多層分析

| 分析層 | 内容 | 指標 |
|--------|------|------|
| 物質的層 | 経済的格差と生活水準の停滞 | ジニ係数、実質賃金の推移、若年失業率 |
| 制度的層 | 民主主義の機能不全 | 投票率低下、議会への信頼度、腐敗認識指数 |
| 文化的層 | 伝統的価値観の崩壊と代替の不在 | 宗教帰属の変化、共同体への参加率 |
| 心理的層 | 個人の尊厳と自己評価の毀損 | 社会的信頼度、孤立感の調査データ |
| デジタル層 | SNSによる比較と怒りの増幅 | ヘイトスピーチの量的推移、エコーチェンバーの形成 |

## 適用場面

### ビジネス戦略への応用
- 新興国市場参入時のリスク評価（近代化への反発の予測）
- ブランド戦略における文化的感受性の分析
- 従業員エンゲージメントにおける「公正感」の設計

### 政策分析への応用
- ポピュリズム台頭の構造的要因の特定
- 移民政策における社会的受容性の評価
- 教育政策における「格差の見える化」の功罪

### 組織マネジメントへの応用
- 組織変革時の抵抗勢力の理解
- M&A後の文化統合における「模倣の欲望」の管理
- 多国籍チームにおける「微差のナルシシズム」の予防

## 注意事項

- ルサンチマンは「怒りの原因」ではなく「怒りの構造」である。具体的な経済・政治的要因の分析を省略してはならない。
- 単一の感情概念で複雑な社会現象を説明しきることはできない。補完的な分析フレームワークとの併用が必要。
- 分析者自身が「文明の衝突」という安易な図式に陥っていないか、常に自己点検すること。
