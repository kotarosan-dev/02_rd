# Skill: エバーグリーン型長寿戦略フレームワーク

## 概要

アンドリュー・スコット『ライフ・シフトの未来戦略』から抽出した、長寿化時代における個人・組織・社会の適応戦略を設計するためのフレームワーク。

## 1. エバーグリーン型の結論（三段論法）

長寿時代の行動原理を導く基本的な推論構造。

```
前提1: 私は高齢になるまで生きる可能性が高い
前提2: 長生きすることに関する最大の不安は、ひどい老い方をすることである
結論 : よい老い方をできる可能性を最大限高めるために、いまできることをするべきである
```

**適用場面**: 個人のライフプラン策定、組織の長期人材戦略、政策立案の基本前提として使用。

## 2. パラダイムシフト変換マトリクス

「高齢化社会」の言説を「長寿社会」の言説に変換するための対照表。

| 高齢化社会の視点 | 長寿社会の視点 |
|---|---|
| 高齢者の数が増えること（問題） | 若者が非常に高齢まで生きること（機会） |
| 人生の最終盤にばかり関心 | 人生全体の設計に関心 |
| 高齢者のニーズへの対処 | 全世代の長寿準備の支援 |
| シルバー・エコノミー | エバーグリーン・エコノミー |
| 老年従属人口指数（負荷） | 長寿化の配当（投資） |
| 老化 = 出来事・目的地 | 老化 = 継続的プロセス |
| 3ステージの人生 | マルチステージの人生 |

**適用場面**: 企業の事業戦略転換、行政の高齢者政策見直し、メディアのフレーミング改善。

## 3. レッドゾーン/グリーンゾーン分析

人生終盤の健康状態を可視化し、介入ポイントを特定するフレームワーク。

```
レッドゾーン = 不健康で虚弱な状態で生きる期間
グリーンゾーン = 健康で生産的に生きる期間

目標: レッドゾーンを圧縮し、グリーンゾーンを最大化する
```

**3層の介入レベル**:
1. **個人レベル**: 睡眠・運動・食事・社会的つながり・生き甲斐
2. **医療システムレベル**: 治療中心 → 予防中心への転換
3. **科学レベル**: 個別疾病対策 → 老化プロセスそのものへの介入

## 4. 長寿時代の資産ポートフォリオ

金融資産だけでなく、無形資産を含めた総合的な「人生のポートフォリオ」管理モデル。

```
金融資産: 貯蓄・投資・年金・保険
人的資本: スキル・知識・職業能力・生産性
健康資本: 身体的健康・精神的健康・生物学的年齢
社会資本: 人間関係・コミュニティ・社会的つながり
変容資本: 変化への適応力・学習能力・レジリエンス
```

**年次レビュー項目**: 各資産の現状評価、枯渇リスクの特定、補充投資の計画。

## 5. 4つの老い方シナリオ（文学モデル）

生物学的老化と寿命の関係を4パターンで整理するフレームワーク。

| シナリオ | モデル | 寿命 | 健康状態 | 評価 |
|---|---|---|---|---|
| ストラルドブラグ型 | 『ガリバー旅行記』 | 不死 | 老化は進行 | 最悪（レッドゾーン無限大） |
| ドリアン・グレイ型 | 『ドリアン・グレイの肖像』 | 通常 | 老化が遅延 | 最も価値が高く実現可能性も高い |
| ピーター・パン型 | 『ピーター・パン』 | 延長 | 老化が停止 | 理想的だが実現は遠い |
| ウルヴァリン型 | マーベル・コミック | 延長 | 再生・回復 | 究極の理想 |

**適用場面**: 老化科学の研究戦略の優先順位づけ、医療政策の目標設定。

## 6. 世代間公平性チェックリスト

長寿化政策が世代間の公平性を確保しているかを検証するための項目。

- [ ] 若い世代の生涯生産性が向上する仕組みがあるか
- [ ] 現在の若者が将来「歴史上最も健康な高齢者」になれる支援があるか
- [ ] 年金改革の負担が特定世代に偏っていないか
- [ ] 長寿リテラシー教育が若年層に提供されているか
- [ ] 世代を超えた交流の機会が制度的に設計されているか
- [ ] 政策決定プロセスに若い世代の声が反映されているか
