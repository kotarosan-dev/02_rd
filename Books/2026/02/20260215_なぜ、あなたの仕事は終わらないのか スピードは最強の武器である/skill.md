# ロケットスタート時間術

## 概要

元マイクロソフト・Windows 95アーキテクトの中島聡が提唱する時間管理メソッド。仕事の期限の最初の2割の期間に全力投球し、仕事量の8割を完了させることで「スラック（余裕）」を確保し、残りの8割の期間を仕上げと調整に充てる手法。核心は「速く終わらせる」ことではなく、「余裕を意図的に作り出す」ことにある。

## 使用場面（When to Use）

- 新しい仕事やプロジェクトを任されたとき
- 締め切りのある業務全般
- 複数の仕事を同時並行で進める必要があるとき
- 先延ばし癖を改善したいとき
- 仕事に追われている状態から、仕事を追う状態に転換したいとき

## フレームワーク/プロセス

### 基本原則：2:8の法則

```
期限全体を10とすると:
[==ロケットスタート期間(2)==][========流し期間(8)========]
  ここで仕事の8割を完了       残り2割の仕上げ＋他の業務
```

### ステップ1：見積もり申請

仕事を受けたら、まず「仕事が終わるかどうか判断したいので、2日間待っていただけませんか」と申し出る。この2日間が見積もり期間であり、同時にロケットスタート期間となる。

### ステップ2：ロケットスタート（最初の2割の期間）

- 全力で仕事に取り組み、8割を完了させることを目指す
- この期間はメール、電話、会議、SNSなど一切のマルチタスクを排除する
- 「界王拳」の比喩：自分の出せる最大能力を最初に投入する
- もし8割終わらなかった場合、それは「この仕事は期限内に終わらない」という重要な情報であり、早期に上司へ報告できる

### ステップ3：流し期間（残りの8割の期間）

- 残り2割の仕上げ作業を余裕を持って行う
- メール返信、打合せ、書類整理など付随業務をこの期間に消化する
- 次の仕事の準備や、スキルアップのための学習に充てる
- 仕事の質を高めるブラッシュアップもこの期間に行う

### ステップ4：1日の時間配分（流し期間中）

```
04:00-06:30  10倍界王拳（メイン業務に全集中、メール・電話不可）
06:30-08:00  朝食・散歩など
08:00-12:00  2倍界王拳（メイン業務の残り）
             ※午前中にメイン業務の8割を終わらせる
12:00-13:00  昼食＋昼寝（約18分）
13:00-22:00  流し（メール返信、打合せ、雑務、翌日のタスクリスト作成）
22:00-       就寝（翌朝の仕事を考えながら眠りにつく）
```

### ステップ5：タスクリスト管理

- 毎晩寝る前に翌日のタスクリストを作成する（所要時間：約5分）
- タスクは15分で終わる粒度まで分解する
- 紙のノートに手書きで、左端にチェックボックスを配置する
- 翌朝はリストに沿ってタスクを消していく（ゲーム感覚で）

## 具体的な実行手順

### 長期プロジェクト（1か月以上）の場合：「縦に切る」

1. 全体を3つの大きなフェーズに分割する
2. 各フェーズをさらに3つに分割する
3. 10日〜2週間の単位まで切り分ける
4. 各単位にロケットスタート時間術を適用する

### 複数業務の同時進行の場合：「横に切る」

1. 抱えている仕事の数で1日を分割する
2. 分割された各時間ブロックを「1日」と見なす
3. 各ブロック内でロケットスタートの原則を適用する
4. 最も重要な仕事を朝の界王拳タイムに配置する

### 他者の仕事が遅れている場合：「モックアップ」

1. 相手の成果物を待たず、仮のモックアップ（模型）を作成する
2. モックアップに合わせて自分の担当部分を先に進める
3. 相手の成果物が到着したら、接合部分だけ調整する

## 入力/出力

### 入力
- 仕事の内容と締め切り
- 仕事の分解可能な構成要素
- 現在の仕事の優先順位

### 出力
- 締め切り前に完了した仕事（余裕を持った状態）
- 正確な見積もり（2日間の全力投球に基づく実測値）
- 余剰時間（自己投資・スキルアップ・好きなことに使える時間）

## 注意点

- いきなり朝4時起きを目指さない。まず1時間早起きから始め、2か月かけて定着させる（習慣の定着には平均66日かかるという研究知見に基づく）
- 「流し」の期間に新しい仕事を入れない。スラックを守ることが最重要
- 睡眠時間は個人差があるため、無理な短眠は避ける。眠いときは仮眠を取る
- マルチタスクの排除が最も効果的な集中力向上策。メール即レスの習慣を見直す
- 8割終わらなかったときは「納期延長の相談」を早期に行う。これは失敗ではなく、見積もりの成果である
- この手法の最終目的は「仕事を速く終わらせること」ではなく、「好きなことに向き合う時間を確保すること」である
