## テーマ別構造マップ

### A. 創作技法論（本書を通底する技法テーマ）

```
ガボの核心的技法論
│
├── 人物論
│   ├── 欲望の一元化：主人公の欲望は一つに絞れ
│   ├── 動機の可視化：「なぜそう動くのか」を見せよ
│   └── 悪人の内的論理：悪人にも自分なりの正義がある
│
├── 構造論
│   ├── 葛藤の三層モデル
│   │   ├── 外的葛藤（主人公 vs. 他者・世界）
│   │   ├── 内的葛藤（主人公 vs. 自分自身）
│   │   └── 哲学的葛藤（物語が問う「どう生きるか」）
│   │
│   ├── 時間制約の活用（30分形式の意義）
│   └── フラッシュバック禁止令
│       └── 代替手法：現在に過去を溶かし込む
│
├── 視点論
│   ├── 視点 = イデオロギー
│   └── 視点選択 = 誰の痛みを感じさせるかの倫理選択
│
├── ジャンル論
│   ├── ジャンル = 観客との約束
│   └── 破るためにはまず守れ
│
└── ファンタジー論
    ├── 非現実設定の内的一貫性が命
    └── ルールを守ることで観客の信頼を得る
```

### B. 創作哲学論（第二部・エピローグで深化）

```
ガボの創作哲学
│
├── 才能論
│   ├── 才能は生まれつき
│   └── しかし技術なくして才能は開花しない
│
├── 書き直し論
│   ├── 第一稿 = 自分へのメモ
│   ├── 第二稿 = 信頼できる一人に見せる
│   └── 第三稿 = 世界に見せる
│
├── 物語衝動論
│   └── 才能の本質 = 語らずにはいられない衝動
│
└── 映像詩学
    ├── モンタージュ = 映画の詩法
    ├── カットの瞬間 = 観客の想像力に委ねる行為
    └── 映画的リアリティ = 記録ではなく感じられる真実
```

## 物語案の問題類型マップ

| 問題類型 | 典型例（物語名） | ガボの診断 |
|---------|----------------|-----------|
| 欲望の不明確さ | 傘の謎・限界の探求 | 「主人公が何を望んでいるか分からない」 |
| 動機の不一貫 | 土曜日の泥棒 | 「キャラクターがなぜそう動くのか不明」 |
| テーマの物語化失敗 | 限界の探求 | 「テーマだけあって人物がいない」 |
| フォアシャドウイング不足 | サマーラの死 | 「予言を信じる前提が作られていない」 |
| ジャンル混濁 | 第一バイオリン | 「コメディかスリラーか決まっていない」 |
| ファンタジー論理の破綻 | ある復讐の物語 | 「設定のルールが途中で変わる」 |
| キャラクターの平板化 | シダリアとベリンダ | 「対比が鮮明すぎて寓話になっている」 |
| 悪人の単純化 | ブラジル官僚の物語 | 「悪人の内的論理が描かれていない」 |

## キーコンセプト相関図

```
     ▲物語の強さ
     │
高   │              [傑作]
     │         欲望 × 葛藤 × 視点 × 一貫性
     │
     │    [良作]
     │  欲望 × 葛藤 × 視点
     │
     │  [弱作]
     │ 欲望 × 葛藤
     │
     │ [失敗作]
低   │ 欲望のみ（葛藤なし）
     │
     └──────────────────────────────▶ 技術の深さ
          低          →         高
     (テーマの宣言)  (物語の実現)
```
