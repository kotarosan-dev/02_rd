# structure-map

書名: ミッドナイト・ライブラリー

ジャンル判定: B ジャンルフィクション

構造の見方:

前半は「後悔の可視化」。

中盤は「別人生の比較実験」。

後半は「理想人生の破綻」から「元の人生の再選択」への反転である。

大きな流れ:

1. 現実人生の崩壊

2. 図書館ルールの理解

3. 後悔ごとの検証実験

4. 生きたい感情の再発見

5. 完璧な人生の不成立

6. 白紙の未来への帰還

## マクロ構造

| 幕 | 範囲 | 機能 |
| --- | --- | --- |
| 第一幕 | 現実崩壊から図書館到着まで | ノーラの認知がどこまで閉じているかを示す |
| 第二幕 | 後悔の書から初期の別人生群 | 未選択の未来の理想化を崩す |
| 第三幕 | 成功人生と極地人生 | 外的成功と生存本能の差を明らかにする |
| 第四幕 | ヒューゴ以降の多数の試行 | 可能性の横断が着地点を保証しないと示す |
| 第五幕 | アッシュとモリーの人生 | 愛を獲得しても主体性の欠如は残ると示す |
| 第六幕 | 図書館崩壊から帰還 | 生の再選択と未来の白紙性を回復する |

## 構造メモ

本作は「より良い人生」探しの反復で読者を引っ張る。

しかし実際の構造的重心は、反復そのものではなく、反復を通じた認知修正にある。

そのため別人生エピソードは、エピソード集ではなく、一つの誤解を砕く連鎖実験として読むと整理しやすい。

最終盤の白紙の本は、選択済みの人生群より強い。

なぜなら、未記入であること自体がノーラの新しい希望の定義だからである。
