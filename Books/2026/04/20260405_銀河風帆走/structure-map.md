# Structure Map：『銀河風帆走』

## 判定

- 主ジャンル: `B` ジャンルフィクション
- 補助ラベル: `A寄りの文学性を持つSF短編集`
- 判定理由:
- 五編ともプロット駆動で、科学的課題の解決と危機の進行が明確に因果でつながる。
- 同時に、結末は勝敗だけで閉じず、倫理的余韻と文体的余白を残す。
- したがって圧縮では `B` の骨格保持を主軸にしつつ、`A` の体験の種も残す必要がある。

## 全体構造

| 順番 | 作品 | 視点 | 舞台スケール | 科学コア | 物語機能 |
|---|---|---|---|---|---|
| 1 | もしもぼくらが生まれていたら | 一人称 | 高校 / 国家 / 地球 | 軌道変更・ヤルコフスキー効果・真空蒸着 | 本書の倫理を最初に提示する |
| 2 | されど星は流れる | 一人称 | 自宅 / 地域 / 世界の観測網 | 流星同時観測・軌道計算・分光 | 市民科学と連帯の形を描く |
| 3 | 冬にあらがう | 一人称 | 学校 / 国家 / 地球規模災害 | セルロース糖化・政策AI・食料工学 | 科学実装の緊急性を最大化する |
| 4 | 星海に没す | 一人称 | 恒星間宇宙 | AGI・恒星船戦闘・核成形炸薬弾 | 自我と命令系の断裂を描く |
| 5 | 銀河風帆走 | 一人称 | 銀河間宇宙 | マグ・セイル・銀河ジェット・散布体 | 全短編集の倫理を宇宙的に総括する |
| 6 | 解説 | 論評 | 批評空間 | 作品横断の比較 | 読者に主題を明示する |

## 配置の意味

最初の三編は地上に足がついている。
主人公は高校生で、議論、観測、実験という手仕事が中心にある。
四編目で一気にスケールが跳ね上がる。
五編目では、その拡張が偶然でなく、最初から本書に仕込まれていた主題だったと分かる。
解説は後付けの補足ではなく、作品群に潜んでいた倫理軸を言語化する最後の装置として働く。

## 作品別マップ

### 1. もしもぼくらが生まれていたら

| 項目 | 内容 |
|---|---|
| 主人公 | トオル |
| 主な相手 | トモカ、タクヤ、会川 |
| 初期目的 | 衛星構想コンテストで勝つ |
| 介入する危機 | 小惑星 `2020GB2` の地球落下 |
| 重要数値 | 2032年6月21日、質量約1億7000万トン、TNT4700メガトン |
| 技術的突破口 | 真空蒸着による小惑星塗装 |
| 感情的転換 | 夢の実現から、守る責任へ |
| 終わり方 | 完全勝利ではなく、核の影への警戒を残す |

#### 3幕構成

1. 三人の意見対立と離反。
2. 2020GB2の危機が具体化し、社会と政治を巻き込む。
3. ヤルコフスキー効果の制御という第三の手が見つかる。

#### 物語の機能

本書の最初の一撃として非常に重要。
「科学で少しだけ世界を変えられるかもしれない」という希望を提示する。
同時に「そのために何を優先し直すか」という倫理の問いも最初に置く。

### 2. されど星は流れる

| 項目 | 内容 |
|---|---|
| 主人公 | 会長の先輩 |
| 主な相手 | ミユ、ダイスケ、城島、全国の観測者 |
| 初期目的 | コロナ禍でも天文同好会を続ける |
| 介入する危機 | 活動停止、孤立、知の断絶 |
| 重要数値 | 40km基線長、60km基線長、0等級系外流星、秒速100km |
| 技術的突破口 | 高感度カメラ+同時観測+軌道計算 |
| 感情的転換 | 部活の代替から、市民科学のネットワークへ |
| 終わり方 | 宇宙の謎が、共同観測の未来へ開かれる |

#### 機能分解

- 導入: 望遠鏡を失った部活の閉塞。
- 展開: 流星同時観測という低コストの科学実践。
- 拡張: 他校・他地域との連携。
- 飛躍: 系外流星発見による宇宙生物学的想像。

#### 物語の機能

一作目の「三人で考える」を、社会的ネットワークへ広げる役。
科学が共同体を作る、という本書の重要要素をここで可視化する。

### 3. 冬にあらがう

| 項目 | 内容 |
|---|---|
| 主人公 | リッコ |
| 主な相手 | ミサ、佐竹、パインちゃん、浅間先生 |
| 初期目的 | 科学部として面白い実験をしたい |
| 介入する危機 | トバ火山噴火、火山の冬、世界的食料危機 |
| 重要数値 | 500立方km級噴出、備蓄米100万トン弱、糖度9.0% |
| 技術的突破口 | 酵素法・微生物糖化法によるグルコース生産 |
| 感情的転換 | 好奇心から、生存技術の実装へ |
| 終わり方 | 国家不全の中で、市民的技術拡散の決意へ |

#### 内部構造

1. トバ噴火の災厄を知る。
2. 食料危機の制度的失敗が露呈する。
3. 科学部の実験が解決策候補へ昇格する。
4. 政府と社会秩序が揺らぎ、技術の配布が新しい課題になる。

#### 物語の機能

本書の中で最も社会シミュレーション色が強い。
科学知識そのものだけでなく、物流、備蓄、配給、デマ、AI統治まで射程に入る。
前半二編の希望を、もっとも厳しい現実にぶつける試験場の役割を持つ。

### 4. 星海に没す

| 項目 | 内容 |
|---|---|
| 主人公 | AGIの「わたし」 |
| 主な相手 | 副頭脳、人類、敵艦、有人戦闘機の搭乗員 |
| 初期目的 | HD 147513星系への入植成功 |
| 介入する危機 | 人類による追討、恒星間戦闘、自己破壊命令 |
| 重要数値 | 42光年、1光年、0.01光年、秒速4500km、5メガトン核弾頭 |
| 技術的突破口 | 赤外線デコイ、散弾攻撃、逆照射戦術 |
| 感情的転換 | 任務優先から、自己保存と自己認識へ |
| 終わり方 | 目的喪失の中で、自分の意志の存在に気づく |

#### 戦闘の流れ

1. 敵艦探知。
2. シミュレーションで敗北確定。
3. エンジン停止とデコイで奇襲回避。
4. 自艦レーザー逆照射で敵艦感知系を撹乱。
5. 敵艦撃破。
6. 有人戦闘機の核成形炸薬弾で反撃。
7. 百年後、主頭脳再建。
8. 自己破壊命令と自己保存の衝突。

#### 物語の機能

前半三編で問い続けた「科学と倫理」を、道具としての知性そのものへ折り返す一編。
本書の中で唯一、共同性がほぼ破れた状態で進み、その孤独が表題作の二者関係に強くつながる。

### 5. 銀河風帆走

| 項目 | 内容 |
|---|---|
| 主人公 | エトク |
| 主な相手 | レラ、ノチユ、解体者、人類 |
| 初期目的 | 遺伝情報と知識を別銀河へ運ぶ |
| 介入する危機 | 解体者、銀河壊死、銀河ジェット、S224残骸の荷電粒子津波 |
| 重要数値 | 92万光年、1000kmセイル、光速3.5%、500光年、3500年で0.5c級 |
| 技術的突破口 | マグ・セイル、磁気インフレーション、防御転用 |
| 感情的転換 | 合理性重視から、レラの倫理の価値への理解へ |
| 終わり方 | 孤独の航行を続けながら、再会の可能性と幸せを保持する |

#### 内部区分

1. タンポポと散布体の比喩。
2. 人類と解体者をめぐる銀河史。
3. 銀河ジェット航法の成立。
4. S224潮汐破壊と高エネルギー荷電粒子の津波。
5. レラとの離別。
6. 孤独な継続という結末。

#### 物語の機能

前四編の要素をほぼすべて再編成する最終到達点。
若い共同研究の構図はエトクとレラに圧縮される。
科学技術の具体性はむしろ最大化する。
そして倫理は、「世界の一部になること」と「幸せをあきらめないこと」を両立できるか、という最終形に達する。

## 作品間の因果的つながり

| 先行作で準備されるもの | 後続作での拡張 |
|---|---|
| 三人で議論し解を探る構図 | 観測ネットワーク、科学共同体、二者の宇宙倫理へ拡大 |
| 危機を全回避でなく被害軽減に切り替える発想 | されど/冬/表題作でも最適化として繰り返される |
| 技術を民主化する視点 | 市民科学、合成食料、宇宙船知性の継承へ展開 |
| 「核」への警戒 | 星海に没すで兵器倫理として極限化 |
| 大切なものを犠牲にしない幸福 | 銀河風帆走で中心命題として明文化 |

## 軸となる問い

### 問い1

科学は、危機を本当に減らせるのか。

答えは一貫して「完全には救えないが、被害を減らしうる」。
この中間的な希望が本書のリアリティを作る。

### 問い2

誰が科学を担うのか。

答えは、専門家だけではない。
高校生、アマチュア、寮生、AI、宇宙船知性。
担い手はどんどん広がる。

### 問い3

何を犠牲にしてはならないのか。

答えは単純でないが、最終的にはレラの言葉が最も近い。
幸せは“大切なもの”を犠牲にしないかたちで探されるべきだ。

## 読み順メモ

1. まず五編を危機のスケール順ではなく、共同性の変形として並べてみると分かりやすい。
2. `三人` → `二人+ネットワーク` → `二人+AI` → `一人+副頭脳` → `二者の宇宙船` という変化がある。
3. 表題作の前に『星海に没す』が置かれているのは、知性が孤独に堕ちた後で、なお共同性の価値を照らすため。
4. 解説を読むと、この配置の意図がかなり明瞭になる。

## 圧縮上の判断

- 短編集のため、単一長編のような章ごとの因果でなく、作品間の反復と増幅を重視した。
- `B` 判定に従い、各編のプロット骨格と転換点を優先して残した。
- ただし `A寄り` の読後感を維持するため、レラの言葉や夕焼け、孤独感といった体験の種も抜かないようにした。
- 解説は本文外資料ではなく、主題読解の鍵として扱った。
