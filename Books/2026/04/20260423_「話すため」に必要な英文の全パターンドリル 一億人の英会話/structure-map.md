## 本書の思想的構造（2軸）

```
「配置のことば」英語のルール
│
├── 軸1: 基本文型の配置
│     「どの位置に置くかが意味を決める」
│     他動型（動+目）/ 自動型（動）/ 説明型（動+説）
│     授与型（動+目a+目b）/ 目的語説明型（動+目+説）
│
└── 軸2: 修飾の配置
      「後ろ → 説明（後置修飾）」
      「前  → 指定（前置修飾）」

この2軸を理解すると学校文法の膨大な「規則」が：
・関係代名詞・分詞構文・同格節        → すべて「説明ルール」に統合
・否定文・助動詞・形容詞修飾・副詞位置 → すべて「指定ルール」に統合
```
