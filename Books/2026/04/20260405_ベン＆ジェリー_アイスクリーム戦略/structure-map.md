# Structure Map: 『ベン＆ジェリー アイスクリーム戦略』

## 1. 全体構造
この本は「創業物語」から始まり、「価値主導」の概念化を経て、それを各経営機能へ埋め込む実験記録へ展開し、最後に社会監査という制度化へ着地する。

```text
序章
→ 第1章 概念定義
→ 第2-6章 機能別実装
→ 第7章 政治的拡張
→ 第8章 国際化での翻訳問題
→ 第9章 制度化と未来
→ 付録 社会監査の定量化
→ 訳者あとがき 日本への読み替え
```

## 2. 章ごとの役割
| セクション | 役割 | 一行要約 |
|---|---|---|
| 序章 | 起源の提示 | 偶然と失敗から価値主導の土壌が生まれる |
| 第1章 | 概念定義 | 価値主導ビジネスは日常業務への価値観統合である |
| 第2章 | 購買 | 最大の社会レバーは寄付でなく仕入れ |
| 第3章 | 財務 | 価格・株式・投資・寄付率にも価値観を埋め込む |
| 第4章 | 小売・提携 | 店舗は地域社会との接点であり社会装置になる |
| 第5章 | マーケティング | 行動が広告になる。関係性がブランドを作る |
| 第6章 | 人材 | 理念企業ほど社内制度の強さが必要 |
| 第7章 | 政治 | 企業は既に政治的存在であり、公共へ責任を持つ |
| 第8章 | 国際化 | 価値主導は文脈翻訳なしには輸出できない |
| 第9章 | 未来 | 社会監査で理念を管理対象へ変える |
| 付録 | 指標化 | 社会成果を数える第二の決算の設計 |
| 訳者あとがき | 日本化 | 日本企業社会への問題提起として読み替える |

## 3. 中核命題の流れ
| 段階 | 命題 | 支える事例 |
|---|---|---|
| 起点 | 企業は楽しさと人間性を失うと続かない | 創業時の「楽しくなければやらない」 |
| 定義 | 価値主導とは社会的使命を本業へ埋め込むこと | 熱帯雨林クランチ |
| 実装1 | 購買が最大の変化レバー | グレイストン、アズテック、家族農園 |
| 実装2 | 財務も中立ではない | 州民向け株式公開、社会的投資 |
| 実装3 | 店舗とブランドは社会関係でできる | CDF、提携店、予防接種企画 |
| 実装4 | 社員制度が弱いと理念が空回りする | 賃金比率論争、社員調査 |
| 拡張 | 企業は政治を避けられない | 平和ポップス、銃規制、軍事費批判 |
| 境界 | 海外では価値観の翻訳が必要 | 英国・ロシア・フランス・イスラエル |
| 制度化 | 測定しなければ維持できない | 社会監査、付録の指標群 |

## 4. 反復する対立軸
| 対立軸 | 本書の答え |
|---|---|
| 利益 vs 理念 | 二者択一ではなく、三重使命の同時最適を目指す |
| 寄付 vs 本業 | 寄付より本業の購買・投資・採用が重要 |
| 広告 vs 行動 | 行動が広告になる設計が強い |
| 社外の善意 vs 社内の公正 | 社外だけ立派でも社内制度が弱いと崩れる |
| 国内成功 vs 海外複製 | 文脈翻訳なしの移植は失敗しやすい |
| 理想主義 vs 管理可能性 | 社会監査で理想を測定可能な対象にする |

| 領域 | 問い | 典型事例 |
|---|---|---|
| 商品開発 | 何を売るか | 熱帯雨林クランチ |

| 財務 | 誰から資金を集め、どこへ置くか | 直接公開、社会的投資 |

| 政治 | どの公共課題へ声を出すか | 1% for Peace、銃規制 |
| 国際 | 価値観をどう翻訳するか | 英国、フランス、ロシア |
| 管理 | 何を測るか | 社会監査、付録指標 |

## 7. 読み落としてはいけないポイント
- 本書はCSRの教科書ではなく、会社の通常業務を社会装置として見直す本である。
- 価値主導の中心は寄付ではなく、購買、資本、雇用、政治参加にある。
- 成功事例だけでなく、社内摩擦や品質問題まで含めて価値主導のコストを書いている。
- 付録は退屈に見えるが、実は本編の理念を測定可能にする鍵である。
- 訳者あとがきは、日本社会へこの思想をどう移植するかの導入になっている。

## 8. 最短理解ルート
最短で全体像をつかむなら、`序章 → 第1章 → 第2章 → 第5章 → 第7章 → 第9章 → 付録` の順で読むとよい。
このルートで、起源、定義、最大レバー、ブランド論、政治哲学、制度化、測定の流れがつながる。

## 9. 圧縮後に残すべき問い
1. 自社の最大の社会的インパクトは寄付ではなくどの通常業務にあるか。
2. 顧客は味や価格だけでなく、どの価値観に投票しているのか。
3. 社外に誇る理念と、社内制度の公平性は釣り合っているか。
4. 会社が政治に関わらないと言うとき、実際には誰の利益に黙って加担しているか。
5. 自社は利益以外に何を第二の決算として測るべきか。
