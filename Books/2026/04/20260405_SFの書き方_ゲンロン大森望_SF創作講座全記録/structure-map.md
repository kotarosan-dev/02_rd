# Structure Map:『SFの書き方 「ゲンロン 大森望 SF創作講座」全記録』

## 1. 全体構造
この本の構造は「理論集」ではなく「講座運営の時系列」に沿っている。
ただし内容面では、次の4層に分けて読むと整理しやすい。

1. 講座の思想と場の設計
2. 各回のテーマ別創作技法
3. 受講生梗概・実作への講評
4. 作家化・公開・デビューへの現実ルート

## 3. 進行パターン
本書は毎回ほぼ同じリズムで進む。

```text
テーマ提示
→ 講師講義
→ 課題文
→ 梗概例
→ 講評レポート
→ 実作例または選出作
→ 次回テーマで視点が更新される
```

この反復の効果は大きい。
読者は講義内容を単独で消費せず、すぐ次に「そのテーマで受講生が何を書いたか」を見られる。
そのため、抽象論と実例の距離が短い。

## 6. 本書の反復モチーフ
- SFは定義ではなく運用で捉える。
- 驚きは論理に支えられていなければ弱い。
- 梗概は雰囲気ではなく設計図である。
- 設定はテーマのためにある。
- 文章は情報を運ぶだけでなく、読む速度を変える。
- 読者に入ってくる扉を一つ作る。
- 古典的骨格を恐れず、現代のディテールで更新する。
- 公開の場に出ること自体が学習を早める。

## 7. 主要な緊張関係
| 緊張関係 | どこで現れるか | 読みのポイント |
|---|---|---|
| 自由 vs 制約 | 法月、冲方、課題設計 | ルールは縛りでも武器でもある |
| 奇想 vs 論理 | 長谷、東、山田 | 変なだけでは弱く、整いすぎても驚きが死ぬ |
| 説明 vs 余白 | 冲方、円城、新井 | 伝えることと残すことの配分が要る |
| 個性 vs 読者配慮 | 新井、山田、大森 | 自分らしさを消さずに入口を作る |
| 作品論 vs 進路論 | 実作篇、付録 | 書くことと売ることは別だが連続している |
| 講義 vs 実例 | 全体構成 | 理論の有効性が即座に試される |

## 9. 一文要約の連鎖
この本は、SFを書くための本である。
しかし同時に、SFを書く人をどう育てるかの本でもある。
さらに、育てる場をどう編集するかの本でもある。
講義録であり、作品集であり、編集論であり、デビュー戦略書でもある。
この多層性が、本書を単純な創作ハウツー以上のものにしている。

## 10. 圧縮時に残すべき核
1. 受講そのものを公開実戦とみなす講座思想。
2. SFを狭く定義しないが、論理を伴う驚きは要求するという前提。
3. 梗概を設計図として扱う実務感覚。
4. 各講師が別方向から「問いの立て方」を教えていること。
5. 実作篇と付録によって、作品化と作家化の現実まで繋いでいること。
