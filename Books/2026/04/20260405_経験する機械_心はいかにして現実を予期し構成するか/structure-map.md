# Structure Map: 『経験する機械』

## 基本メタ

- 書名: 経験する機械 心はいかにして現実を予期し構成するか
- 著者: アンディ・クラーク
- 判定ジャンル: D（学術書・思想書）
- 中心テーゼ: 心は、感覚入力を受け取ってから世界像を作るのではなく、世界と身体を予測し、その誤差を調整することで経験を構成する。
- サブテーゼ: 知覚と行動は同一原理の両面であり、情動・病理・拡張認知・介入可能性も同じ地図で読める。

## 全体アーキテクチャ

## 論証チェーン

1. 錯覚は知覚の失敗例ではなく、知覚の通常設計を露出する。
2. 通常設計とは、生成モデルが先に予測を出し、感覚的証拠がその誤差を返す構造である。
3. 誤差だけでは足りず、何を重く扱うかを決める適合率が必要になる。
4. 適合率の配分は注意、信頼、期待、文脈、学習履歴と深く結びつく。
5. この図式は知覚だけでなく、慢性疼痛、機能性障害、妄想、PTSDの説明にも流用できる。
6. 同じく、行動は未来状態の予測を自己充足化する能動的推論として理解できる。
7. さらに脳は外界だけでなく体内状態も予測し、身体予算を管理する。
8. そのため情動や気分は、身体シグナルの受信結果ではなく予測的編成の一部になる。
9. 意識のハードプロブレムは残るが、経験形成の輪郭は予測論でかなり説明できる。
10. バイアス修正、VR、プラシーボ、瞑想などは、適合率と予測習慣の再設計として理解できる。
11. 外部ツールや社会環境も、予測誤差を減らす安定ループなら心の構成要素たりうる。
12. 結果として心は、脳内装置というより、身体と世界に開かれた予測の生態系になる。

## 概念フロー

```text
生成モデル
  ↓ 予測を出す
予測
  ↓ 感覚と照合
予測誤差
  ↓ どちらを重く扱うか
適合率
  ↓
知覚 / 行動 / 情動 / 病理 / 学習 / 外部資源利用
```

## 主要対立軸

| 対立軸 | 旧来モデル | 本書のモデル |
|--------|------------|--------------|
| 知覚 | 入力受信から世界像を構築 | 予測先行、誤差修正で世界像を維持 |
| 行動 | 命令実行 | 未来予測の自己充足 |
| 痛み | 損傷の単純反映 | 予測と身体シグナルの相互作用 |
| 情動 | 反応・感情の出力 | 身体予測の編成結果 |
| 心の境界 | 脳内に限定 | 身体・道具・世界へ拡張可能 |

## 重要な転換点

1. 第1章で「錯覚」が例外から基本原理へ格上げされる。
2. 第2章で「症状」も同じ原理で説明できると示される。
3. 第3章で知覚理論が行動理論へ反転する。
4. 第4章で外界予測が身体内部予測へ拡張される。
5. 第6章で心の境界が頭蓋から外へ破られる。
6. 第7章で理論が介入実践の設計図として読み替えられる。

## 圧縮時に保持すべき核

- 知覚は「統制された幻覚」であるが、現実抵抗に縛られる。
- 適合率は、本書全体の隠れた主役である。
- 病理は正常機構の失敗ではなく、正常機構の偏りとして説明される。
- 行動は予測に沿って世界を変えることだ。
- 情動は身体予測の政治学でもある。
- 心は外部資源を巻き込みうる。
- ハック可能性は希望だが、倫理的含意も大きい。

## 再読優先順

1. 理論だけ押さえるなら: 補遺 → 第1章 → 結論
2. 臨床的関心が強いなら: 第2章 → 第4章 → 第7章
3. 哲学的関心が強いなら: 間奏 → 第6章 → 結論
4. 実践に引き寄せるなら: 第5章 → 第7章 → 第3章

## 一行要約集

- はじめに: 心は現実を受け取る前に、かなりの部分を先に描いている。
- 第1章: 錯覚は壊れた知覚ではなく、知覚の設計図である。
- 第2章: 痛みも妄想も、予測の重みづけの乱れとして読める。
- 第3章: 行動とは、予測された未来を現実化することだ。
- 第4章: 感情は身体予算をめぐる予測の感じられた姿である。
- 間奏: 意識の謎は残るが、経験形成の手がかりは増えた。
- 第5章: 予測は誤るが、鍛え直すこともできる。
- 第6章: 心はスマホやノートや社会的足場のなかへ伸びていく。
- 第7章: 期待・儀式・没入・瞑想は予測機械の設定変更として働く。
- 結論: 人間の心は世界に開いた予測ループの束である。

## この本の効能

- 認知科学の複数領域を一つの言語で見渡せる。
- 精神症状を道徳化せず、機構的に捉えやすくなる。
- 身体・環境・技術を含む広い意味での心を考えられる。
- ただし、万能理論として呑み込まず、各領域の実証強度を見分ける必要がある。
