# Structure Map: 『新版 財務3表図解分析法』

## 1. 本書の骨格
本書は「財務分析の考え方」→「図解比較の実戦」→「自分で作図する方法」→「補論」という4層構造でできている。
したがって、読む順番を間違えると、図の意味だけ分からない、または指標だけ暗記する、という状態になりやすい。
正しい読み順は `概念` → `比較事例` → `再現手順` → `精密化` である。

```text
導入
  ↓
第1章: 4指標 + CS8パターン
  ↓
第2章: 企業比較で戦略を読む
  ↓
第3章: 自分で図を作る
  ↓
附章: 粉飾 / 指標補足 / 投資家指標 / 定義差
```

## 3. 概念依存マップ
### 第1層: 最小単位
- お金を集める
- 投資する
- 利益をあげる
- 現金が増減する

### 第2層: 財務3表との対応
- `BS右側` = お金の集め方
- `BS左側` = 何に投資したか
- `PL` = 利益化の結果
- `CS` = 現金の動き

### 第3層: 分析指標
- `ROE`
- `財務レバレッジ`
- `総資本回転率`
- `当期純利益率`
- `営業CF / 投資CF / 財務CF`

### 第4層: 読み取れる経営現象
- 安全性
- 収益性
- 成長性
- 再生
- 集中化
- 多角化
- M&A拡大
- 株主重視
- 長期投資志向

## 6. 第2章の内部構造
第2章は単なる事例集ではなく、論理順に並んでいる。

1. まず `図の読み方` を教える。
2. 次に `同業比較・期間比較` の基本形を見せる。
3. その後 `再生`、`戦略`、`Policy` と、読める意味を深くしていく。
4. `CS分析` で現金面から裏を取る。
5. `世界比較` と `GAFA` で時代構造まで視野を広げる。

```text
読み方
  ↓
比較の基本
  ↓
再生
  ↓
戦略
  ↓
Policy
  ↓
CSで裏取り
  ↓
世界市場
  ↓
知識社会
```

## 9. キーワードの接続図
`ROE` は単独で読むと危険だが、次の接続で意味が立ち上がる。

```text
ROE
 ├─ 財務レバレッジ → 資金調達姿勢
 ├─ 総資本回転率   → 資産効率
 └─ 当期純利益率   → 収益構造

CS
 ├─ 営業CF → 稼ぐ力
 ├─ 投資CF → 将来への賭け方
 └─ 財務CF → 借入・返済・株主還元
```
