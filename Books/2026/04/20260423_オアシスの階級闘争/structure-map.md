## 全体構成の見取り図

```
序（はじめに）
│ └ テーゼ: 批評的に対象を真剣に取り上げることの宣言
│
├── 序章「新自由主義の孤独の中で、アンセムを歌ったオアシス」
│    └ 主命題: 「冷笑主義への反駁」と「ラジカルなポジティブさ」の提示
│
├── 第1章「地（Earth）」
│    └ 社会的土台: マンチェスターの歴史・コラージュ文化・共同体バンド論
│
├── 第2章「水（Water）」
│    └ 拡張と希望: 海洋的感覚・グランジ/シューゲイザー影響・〈Live Forever〉分析
│
├── 第3章「火（Fire）」
│    └ 反骨と快楽主義: パンクの継承・孤独の詩学・アルコールの政治性
│
├── 第4章「空気（Air）」
│    └ 政治的空白と蒸発: ジョン・スミス→ブレア問題・上昇と堕落の弁証法
│
├── 追記
│    └ 〈Don't Look Back in Anger〉と遺産
│
└── 解説（北村紗衣）
     └ 日本語版独自: 《Morning Glory》派の視点・ジェンダー批評
```

## 論証の連鎖（章間の接続）

| 接続 | 内容 |
|------|------|
| 序章 → 1章 | 「ラジカルなポジティブさ」の社会的基盤を「地」（マンチェスター・労働者階級）に求める |
| 1章 → 2章 | コラージュ文化論が「海洋的感覚」論に接続。グランジとの交差点でプロダクション分析へ |
| 2章 → 3章 | 楽観主義の頂点〈Live Forever〉から、その影の部分（孤独・反骨・快楽主義）へ |
| 3章 → 4章 | 「火」が燃え尽きることなく「空気」に溶けていく——「蒸発」概念の布石 |
| 4章 → 追記 | 「蒸発」が完了した後に何が残るか——民衆記憶としての遺産論へ |

## 主要概念の射程図

```
サッチャリズム（1979〜1990年）
│
├── 工業都市の破壊（マンチェスター・失業300万人超）
│    └──────────────────────────────── 「地」
│
├── 海洋的感覚（グローバリゼーション・歴史の終わり）
│    └──────────────────────────────── 「水」
│
├── 労働者階級の怒り・快楽主義への逃避
│    └──────────────────────────────── 「火」
│
└── 政治的空白（スミス→ブレア 1994〜1997年）
     └──────────────────────────────── 「空気」

↓ これらが交差する地点に《Definitely Maybe》（1994年）が存在する

ラジカルなポジティブさ
= コラージュ文化（過去の再利用）+ 海洋的感覚（連帯への渇望）
+ パンク的反骨（体制への怒り）+ 政治的可能性（スミス的瞬間）
```

## 弁証法的構造（著者の中心的論理）

```
テーゼ（正）: 《Definitely Maybe》は労働者階級の希望と連帯の純粋な体現だった

アンチテーゼ（反）: 成功はその本物性を必然的に「蒸発」させた
                  ノエルとブレアの握手がその象徴

ジンテーゼ（合）: 「蒸発」後も《Definitely Maybe》の音楽には
                「忘れられたかけら」が残存している
                → 批評的救済（ベンヤミン的）の可能性
```

この弁証法は本書全体を貫く論理的骨格であり、序章（テーゼ提示）→1〜3章（内実の充填）→4章（アンチテーゼの展開）→追記（ジンテーゼの示唆）という構成に対応している。

## ジャンル判定根拠

本書を **D（学術書・思想書）** と判定した理由:

1. **構造的論証**: 各章が「テーゼ→証拠→反論→結論」の学術的論証構造を持つ
2. **広範な学術参照**: ベンヤミン・フロイト・マルクスなど哲学的参照系が体系的
3. **先行研究との明示的対話**: ハリス、ハザリー、マーク・フィッシャーへの明示的言及と批判的対話
4. **概念装置の構築**: 「ラジカルなポジティブさ」「蒸発」「コラージュ文化」など分析概念の明示的定義
5. **音楽批評を超えた社会思想**: サッチャリズム・新自由主義・階級理論の骨格上に音楽分析を組み込む

ただし「文化批評エッセイ」の要素も強く（F-ジャンル的な主観的声調）、D/Fの境界領域に位置する。
