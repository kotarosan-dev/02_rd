# Structure Map: 時の子供たち 上

## 判定
- ジャンル: `B: Genre Fiction`
- 中心駆動: 文明進化、異種知性、宇宙漂流、ファーストコンタクト前夜
- 中心問い:
- 人類は本当に未来の継承者なのか。
- 知性はどの身体からでも立ち上がりうるのか。
- 宗教と科学は異文明の中でどう絡み合うのか。
- 生き延びることと、文明として前進することは同じなのか。

## 全体骨格
- 第1部: 創造の失敗
- 第2部: 二千年後の再接近
- 第3部: 戦争と墜落
- 第4部: 疫病と発信

## 時間構造
- 人間側の時間: 冷凍睡眠により断続する、飛び飛びの時間。
- 蜘蛛側の時間: 世代継承により連続する、蓄積の時間。
- 読者体験: 人間にとっての「失われた二千年」を、蜘蛛側の文明史として読む。

## 視点構造
- 人間視点: ホルステン、レイン、グイエン、反乱派を中心とする閉鎖系ドラマ。
- 蜘蛛視点: ポーシャ、ビアンカ、ヴァイオラらを通じた文明成長ドラマ。
- 機械/亡霊視点: カーン衛星の声が、両者のあいだに過去の呪いとして割り込む。

## フェーズ別マップ

### フェーズ1: 未来の設計と破綻
- 範囲: 第1部前半
- 要約: カーンは猿を文明化しようとするが、サリングの破壊工作で計画は崩壊する。
- 重要性: 人類中心の未来設計が失敗し、偶然の進化へ主導権が移る起点。
- 伏線:
- カーンの人格が衛星に残ること。
- ナノウイルスが別種へ流れること。
- 「猿」が将来の主役にならないこと。

### フェーズ2: 小さな知性の点火
- 範囲: 第1部後半
- 要約: ポーシャたち蜘蛛が、協力・計画・空間把握を備えた新しい知性として立ち上がる。
- 重要性: 文明の萌芽が身体の小ささとは無関係に成立することを示す。
- 伏線:
- 共同狩猟。
- 記号的コミュニケーション。
- 世代継承される知の形式。

### フェーズ3: 人類の帰還と門前払い
- 範囲: 第2部前半
- 要約: `ギルガメシュ` が二千年後に到着するが、カーン衛星は彼らを拒絶する。
- 重要性: 人類が自分たちの遺産からさえ正統後継者と認められない。
- 伏線:
- ホルステンの翻訳能力。
- グイエンの強権。
- カーンの「わたしの世界」という所有観。

### フェーズ4: 蜘蛛文明の都市化
- 範囲: 第2部後半
- 要約: 蜘蛛たちは都市、農業、工学、信仰、研究を発展させ、完全な文明圏を築く。
- 重要性: 彼女らが単なる変異動物ではなく、独自文明の担い手になったと確定する。
- 伏線:
- 大きな巣。
- 聖堂。
- 〈使徒〉信仰。
- 蟻の利用と対立。

### フェーズ5: 人類内部の分裂
- 範囲: 第2部末から第3部前半
- 要約: 月コロニー送りをめぐる不満が反乱へ発展し、人類は内戦状態へ入る。
- 重要性: 人類は外敵と出会う前に、自分たちの時間制度に壊されている。
- 伏線:
- 眠る者と眠れない者の格差。
- 反乱派の切迫。
- レインとホルステンの拘束。

### フェーズ6: 蟻との総力戦
- 範囲: 第3部中盤
- 要約: 蜘蛛文明は蟻の大群との非対称戦へ入り、社会制度そのものの限界が問われる。
- 重要性: 上昇する文明にも脆弱性があることが示される。
- 伏線:
- 指揮系統の弱さ。
- ビアンカの化学技術。
- 宗教的解釈が動員力に変わること。

### フェーズ7: 墜落としてのファーストコンタクト
- 範囲: 第3部後半
- 要約: 反乱派のシャトルが惑星へ降下し、人間と蜘蛛がついに物理的に接触する。
- 重要性: 題材としての「接触」が、対等な対話ではなく誤読の連鎖として始まる。
- 伏線:
- 人間は蜘蛛を怪物視する。
- 蜘蛛は人間を天から来た巨人として読む。
- ポーシャは相手を殺すより解読しようとする。

### フェーズ8: 疫病と内的革命
- 範囲: 第4部前半
- 要約: 蜘蛛文明は都市化の副作用として疫病に襲われ、治療と免疫研究を進める。
- 重要性: 文明が次段階へ進むには、外敵だけでなく内的問題の解決が必要だと示す。
- 伏線:
- 近親化の弊害。
- ヴァイオラの研究。
- 〈理解〉の再定義。

### フェーズ9: 発信の成立
- 範囲: 第4部後半
- 要約: 蜘蛛たちは大人同士で〈理解〉を伝達しうる可能性に達し、ついに〈使徒〉へ通信する。
- 重要性: 上巻のクライマックス。蜘蛛文明が宇宙に向けて主体的に名乗りを上げる。
- 到達点:
- 治療法の成立。
- 新たな知識経済の萌芽。
- 「わたしたちはここにいる」という文明宣言。

## 章別索引

### 第1部
- `1.1 樽いっぱいの猿たち`: カーンの創造計画とサリングの破壊工作。
- `1.2 いさましいちびの狩猟者`: ポーシャの知的狩猟と協力の萌芽。
- `1.3 消える明かり`: 支配者不在のあとに残る進化の自由。

### 第2部
- `2.1 故郷から二千年`: `ギルガメシュ` 到着と人類の老い。
- `2.2 地球の別の子供たち`: 蜘蛛社会の共同体化。
- `2.3 謎の変奏曲`: カーン衛星との不気味な交信。
- `2.4 劣る者たち`: 価値転倒と人類の自己否定。
- `2.5 これらの世界はすべてあなたたちのもの`: 所有と排除のメッセージ。
- `2.6 巨大都市`: 文明としての大きな巣の成立。
- `2.7 脱出`: 人間社会の破綻と反乱。

### 第3部
- `3.1 突然の目覚め`: ホルステン拘束下での再覚醒。
- `3.2 火と剣`: 蜘蛛側の戦争本格化。
- `3.3 進退きわまる`: 反乱派の怒りと時間的不平等。
- `3.4 西の海のほとりで`: 大きな巣の防衛と敗北感。
- `3.5 燃える剣を帯びて`: シャトル脱出と地表行き。
- `3.6 そは甘美で名誉なり`: 蜘蛛社会制度と戦時の限界。
- `3.7 天界での戦い`: 軌道上の三つ巴。
- `3.8 非対称の戦争`: 化学兵器と決死の潜入。
- `3.9 最初の接触`: 人類の墜落着陸。
- `3.10 大地の巨人たち`: 蜘蛛側から見た人間。
- `3.11 この島グラーグ`: 月コロニー計画の固定化。
- `3.12 荒れ野で叫ぶ声`: まだ言語にならない呼びかけ。

### 第4部
- `4.1 不思議の洞窟`: 残骸・設備・人工空間の解読。
- `4.2 死神の到来`: 疫病の蔓延。
- `4.3 灰色の惑星からの手記`: 人間側の調査と自己認識。
- `4.4 探究心`: 危機下の科学。
- `4.5 古代人の夢`: 遺産の意味をめぐる視差。
- `4.6 内なる〈使徒〉`: 免疫と〈理解〉の再発見。
- `4.7 ハムレット王子ではない`: ホルステンの位置取り。
- `4.8 進歩の時代`: 通信革命と文明宣言。

## 人物アーク
- `ホルステン`: 翻訳者 -> 捕虜 -> 接触の媒介者候補
- `レイン`: 技術担当 -> 権力の板挟み -> 地表生還者
- `グイエン`: 存続戦略の指導者 -> 選別の執行者
- `ポーシャ`: 狩猟者 -> 戦士 -> 司祭科学者 -> 通信主体
- `ビアンカ`: 異端研究者 -> 戦時技術者 -> 医学革命の共作者
- `カーン`: 計画者 -> 亡霊 -> 門番

## 主要対比
- 人類の巨大技術 / 蜘蛛の精密技術
- 冷凍睡眠の断続時間 / 世代継承の連続時間
- 所有の論理 / 居住と適応の論理
- 旧文明の再利用 / 新文明の発明
- 命令系統の合理性 / 流動社会の創造性
- 宇宙からの視線 / 地表から編み上げる視線

## 伏線と回収
- カーンの「猿」計画 -> 蜘蛛が真の後継者になる逆説で回収。
- 〈使徒〉信仰 -> 実際には人工衛星との通信へ接続。
- 小さな狩猟知性 -> 都市、戦争、医学、通信まで拡張。
- 人間の時間格差 -> 反乱と月コロニー問題で噴出。
- 古帝国遺産への依存 -> 人類の停滞感を強める。
- 「わたしたちはここにいる」 -> 上巻全体の文明成長の結晶。

## 最終到達点
- 人間はまだ未来の所有者だと考えている。
- 蜘蛛はすでに未来の発明者になっている。
- 上巻はこの非対称を確定させたうえで、両者の本格衝突と交渉を下巻へ送る。
