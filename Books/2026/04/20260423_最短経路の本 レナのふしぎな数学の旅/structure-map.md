## アルゴリズムの計算量比較表

| アルゴリズム | 問題 | 計算量 | 最適性 |
|---|---|---|---|
| ダイクストラ法 | 単一始点最短経路（非負重み） | O(n²) | 最適 |
| ベルマン・フォード法 | 単一始点最短経路（負重み可） | O(nm) | 最適 |
| ワーシャル・フロイド法 | 全対最短経路 | O(n³) | 最適 |
| プリム法 | 最小全域木 | O(n²) | 最適 |
| クルスカル法 | 最小全域木 | O(m log m) | 最適 |
| オイラー路構成 | 全辺一度通過 | O(m) | 最適 |
| 中国の郵便配達員（無向） | 全辺最小閉路 | O(m²) | 最適 |
| 中国の郵便配達員（有向） | 全辺最小閉路（有向） | O(m²) | 最適 |
| 最隣接ヒューリスティック | TSP | O(n²) | 非最適（誤差無制限） |
| クリストファイズ法 | TSP（三角不等式条件下） | O(n³) | 1.5倍近似保証 |
| 分枝限定法 | TSP・組合せ最適化全般 | 最悪 O(n!) | 最適（実際は高速）|
| ブラッサム（マッチング） | 一般マッチング問題 | O(n³) | 最適 |
