# Structure Map: 『新版 財務3表一体理解法 発展編』

## 判定
- 主ジャンル: `C`（ビジネス書・自己啓発）
- 補助タグ: `finance/accounting`, `G-like decision support`
- ねらい: 会計基準の暗記ではなく、財務3表連動で制度の意味を読む

## 本書の主質問
1. 会社の「正しい姿」は、なぜ現金の出入りだけでは表せないのか。
2. 新会計基準は、どのリスクや価値変動を前倒しで見せるのか。
3. 純資産の部は、誰を守り、何を制限し、何を柔軟にするのか。
4. 組織再編や事業再生では、どの数字が動き、どの数字は見かけだけ変わるのか。
5. IFRS・連結・英文会計を前にしても、何が不変の骨格なのか。

## コア概念マップ

### 1. 財務3表一体理解法
- すべての会社は `資金調達 -> 投資 -> 収益化`
- BS右側 = 調達
- BS左側 = 投資
- PL = 収益化の成果
- CS = 現金版の流れ

### 2. 新会計基準の共通原理
- 退職給付会計: 将来負担の前倒し
- 貸倒引当金: 回収不能リスクの前倒し
- 時価会計: 市場価値変動の反映
- 減損会計: 回収可能性低下の反映
- 税効果会計: 会計と税法のタイミング差の調整

### 3. 純資産の分解
- 資本: 株主が入れた元本
- 留保利益: 会社が稼いで残した利益
- 準備金: 債権者保護 + 将来欠損へのクッション
- 自己株式: 純資産の控除項目

### 4. 再編会計の読み筋
- 合併: BS足し算
- 買収: 株式取得 or 合併対価
- のれん: 支払超過プレミアム
- DES: 負債を資本へ変換
- 無償減資: 見た目改善、現金増加なし

## 章内の因果連鎖
| 起点 | 中間処理 | 結果 |
|---|---|---|
| 将来退職金支払 | 引当金計上 | 利益圧縮 + 負債見える化 |
| 売掛金回収懸念 | 貸倒引当金 | 実態純資産の保守化 |
| 市場価格下落 | 時価評価 | PLまたはOCIで価値反映 |
| 固定資産の稼ぐ力低下 | 減損 | 帳簿価額引下げ |
| 課税タイミング差 | 税効果会計 | 繰延税金資産計上 |
| 債務の重さ | DES / 債権放棄 | 負債減少、ただし税・持分問題発生 |
| 欠損累積 | 減資 / 欠損てん補 | 純資産表示改善、現金増加なし |
| 買収価格が純資産超過 | のれん | 日本基準とIFRS差が拡大 |

## Cジャンル要件への対応
- コアフレームワーク: 3活動、5つのつながり、新会計基準共通原理、純資産分解、再編判断木
- 章別エッセンス: はじめに、第1章〜第5章、附章、おわりにを網羅
- 批判的視座: 実装コスト、見積り依存、制度と実務のギャップを補記
- 実践チェックリスト: 20項目超
- よくある落とし穴: 14項目
- 原文引用: 3本

## G補助視点で残したもの
- いつPLが動き、いつCSが動かないか
- どの場面で `のれん` / `DES` / `無償減資` を使うか
- 企業比較でIFRS差がどこに効くか
- 全部連結と持分法で売上・利益の見え方がどう変わるか

## 再読ガイド
- 会計初心者が再読すべき順: 第1章 -> 第2章 -> 第3章
- 経営企画・財務担当が再読すべき順: 第3章 -> 第4章 -> 第5章

- 事業再生案件を読む人が再読すべき順: 第2章(税効果) -> 第4章(実例)
