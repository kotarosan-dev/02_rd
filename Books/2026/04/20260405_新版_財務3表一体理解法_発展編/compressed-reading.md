## 読書ナビゲーション
この本は、簿記の細かい仕訳暗記から会計を理解するのではなく、PL・BS・CSのつながりを軸にして、2000年以降の新会計基準、純資産の部、組織再編、IFRS、連結、英文会計まで一気に見通すための発展編である。
第1章は基礎編の要点を圧縮し、「なぜ財務3表一体理解法で会計がわかるのか」を再確認する導入になっている。
第2章は退職給付、貸倒引当金、時価会計、減損、自己株式、法人税、税効果会計を、1取引ごとに4表がどう動くかで理解させる実践章である。
第3章は本書の核心で、資本金・資本準備金・利益準備金・繰越利益剰余金の意味を、株主保護ではなく最終的には債権者保護という視点で整理する。
第4章はM&A、のれん、会社分割、株式交換、DES、減資、欠損てん補を、三菱自動車・シャープ・東芝の実例までつなげて、会計が企業再生の言語でもあることを示す。
第5章はIFRSと日本基準の差を、読む側に必要な論点だけに絞って整理し、最後に連結の範囲・相殺消去・全部連結と持分法連結を説明する。
附章は英文会計の名称差と構造差を押さえ、「英語だから別物」ではなく「同じ仕組みの別表記」だと認識を矯正する。
したがって本書の読み方は、会計基準の百科事典としてではなく、「会社の正しい姿をどう表すか」「数字が動いても現金は動くのか」「どの数字が誰を守るために存在するのか」という3つの問いを持ちながら辿るのがよい。

## コアフレームワーク

### 1. すべての会社を貫く3活動と4表
本書の全体を貫く最小モデルは、どの企業も「お金を集める」「投資する」「利益をあげる」の3活動しかしていない、という整理である。
この3活動が、BS右側・BS左側・PLに対応する。
さらに現金の流れを切り出したものがCSである。
著者はここに直接法CSと間接法CSを並べることで、実質的に「4表」で理解する。

```text
Financing -> Investing -> Earning
   |             |            |
 BS右側        BS左側        PL
   \_____________|____________/
             CS
```

- `BS右側`: どうやって資金を集めたか。資本金、借入金、各種純資産。
- `BS左側`: 集めた資金を何に投じたか。現金、在庫、固定資産、株式、繰延税金資産。
- `PL`: その投資からどう利益を上げたか。売上、費用、利益。
- `直接法CS`: 実際の現金収支。
- `間接法CS`: 利益から逆算して現金に戻す表。

この整理の強みは、簿記ルールの細部を知らなくても、数字が「どこから来て、どこへ行ったか」を追跡できる点にある。
新会計基準も、純資産論も、組織再編も、全部この骨組みの上に載る。

### 2. 財務3表一体理解法の5つのつながり
著者が繰り返すのは、個々の勘定科目を暗記することではなく、表と表の接点を見ることだ。
その接点は5つに整理される。

| つながり | 内容 | 読む時の要点 |
|---|---|---|
| A | PLの当期純利益とBSの繰越利益剰余金 | 利益はBS右側を増減させる |
| B | BS左右の一致 | どんな処理でもバランスは崩れない |
| C | BSの現金とCS期末残高 | 現金残高は最終的に整合する |
| D | PL税引前当期純利益と間接法CS起点 | 非資金項目の足し戻しが見える |
| E | 直接法CS営業CFと間接法CS営業CF | 計算経路は違っても結果は同じ |

この5つを毎回点検すれば、複雑そうに見える会計処理も「利益が動いたか」「現金が動いたか」「負債か純資産のどこで受けたか」の3点に還元される。
本書のドリルは、各お題でまずPLを見る、次にBSを見る、次に直接法CS、最後に間接法CSを見る順番を徹底している。
ここが、会計を「記帳技術」ではなく「構造理解」に変える。

### 3. 新会計基準を読むための共通判断式
第2章の各論は別々に見えるが、共通原理はひとつしかない。
それは「会社の正しい姿を表すために、見えにくいリスクや価値変動を表に乗せる」というものだ。

| 論点 | 何を見える化するか | PL | BS | CS |
|---|---|---|---|---|
| 退職給付会計 | 将来支払う退職金の今期負担 | 費用化 | 引当金計上 | 現金動かず |
| 貸倒引当金 | 売掛金の回収不能リスク | 費用化 | 売掛金控除 | 現金動かず |
| 時価会計 | 市場価格変動 | 評価損またはOCI | 有価証券評価替え | 現金動かず |
| 減損会計 | 固定資産の回収不能 | 特別損失 | 帳簿価額引下げ | 現金動かず |
| 自己株式 | 自己資本の縮小 | 原則PL影響なし | 純資産マイナス | 財務CF減少 |
| 税効果会計 | 税法と会計のタイミング差 | 法人税等調整額 | 繰延税金資産 | 現金動かず |

ここで大事なのは、`PLに費用が立つ = 現金が出た` ではないことだ。
むしろ新会計基準ほど、PLは動くのにCSは動かない処理が多い。
だからこそ間接法CSの足し戻し項目が理解の要になる。
著者は、退職給付引当金増加、貸倒引当金増加、有価証券評価損、減損損失などを、非資金損益として一つずつ戻しながら、会計利益と現金収支のズレを可視化する。

### 4. 純資産の部を解く基本図
第3章の到達点は、純資産の部を「細かい科目の集合」ではなく、`資本` と `留保利益` に分解して理解することにある。

```text
株主資本
├─ 資本
│  ├─ 資本金
│  └─ 資本剰余金
│     ├─ 資本準備金
│     └─ その他資本剰余金
└─ 留保利益
   ├─ 利益剰余金
   │  ├─ 利益準備金
   │  └─ その他利益剰余金
   │     ├─ 任意積立金
   │     └─ 繰越利益剰余金
   └─ 自己株式（控除）
```

- `資本金`: 基本的に配当できない。債権者保護のための余裕分。
- `資本準備金`: 資本の中のクッション。減資や欠損てん補の柔軟性を持たせる。
- `利益準備金`: 利益から強制的に積む安全弁。
- `繰越利益剰余金`: 稼いだ利益の蓄積。配当原資の中核。
- `その他資本剰余金`: 減資や自己株式処分差額の受け皿。
- `自己株式`: 自分で自分を食べている状態をマイナス表示する。

著者の説明で重要なのは、「株主保護よりも、資本金維持の真の理由は債権者保護である」という逆転である。
帳簿価額通りに資産が換金できるとは限らない。
だから、資本金は債権者へ返すべき財産を確保するための余裕分として、配当制限の対象になる。

### 5. 組織再編とIFRSを読む判断木
後半章では、会計を制度ではなく意思決定の道具として使う。
その際の判断は、次のように整理できる。

| 状況 | まず見るもの | 主論点 | 注意点 |
|---|---|---|---|
| 他社を買う | 純資産時価と買収価格差 | のれん | 日本は償却、IFRSは非償却 |
| 借金を軽くする | 債権放棄かDESか | 債務免除益 / 資本化 | 債務免除益課税に注意 |
| 欠損解消したい | 無償減資・準備金取崩し | 欠損てん補 | 現金は増えない |
| 連結範囲を判断する | 支配か影響か | 子会社/関連会社 | 直接保有だけでなく間接保有を見る |
| 企業比較をする | 会計基準差異 | のれん・リース・収益認識 | 時系列比較が歪む |

G的に言い換えると、本書は「何を覚えるか」より「どの局面でどの表を見るか」を教える本である。
会計に強い人とは、科目名を全部知っている人ではなく、数字の変化を見て「これは現金なのか、評価なのか、法制度上の振替なのか」を即座に仕分けられる人である。

## 章別エッセンス

### はじめに
**核心命題**: 本書は旧『増補改訂 財務3表一体理解法』を2分冊化した下巻であり、基礎編だけでは残るモヤモヤを解消するための発展編である。対象は、複式簿記会計の全体像をつかんだあとに、新会計基準、純資産の部、組織再編、IFRSに踏み込みたい読者だ。
**事例サマリー**: 2007年の初版、2016年の増補改訂を経て、2021年版では「退職給付会計」「時価会計」「減損会計」「自己株式」「税効果会計」「IFRS」「連結」「英文会計」までを扱う構成に整理し直している。
**行動指針**: 基礎編を終えた読者は、第3章を最優先で読む価値が高い。純資産の部が腹落ちすると、会計への恐怖が大きく下がる。
本書全体の態度は一貫していて、会計専門家が実務詳細を語る本ではなく、専門家ではない人が「なぜその科目が必要なのか」を理解する本である。
したがって、ここでの学び方も、条文や厳密定義の丸暗記ではない。
「そもそも資本金があるのに、なぜ資本準備金が要るのか」
「なぜ株主から注入された資本は自由に配れないのか」
「なぜ貸倒引当金や退職給付引当金を、まだ払っていないのに費用化するのか」
こうした素朴な問いを潰していくことが読書の中心になる。
また、後半では実在企業として三菱自動車、シャープ、東芝が登場し、会計が抽象理論ではなく企業運命を左右することが見えてくる。
新会計基準の説明だけで終わらず、M&Aや事業再生まで接続するのがこの本の強みである。
「発展編」という名前に反して、目的は難しくすることではなく、会計の奥にある一貫性を見せることにある。

### 第1章 序
**核心命題**: 「財務3表一体理解法」が効く理由は、すべての会社に共通する3活動を先に見せ、その後でPL・BS・直接法CS・間接法CSの4表を一体で追うからである。簿記の仕訳を先に覚えなくても、構造から理解できる。
**事例サマリー**: 会社の活動は「お金を集める→投資する→利益をあげる」の3つだけで、BS右側・BS左側・PLに対応する。CSは営業・投資・財務に分かれ、この3活動の現金版になっている。
**行動指針**: 取引を見る順番を固定する。PLを見る、BSを見る、直接法CSを見る、間接法CSを見る、という順番で毎回確認する。
第1章では、PLを収支計算書だと誤解することが、初心者の最大のつまずきだと指摘する。
売掛や買掛、減価償却費がPLに入るのは、現金収支ではなく「正しい利益」を出すためである。
BSは集めたお金と投資先のストック表、CSは現金のフロー表であって、それぞれ役割が違う。
ここで著者は、複式簿記の試算表を上下に切るとBSとPLになる、という説明も入れている。
つまりPLとBSは、もともと試算表という同じ母体から来ている。
一方、CSは単式簿記的な現金収支の系譜を引く。
だから財務3表一体理解法とは、複式簿記が作るPL・BSに、私たちが直感的に理解しやすい現金収支表を並べる技法だといえる。
新会計基準編に入る前に、この地ならしをしている点が重要である。
ここを飛ばすと後の「非資金損益の足し戻し」が何をしているのか見えなくなる。
逆にここが腹落ちすると、会計処理を「現金か非現金か」「利益を変えるか変えないか」で仕分けられるようになる。

### 第2章 財務3表一体理解法 新会計基準編
**核心命題**: 2000年以降の新会計基準は、基本を壊したのではなく、「会社の正しい姿」をもっと前倒しで、もっと正確に開示するために導入された。したがって理解のコツは、新ルールを暗記することではなく、各処理が4表のどこをどう動かすかを追うことである。
**事例サマリー**: 退職給付費用5万円ではPL費用化、BS固定負債に退職給付引当金5万円、CSは不変。貸倒引当金10万円では売掛金500万円に対してマイナス10万円を置き、PL費用化、CS不変。時価会計では売買目的有価証券10万円、投資有価証券20万円、関係会社株式30万円を取得し、その後10→6、20→15、30→20へ下落させ、売買目的はPLの有価証券評価損4万円、その他有価証券は純資産のその他有価証券評価差額金マイナス5万円、関係会社株式は簿価据え置きという対比で説明する。
**行動指針**: 新会計基準を学ぶときは、「何をリスクとして前倒し認識しているか」「CSが動かないのにPLが動く理由は何か」を必ず言語化する。
退職給付会計は、退職金を支払った年だけの費用にすると、在籍期間との対応が崩れるから、将来支払いを期間按分する。
貸倒引当金は、売上高を直接減らすのではなく、費用として処理することで営業利益以下を押し下げる。
時価会計では、有価証券の保有目的ごとに、PLに入れるか純資産直入れにするかが分かれる。
減損会計では、機械装置の価値は将来キャッシュフローで決まるというDCF的な発想が前面に出る。
自己株式の取得では、現金50万円が減り、純資産の「自己株式」にマイナス50万円が立つだけで、PLは原則動かない。
税法に基づく法人税計上では、税引前当期純利益720万円に貸倒引当金10万円を足し戻して課税所得730万円、税率30%で法人税等219万円という数値例が示される。
さらに税効果会計では、会計上あるべき税額216万円との3万円差を「法人税等調整額マイナス3」と「繰延税金資産3」で表す。
この章の白眉は、りそな銀行2003年国有化の意味を、繰延税金資産を積めるかどうかという文脈で説明するくだりである。
将来の課税所得見込みがなければ繰延税金資産は積めず、純資産を膨らませることもできない。
つまり税効果会計は単なるテクニカル論点ではなく、自己資本比率や銀行存続に直結する。
また、東芝の後の事例に接続する伏線として、税効果会計が将来見積り依存である点もここで効いてくる。
本章全体を通じて、PLに出る数字と現金のズレをCSで戻す感覚が身につけば、複雑な会計用語への恐怖はかなり薄くなる。

### 第3章 純資産の部の徹底理解
**核心命題**: 会計理解の最終関門は純資産の部であり、これを「資本」と「留保利益」に分け、さらに配当制限と債権者保護の観点で再配置すると霧が晴れる。資本金・資本準備金・利益準備金・繰越利益剰余金は、単なる欄の違いではなく、誰を守り何を可能にするかが違う。
**事例サマリー**: 出資額のうち2分の1超を資本準備金にできる。利益配当時には配当額の10分の1を利益準備金へ積む必要があるが、資本準備金と利益準備金の合計が資本金の4分の1を超えれば追加積立て不要。資本金5億円超で大会社、資本金1億円超で税務管轄や税率への影響が出る。トヨタ自動車は総資本約25兆円に対し資本金約4000億円で、総資本比1%未満という例も挙げられる。
**行動指針**: 純資産の部を見るときは、まず「株主から入った元本か」「会社が稼いだ蓄積か」を分け、その後に「配当できるか」「欠損てん補に使えるか」を確認する。
著者は、資本金を配当できない真の理由は株主保護ではなく債権者保護だと明言する。
帳簿価額通りに資産が換金できる保証がない以上、資本は債権者への返済余力として維持されるべきだ、という論理である。
資本準備金は、出資額のうち資本金に入れない部分を、勝手に配当されないよう資本として保持しつつ、将来の欠損てん補に柔軟性を持たせるための仕組みである。
利益準備金は、配当時に利益の一部を安全弁として留める制度であり、こちらはより明確に債権者保護目的を持つ。
その他資本剰余金は、減資や自己株式処分差額の受け皿になる。
自己株式を帳簿価額より高く売却すれば差額がその他資本剰余金に入り、低く売却すればまずその他資本剰余金、それでも足りなければ繰越利益剰余金を減らす。
その他利益剰余金では、任意積立金と繰越利益剰余金の役割が分かれる。
欠損てん補のために資本を利益へ振り替える処理は、現金が動かない単なる表示上の変化だが、会社が資本を食いつぶした事実を見せる意味を持つ。
2009年会社法改正で、利益剰余金を資本金へ振り替えることも可能になった点も、資本と利益の境界が絶対固定ではなく、株主総会決議を通じて再設計可能であることを示す。
Coffee Breakでは、資本金減少には特別決議と債権者保護手続き、準備金減少による欠損てん補には普通決議で足り債権者保護手続き不要、といった実務差まで押さえる。
本章を通読すると、株主資本等変動計算書が、単なる付表ではなく会社法と会計の接点だと見えてくる。

### 第4章 組織再編に係る会計について
**核心命題**: 組織再編会計の理解は、仕訳暗記よりも、BSがどう組み替わるかを見る方が速い。合併、買収、のれん、会社分割、株式交換・移転、DES、減資、欠損てん補は、すべて「誰の負債が消え、誰の資本が増え、現金が動くか」で解ける。
**事例サマリー**: A社80・負債30・資本金50、B社現金30・負債10・資本金20の例で、吸収合併では資本金70、買収ではA社の資産側にB社株式20が立つ。B社を20ではなく40で買えば差額20がのれん。DESでは負債20を株式化し、A社負債30→10、資本金50→70。三菱自動車では2014年に資本金約6250億円を資本剰余金へ振替え、総額約9241億円を欠損てん補に充当。シャープでは2015年度に2250億円増資のうち2000億円がみずほ・三菱東京UFJによるDES。東芝では2016年3月期にのれん減損約2950億円、繰延税金資産取崩し2517億円、東芝メディカルシステムズ株式売却収入6384億円が重要数字として出る。
**行動指針**: 事業再生案件を見るときは、「財務諸表上の見た目改善」と「現金増加」を分けて考える。無償減資や欠損てん補は後者を生まない。
合併と買収の違いは、旧会社が消えるか、株主だけ入れ替わるかである。
のれんは「形のない資産」そのものというより、時価純資産を超えて払ったプレミアムの資産計上だと理解する方が実務感覚に近い。
日本基準では20年以内に規則償却、IFRSや米国基準では非償却・減損テスト中心という差が、M&A戦略にも影響する。
会社分割は、切り出した事業の純資産に対応する新会社株式が元会社の資産に立つ処理として理解すると迷いにくい。
株式交換・株式移転は、実体会社のBSを変えるより、株主持分の帰属先を変える手続きである。
新株予約権は、発行時に純資産の新株予約権として入り、行使時に資本金化される。
債権放棄は借金を減らせるが、債務免除益が立って税負担を招く可能性がある。
DESは、負債を資本へ置き換えて返済義務を軽くしつつ、債権者に株式という残存価値を残す方法である。
有償減資は現金払い戻しを伴うが、無償減資は欠損てん補に使われ、現金は動かない。
三菱自動車のケースでは、無償減資と優先株式の買戻し・消却によって、欠損解消後に普通株主への配当再開が可能になる過程が見える。
シャープのケースでは、DESと減資で財務諸表を整えても営業CFがマイナス189億円で、事業そのものが現金を生めない限り再生は完結しないことが示される。
東芝のケースでは、のれん減損と繰延税金資産取崩しが連鎖し、将来性ある事業売却まで追い込まれる。
本章の含意は明快で、会計は経営の結果を後追い表示するだけでなく、再生余地や破綻危機を先に告げるセンサーでもある。

### 第5章 国際会計基準（IFRS）と連結会計について
**核心命題**: 読む側にとって重要なのは、日本基準とIFRSの細部差を全部覚えることではなく、企業比較を誤らせる大きな差を押さえること、そして連結を前提に会社を見ることだ。
**事例サマリー**: 大きな差は3つで、のれん非償却、収益認識基準、リース。収益認識は2018年開始IFRSの新基準と、日本でも2021年4月以後開始事業年度からの強制適用が紹介される。リースでは、IFRSで本社ビル等の家賃相当を使用権資産として資産計上するケースが出る。連結範囲はおおむね50%超で子会社、20-50%で関連会社だが、支配力・影響力基準も加味する。全部連結と持分法連結の例では、持株40%会社の純利益280万円に対し、親会社帰属分112万円が全部連結でも持分法でも最終的に親会社利益として見える。
**行動指針**: 企業比較では、会計基準差で利益率や総資産回転率が歪む可能性を常に疑う。とくにのれん償却と使用権資産計上は、PL・BS両方の見え方を変える。
包括利益の説明も本章の重要論点である。
当期純利益だけではなく、その他有価証券評価差額金、為替換算調整、退職給付関連調整など、資本取引以外の純資産変動全部をまとめて包括利益と考える。
ここには「会社の事業利益だけでなく、会社価値の変動まで見よう」というIFRS的発想がある。
日本の個別BSと連結BSが泣き別れになり、連結の方がIFRSに近い表示を取っている理由も、配当可能額との調整が必要な個別と、グループ実態を見る連結の役割差から説明される。
連結プロセスでは、投資と子会社純資産の相殺、親子間貸付の相殺、売上仕入の相殺、配当相殺、未実現利益消去が基本になる。
持株比率の判定でも、親会社と子会社を一体として間接保有を考える点が重要で、P社70%子会社C社がM社60%保有なら、P社もM社を子会社として扱う。
全部連結は関係会社の資産負債を取り込み、非支配株主持分を切り分ける。
持分法連結は、関係会社の利益のうち親会社持分だけを「持分法による投資利益」として取り込み、関係会社株式の帳簿価額もその分だけ増やす。
この違いを知らないと、売上規模と利益規模の比較で誤読する。
たとえば売上が連結に全部乗るのか、投資利益1行だけで入るのかで、PLの印象はまったく変わる。
本章は、会計基準の国際化とグループ経営の進展で、「単体会社を見る」だけでは不十分になったことを教えている。

### 附章 英文会計の基礎知識
**核心命題**: 英文会計は別体系ではない。Income statement、Balance sheet、Cash flow statement、Statement of stockholders' equity は、日本で学んだPL・BS・CS・株主資本等変動計算書と同じ構造を、英語の用語で表示しているだけである。
**事例サマリー**: Income statement には多段階形式と無段階形式があり、Gross profit、Operating income、Income before income tax、Net income という4段階利益が示される。Balance sheet では Stockholders' equity や Shareholders' equity を使い、固定ではなく Non-current assets / liabilities と呼ぶことが多い。Cash flow statement は米国基準で Net income 起点、日本は税引前当期純利益起点。Statement of stockholders' equity では Common stock、Retained earnings、Dividends declared、Common stock issued が並ぶ。
**行動指針**: 海外企業の財務諸表を見るときは、用語差に怯えず、どれがPL・BS・CS・純資産変動表かをまず対応づける。その上で、のれん処理や間接法CSの起点など、少数の差分だけ意識する。
著者は、複式簿記が15世紀ベニス商人由来で世界に広がった以上、基本構造が世界共通なのは当然だと説く。
この言い方は、英語アレルギーを外すのに有効である。
GrossとNetの説明も実務的で、Gross salesから値引き・割戻しを引いたものがNet salesであり、財務諸表の売上高は普通Net salesだと整理される。
米国のIncome statementには経常利益概念がなく、特別利益・特別損失も一般に営業外損益側や別表示で処理される。
Balance sheet では流動性重視ゆえにCurrent / Non-currentの切り方が強い。
Cash flow statement では、米国の間接法がNet income起点、日本が税引前当期純利益起点である差が、利息・税金支払の見せ方にも反映される。
したがって、英語だから本質が違うのではなく、開示慣行とラベルが違うだけである。
英文会計を読む入口として、本書の附章は十分に機能する。
とくに海外IR資料に初めて触れるビジネスパーソンには、恐怖心を消す導入として価値が高い。

### おわりに
**核心命題**: この本のゴールは、会計専門家になることではなく、会計を恐れないビジネスパーソンになることだ。わからない箇所があれば専門家に質問できるだけの地図を持つことが、著者のいう「理解」である。
**事例サマリー**: 著者は次の学習先として『財務3表実践活用法』と『できる人になるための「財務3表」』を挙げ、前者で投資評価・企業価値評価・予算管理、後者で金融商品・減損・研究開発費・リース・税効果会計などの深化を示す。
**行動指針**: 本書を読み終えたら、次にするべきは難解論点へ無限に進むことではなく、自社や関心企業のPL・BS・CS・株主資本等変動計算書を実際に読み、どこが本書のフレームで説明できるか試すことである。
会計の学習は、知識蓄積と同じくらい「見慣れること」が重要である。
本書はそのための言語を与える。
著者自身が会計専門家ではない立場から書いていることも、読者にとって大きい。
専門外の人が何につまずくかを熟知しているからだ。
「一流のビジネスパーソンになるためには会計の知識は不可欠」という締めは、精神論ではなく、ここまでの全章で裏付けられている。
銀行の国有化、M&Aのプレミアム、優先株の償還、欠損てん補、のれん減損、リース資産化。
どれも会計を知らないと、ニュースの意味が半分しか読めない。
本書はその断絶を埋める本である。

> 「新しい会計基準だからといって、基本が変わったわけではありません。これらが採用された主な理由はすべて同じです。それは、『新版 財務3表一体理解法』でも繰り返し述べたように、『会社の正しい姿を表すため』です。」
第2章の核心。個別基準を覚える前に、この一文を軸に置くと、引当金・時価・減損・税効果が全部つながる。

第3章の宣言。この本が純資産論に大きく紙幅を割く理由を端的に示している。

> 「それは、純資産の部の科目に入っている数字は現金ではないということです。資本金を取り崩して欠損をてん補するといっても、それは現金が動いているわけではありません。」
第3章と第4章を貫く注意点。減資や欠損てん補を「お金が湧いた」と誤解しないためのガードレールになる。

## 批判的視座
- **新会計基準の正当化は強いが、導入コストの議論は薄い**: 「会社の正しい姿」を示す意義は繰り返されるが、見積り依存が増えることによる恣意性や、実務負荷の増大への批判は深くは扱わない。
- **制度設計の背景説明は巧みだが、国際比較の定量性は限定的**: IFRS差異は要点に絞られている一方、実際にどの業種でどの指標がどれほど歪むかのケース比較は少ない。
- **純資産論は説得的だが、会社法実務の細部は簡略化されている**: 著者自身も実務家向け詳細ではないと断っており、精密運用は専門書が必要になる。
- **実例の選定は印象的だが、成功例・失敗例のバランスはやや偏る**: 三菱自動車、シャープ、東芝はいずれも大企業のドラマチックな例で、中堅企業の地味なケースは少ない。
- **会計を構造理解へ変える力は大きいが、税務・法務との境界管理は読者側に委ねられる**: とくに税効果会計や欠損てん補は、理解したつもりで実務判断すると危険である。

## 用語・前提の整理
- **PL**: 損益計算書。正しい利益を計算する表であって、現金収支表ではない。
- **BS**: 貸借対照表。集めたお金と投資先の残高を示す表。
- **CS**: キャッシュフロー計算書。現金の出入りを営業・投資・財務に分けた表。
- **退職給付引当金**: 将来支払う退職金のうち今期負担分を表す固定負債。
- **貸倒引当金**: 売掛金の回収不能見込み分を示す控除項目。
- **その他有価証券評価差額金**: 投資有価証券などの時価変動を純資産直入れした部分。
- **繰延税金資産**: 将来の税額減少につながる一時差異を資産化したもの。
- **のれん**: 買収価格が時価純資産を上回る差額。
- **DES**: Debt Equity Swap。借入金などの負債を株式へ転換する再生手法。
- **包括利益**: 資本取引を除く純資産変動全部。当期純利益 + その他の包括利益。
- **全部連結**: 関係会社の資産負債を取り込み、非支配株主持分を分ける方法。
- **持分法連結**: 関係会社利益のうち親会社持分のみを投資利益として取り込む方法。

## 実践チェックリスト
□ 会社を見るとき、まず「お金を集める・投資する・利益をあげる」の3活動に分解する  
□ PLを収支計算書だと思わず、利益計算の表だと意識する  
□ 現金が動かないのにPLが動く項目を、非資金損益として見分ける  
□ 間接法CSの足し戻し項目を見て、PLと現金のズレを読む  
□ 貸倒引当金を売上減少ではなく、費用化による利益圧縮として理解する  
□ 退職給付会計を、将来支払の期間按分として捉える  
□ 時価会計では、保有目的によってPL計上か純資産直入れかを確認する  
□ 減損会計では、将来キャッシュフロー低下が価値低下の根拠だと押さえる  
□ 自己株式取得を、ROE改善と純資産縮小の両面から読む  
□ 法人税等と法人税等調整額を分けて見る  
□ 繰延税金資産は「将来税金を減額してもらえる権利」だと説明できるようにする  
□ 純資産の部を「資本」と「留保利益」に分けて読む  
□ 資本金を配当不可にしている真因が債権者保護だと理解する  
□ 資本準備金が経営の柔軟性を担うクッションだと押さえる  
□ 欠損てん補は現金創出ではなく、計数振替だと確認する  
□ 組織再編を見るときは、まずBSがどう組み替わるかを見る  
□ のれんはプレミアムの資産計上であり、日本は償却・IFRSは非償却と区別する  
□ 債権放棄では債務免除益課税の可能性を忘れない  
□ DESは借金を株式へ替える手法だと一言で説明できるようにする  
□ 無償減資・欠損てん補で財務諸表が改善しても、営業CFが改善したとは限らない  
□ 連結では親子間取引を相殺消去する前提で数字を見る  
□ 持分法連結では売上が乗らず、投資利益だけが乗るケースを見分ける  
□ 英文会計は別体系ではなく、用語差にすぎないと理解する

## よくある落とし穴
1. **PLを現金収支表だと思う**: 売掛や減価償却費が入る時点でPLは現金表ではない。
2. **BS純資産の数字を現金だと思う**: 資本金や欠損てん補は現金移動ではなく計数の振替が多い。
3. **貸倒引当金で売上高を直接減らすと誤解する**: 実際には費用計上で利益を下げる。
4. **時価会計は全部PLに入ると思う**: その他有価証券は純資産直入れ、関係会社株式は原則簿価維持である。
5. **減損会計を時価会計の固定資産版だと単純化しすぎる**: 上げる評価はせず、回収可能価額低下への片道処理である。
6. **自己株式取得を単なる株価対策とだけ見る**: 実際には純資産を減らし、資本維持との緊張関係を持つ。
7. **税効果会計を節税策と誤解する**: 目的は会計と税法のタイミング差を埋める表示調整である。
8. **繰延税金資産を将来必ず現金回収できる資産と思う**: 実際は将来黒字が出て税金を払う前提が要る。
9. **減資で会社が元気になると錯覚する**: 無償減資は見た目を整えるだけで現金を増やさない。
10. **債権放棄は無条件で得だと考える**: 債務免除益に税金がかかりうる。
11. **DESは現金調達と同じだと考える**: 返済義務を消すが、新しい現金が入るとは限らない。
12. **IFRS移行後の利益増を経営改善と早合点する**: のれん非償却やリース資産化など、会計基準差が大きい。
13. **連結売上と持分法利益を同列比較する**: 取り込み方が違うので収益規模の見え方が変わる。
14. **英語表記の違いを本質差だと思う**: 名称差と構造差を分けて読まないと無駄に怖くなる。

| 第2章(1) | 新会計基準は「会社の正しい姿」を見せるための前倒し認識 |
| 第2章(2)-退職給付 | 将来退職金を費用と引当金で今期へ配分 |
| 第2章(2)-貸倒引当金 | 売掛金の回収不能見込みを費用と資産控除で表現 |
| 第2章(2)-時価会計 | 保有目的ごとにPL計上・純資産直入れ・簿価維持を使い分ける |
| 第2章(2)-減損 | 固定資産の回収可能性低下を一括で表に出す |
| 第2章(2)-自己株式 | 現金流出と純資産控除で自己資本を縮める |
| 第2章(2)-法人税 | 課税所得と税引前利益の差を意識して税額を計上 |
| 第2章(2)-税効果 | 会計と税法のタイミング差を調整し、繰延税金資産へ接続 |
| 第3章 | 純資産の部を資本と留保利益に分解し、配当制限の意味を再定義 |
| 第4章(1) | 合併・買収・のれん・分割・株式交換・DES・減資をBSで理解 |
| 第4章(2)-三菱自動車 | 無償減資と欠損てん補、優先株整理による財務正常化 |
| 第4章(2)-シャープ | DESと減資で見た目を整えても営業CF不足は解決しない |
| 第4章(2)-東芝 | のれん減損と繰延税金資産取崩しが資産売却を迫る |
| 第5章(1) | IFRSとの差は、のれん・収益認識・リースの3点を押さえる |
| 第5章(2) | 連結は支配の範囲、相殺消去、全部連結と持分法の違いが核心 |
| 附章 | 英文会計は用語差こそあれ構造は同じであることを確認 |
