## 下巻 章別構造

### 【信仰】

```
ヴィクター・シュワルツマン（ラビ、74歳）
 └─ 中心命題：「tzaddik（義人）」——善は死後も伝達される
 └─ 核心の言葉：「あなたが誰かに与えたものは残る。それが永遠です」

トーマス・ショー司教（聖公会）
 └─ 中心命題：信仰は恐怖を消さない。恐怖と共に生きる勇気を与える
 └─ 背景：AIDS患者の臨終に繰り返し立ち会う

ジョージ・ライアン大司教（カトリック）
 └─ 中心命題：「死は招かれざる客。来るのは止められない」
 └─ 80代の身体的衰えを直視する老境の知恵

ラフール・イスラエル（アメリカン・ムスリム）
 └─ 中心命題：「この世は旅の途中の宿屋」
 └─ 改宗者ゆえの信仰への自覚的距離感
```

### 【疫病 I】

```
ジム・ミューラー（エイズ患者・NAMESプロジェクト活動家、65歳）
 └─ 中心命題：パートナーとの対話を死後も続ける。コミュニティが喪の形式を作る
 └─ 背景：エイズ・メモリアルキルト、30〜40人の友人を次々と失う
 └─ 核心の言葉：「今日のインタビューを、エドワードに報告するつもりです」
```

### 【疫病 II】

```
ナンシー・ラヌー（空手教室経営者・乳がんサバイバー、48歳）
 └─ 中心命題：身体を隠さない。がんは個人の責任ではなく社会問題
 └─ 参照：オードリ・ロード「がん日記」、モーシェ・ダヤンとの比較
 └─ 活動：レズビアンコミュニティがんプロジェクト発足
 └─ 核心の言葉：「闘うとはどういうことか、女性たちに教えている」
```

### 【はるかな異国で】

```
ゲーリー・スラトキン（暴力防止活動家・医師、50歳）
 └─ 軌跡：サンフランシスコ結核対策→ソマリア難民医療3年→WHO東アフリカ7年→シカゴ
 └─ 数字の対比：年間1500〜3000万ドルで国を救える vs イリノイ道路工事140億ドル
 └─ 問い：「わかりますか?」（繰り返される読者への直接の問い）
 └─ 核心の言葉：「死は究極の教師。助け合うために何をしていないか教える」
```

### 【芸術に生きる】

```
ウィリアム・ウォーフィールド（バリトン歌手、80歳）→2002年没（82歳）
 └─ 中心命題：芸術と宗教は同じ。歌は神への接近
 └─ 転機：黒人霊歌を歌いながら「死への確信」を得た瞬間
 └─ 影響：レオンタイン・プライス、マリアン・アンダーソン、ポール・ロブソン
 └─ 核心の言葉：「自分は管のようなもの。神様から授かった才能が流れている」

ユタ・ヘイゲン（女優、81歳）→2004年没（84歳）
 └─ 中心命題：仕事が生の根拠。芸術への信仰
 └─ 死について：81年間、一度も死体を見ていない。夫も母も最期を見なかった後悔
 └─ 核心の言葉：「仕事ができなくなったら生きていたくない。でも今は生きていたい。仕事があるから」

ミック・ベタンコート（コメディアン、26歳）
 └─ 中心命題：死に囲まれた幼少期→「性急さ」→笑いで人とつながる
 └─ 背景：父（6歳）、祖父（13歳）を喪失。多くの周囲の喪失体験
 └─ 核心の言葉：「死はすぐそこにあるから、コメディアンとして人を幸せにしたい」
```

### 【死者の日】

```
カルロス・コーテズ（版画家・詩人、76歳）→2005年没（81歳）
 └─ 中心命題：死者の日の文化——死を祝い、命を祝福する
 └─ 実践：毎年「死者の日」に祭壇を作り、死者と食事をする
 └─ 核心の言葉：「生きることを恐れるなってことだ」

ヴァイン・デロリア（ネイティブアメリカン作家、60代）→2005年没（72歳）
 └─ 中心命題：世界は精神的なもの。死後の霊的継続性。大盤振る舞いの慣習
 └─ 逸話：「炎の雷のビリー」、30年旅したパイプが作者のもとへ
 └─ 核心の言葉：「生と死は輪のようにつながっている」

ヘレン・スクレア（元教師・墓地探索家、69歳）
 └─ 中心命題：だれも死について語らなくなった。墓地は死者との対話の場
 └─ 背景：生母が自分を産んで数日後に死んだ。里親に育てられた
```

### 【もうひとりの息子】

```
スティーブ・ヤング / モーリーン・ヤング（成人した息子を失った親）
 └─ 中心命題：親が子を失う——父と母それぞれの悲嘆の形
 └─ 「もうひとりの息子」：亡き息子が生者の中に生き続けるという感覚
```

### 【仕事】

```
ウィリアム・ハーデジェン（葬儀師）
 └─ 中心命題：職業的倫理と死の尊厳
 └─ 核心の言葉：「だれかが、エイズで亡くなった人たちの葬儀をしなきゃならない」

グアダルーペ・レイズ（障害のある息子の母）
 └─ 中心命題：「逝かせてやるべきだ」という決断と、その後の夢
 └─ 背景：知能・身体に障害のある息子ボビーを33年間育てた

ローリー・モイナ
 └─ 仕事を通じて死に接する別の視点
```

### 【終わりと始まり】

```
メイミー・モブリー / マービン・ジャクソン
 └─ 中心命題：老境の社会活動家たちの「終わりと始まり」
 └─ 「自分は何をしてきたか」「これから何が残るか」という問い
```

### 【エピローグ】

```
リンダとキャシー（レズビアンカップル、対話形式）
 └─ 登場人物：リンダ（元パートナー）、キャシー（ロンの友人）、ロン（エイズで死亡）
 └─ 構造：ロンの死が、リンダとキャシーを繋げ、新しい家族を生む
 └─ タイムライン：
     1993年 復活祭の週末：ミシガン湖で初めて4人で過ごす
     1993年7月：ロンをたたえるパーティ
     1993年秋：ロン死去
     1994年：リンダとマットがクリーブランドへ移住、4人一緒に暮らし始める
 └─ 核心の言葉：「ロンは私にマシューをくれた。キャシーをくれた。家族をくれた」
 └─ 本書の着地：死がきっかけで生まれた愛と家族
```
