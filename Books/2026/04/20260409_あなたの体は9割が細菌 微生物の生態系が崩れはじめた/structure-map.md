## プロローグ「回復はしたけれど」

**位置づけ**: 個人的動機の開示と問い立て  
**要旨**: 生物学者の著者がマレーシアでバベシア症に感染。大量の抗生物質で救命されるが、治療後から新たな慢性症状（皮膚炎・胃腸障害・感染症脆弱性）が多発。「抗生物質は良い菌も殺したのではないか？」という疑問が本書の動機。  
**重要引用**: 「家畜の群れをまるごと治療できるほどの量だった」

---

## 序章「一〇%のヒト」

**位置づけ**: 概念フレームの設定  
**核心概念群**:
- ヒトゲノム21,000遺伝子 vs マイクロバイオーム440万遺伝子
- ゴードンの無菌マウス実験：微生物が腸形態そのものを形成
- 虫垂＝腸内細菌の「種の保存庫」
- ヒトマイクロバイオーム・プロジェクト（2008〜、1億7000万ドル）
- マイクロバイオーム概念：ヒトは「10%のヒト＋90%の微生物」

**論理的役割**: 「個体としてのヒト」から「ホロビオント（複合体）としてのヒト」へ読者の認識を転換させる。

## 第1章「二一世紀の病気」

**位置づけ**: 問題提起  
**二〇世紀の医学的成果**:
- 4大革命：予防接種・上下水道・衛生管理・抗生物質
- 平均寿命：31歳（1900年）→ 66歳以上（現代）

**「二一世紀病」の疫学的特徴**:

| 特徴 | 詳細 |
|------|------|
| 発生域 | 欧米先進国・都市部に偏重 |
| 時代 | 1940〜50年代から急増開始 |
| 年齢 | 若者・子どもに多い（老齢化疾患ではない） |
| 性差 | 自己免疫疾患は女性に9:1で多い |
| 社会 | 豊かな国ほど多い |

**主要疾患カテゴリ**:
- アレルギー（喘息・湿疹・花粉症・食物アレルギー）
- 自己免疫疾患（1型糖尿病・多発性硬化症・セリアック病・関節リウマチ）
- 肥満・2型糖尿病
- 精神・発達疾患（自閉症・ADHD・OCD・うつ・不安障害）

**著者の仮説**: 腸内マイクロバイオータの破壊が全疾患群に通底する共通因子。

## 第2章「あらゆる病気は腸からはじまる」

**位置づけ**: 代謝・肥満分野でのテーゼ実証  
**論証の要**:
1. ニワムシクイ＝「同じカロリーでも吸収量が異なる」の自然界の実証
2. ゴードン/ターンバウの糞便移植実験：肥満は「伝達可能」
3. ルース・レイ：肥満マウスはフィルミクテス↑バクテロイデーテス↓
4. アッカーマンシア・ムシニフィラ：腸壁保護細菌の消失と肥満の関係
5. 胃バイパス手術の真の機序：食事制限より微生物リセット
6. IBS（過敏性腸症候群）：感染後・抗生物質後のディスバイオシスが根本

**数値**: 食物から2%多くカロリーを吸収するだけで10年で19kg増

## 第3章「心を操る微生物」

**位置づけ**: 脳・神経・精神疾患分野でのテーゼ拡張  
**段階的論証**:

```
Step 1: 自然界の行動操作（冬虫夏草・トキソプラズマ・狂犬病）
    ↓
Step 2: エレン・ボルトの仮説（抗生物質→C. diff→神経毒素→自閉症）
    ↓
Step 3: バンコマイシン試験（サンドラー医師）→一時的自閉症症状改善
    ↓
Step 4: プロピオン酸実験（マクフェイブ）→自閉症様行動の化学的再現
    ↓
Step 5: 無菌マウスのHPA軸異常（須藤・千田）→腸内細菌がストレス感度設定
    ↓
Step 6: セロトニン95%が腸内→迷走神経→「第2の脳」としての腸
```

**関連研究者**: エレン・ボルト、リチャード・サンドラー、デリック・マクフェイブ、エマ・アレン=ベルコ、シドニー・ファインゴールド

## 第4章「利己的な微生物」

**位置づけ**: 免疫・アレルギー分野でのテーゼ拡張  
**理論的進化**:

```
衛生仮説（ストラカン 1989）
「感染症暴露不足→免疫訓練不足→アレルギー」
    ↓（修正）
旧友仮説（ルーク）
「問題は感染症ではなく共生微生物の喪失」
    ↓（超克）
ホロゲノム進化論（ローゼンバーグ）
「宿主＋微生物が進化的単位（ホロビオント）」
```

**ゾニュリン・経路（ファサーノ）**:  
グルテン/感染/抗生物質 → ゾニュリン産生 → タイトジャンクション開放 → Leaky gut → LPS血中流入 → 自己免疫反応

## 第5章「微生物世界の果てしなき戦い」

**位置づけ**: 抗生物質・抗菌製品の副作用告発  
**主要テーマ**:

**耐性菌問題**:
- MRSA：メチシリン導入3ヶ月で耐性菌出現
- クロストリジウム・ディフィシル：芽胞形成で消毒・抗生物質に耐性

**マイクロバイオータ破壊**:
- クリンダマイシン7日投与→バクテロイデス多様性が2年間回復せず
- 幼小マウスへの低用量ペニシリン→体脂肪増加・代謝プログラム変容
- 生後6ヶ月以内の抗生物質投与→7歳時の過体重リスク上昇

**抗菌製品トリクロサン問題**:
- アメリカ人75%の尿から検出
- 普通の石鹸との感染症予防効果差なし
- 黄色ブドウ球菌鼻腔定着を促進、アレルギーと相関

**農業用抗生物質**: 家畜の75%が糞として排出→有機肥料→野菜→ヒト

## 第6章「あなたはあなたの微生物が食べたものでできている」

**位置づけ**: 食事・ライフスタイル改善の科学的根拠  
**食物繊維の役割**:
- 短鎖脂肪酸（SCFA）産生：酪酸・酢酸・プロピオン酸
- 酪酸：腸壁細胞のエネルギー源（腸細胞の70%を供給）
- アッカーマンシア栄養源：腸壁保護

**食文化と腸内細菌の共進化**:
- バンラジャ（ブルキナファソ農村）vs イタリア都市部：食物繊維分解菌の分布差
- 海苔分解酵素（ポルフィラナーゼ）：日本人固有の腸内細菌適応

**代謝性エンドトキシン血症（カニ）**:  
高脂肪食 → アッカーマンシア減少 → 腸壁菲薄化 → LPS血中流入 → 慢性炎症 → 肥満・インスリン抵抗性

**著者の食物繊維実験**:  
食物繊維増加後のアッカーマンシア60倍増加、フィーカリバクテリウム増加、症状改善

## 第7章「産声を上げたときから」

**位置づけ**: 出生・育児政策への科学的警告  
**帝王切開の二重剥奪**:

```
帝王切開
  ├── 産道通過なし → ラクトバチルス優位微生物を受け取れない
  └── 陣痛なし → ホルモンによる乳房への細菌移送が起きない
              → 初乳の微生物組成も経膣分娩と異なる
```

**帝王切開率**: WHO推奨5〜15% → 現実30〜50%（多数の先進国）

**母乳の構造**:
| 成分 | 機能 |
|------|------|
| 130種のオリゴ糖 | ビフィドバクテリウム属の選択的培養（プレバイオティクス） |
| 生きた細菌 | 樹状細胞が腸→血流→乳房へ運搬（初乳に1mL当たり1000個体） |
| 免疫グロブリン | 病原体に対する受動免疫 |
| 乳酸塩 | 大腸細胞への吸収・免疫発達 |

**母乳vs粉ミルクリスク比較**:
| 疾患 | 粉ミルクのリスク倍率 |
|------|---------------------|
| 耳感染症 | 2倍 |
| 呼吸器感染入院 | 4倍 |
| 胃腸感染症 | 3倍 |
| 壊死性腸炎 | 2.5倍 |
| 乳幼児突然死症候群 | 2倍 |
| 肥満 | 2倍 |
| 2型糖尿病（成人後） | 1.6倍 |

## 第8章「微生物生態系を修復する」

**位置づけ**: 治療・修復技術の現状整理  
**プロバイオティクスの限界と可能性**:
- 起源：メチニコフ（1908年）のヨーグルト長寿説
- 現状：菌株・製品・疾患特異性が高く「万能薬」ではない
- 有効な対象：C. difficile関連下痢、壊死性腸炎（未熟児）、IBS（一部）

**糞便微生物移植（FMT）**:
| 項目 | 内容 |
|------|------|
| 歴史 | 1958年アイゼマン初報告、1989年ボロディ炎症性腸疾患応用 |
| C. difficile治癒率 | 90〜95%（抗生物質単独は60〜70%） |
| 実施機関 | オープンバイオーム（マサチューセッツ）ほか |
| 将来応用研究 | 肥満・多発性硬化症・IBD・精神疾患 |

**ペギー・カン・ハイの症例**（本書の主要ケーススタディ）:  
交通事故後手術 → 抗生物質 → C. difficile感染 → 一日30回下痢 → 複数クール抗生物質失敗 → 夫からのFMT → 回復

## エピローグ「一〇〇%の世話をする」

**位置づけ**: 著者の個人的実践と結論  
**著者の選択**:
- 自然分娩（帝王切開を優先的に回避）
- 万一の帝王切開時：ドミンゲス=ベロ法（膣スワブ塗布）を実施
- 完全母乳育児（WHOガイドライン：生後6ヶ月は母乳のみ、その後2歳まで継続）
- 食物繊維を永続的に増加
- 不必要な抗生物質を避け、必要時にはプロバイオティクスを併用
