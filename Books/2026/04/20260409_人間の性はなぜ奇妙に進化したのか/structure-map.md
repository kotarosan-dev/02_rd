## 論証の流れ（マクロ）

```
哺乳類・類人猿との対照（何が変か）
    → 性間の利害対立という一般枠（投資・機会・父性確信）
        → 具体メカニズム（授乳の性差・排卵隠蔽・男性の貢献・閉経・第二性徴）
            → 「なぜそう見えるか」＝進化史・比較・未確定部分の明示
```

## 転換点（読み手が掴むべき山）

1. **第1章終盤**：「奇妙なのはヒト」への視点転換（種差別主義批判）。
2. **第2章中盤**：マダラヒタキのEPC/MRSと数字（扶養格差・欺瞞）で抽象理論が実証に接続。
3. **第4章後半**：シレーン=トゥルベリ／メラーの系統樹分析で、二つの対立仮説を時間軸で統合。
4. **第5章クライマックス**：ホークスのアチェデータが「家にベーコン」神話を覆す。
5. **第6章**：閉経を「老化の例外」としてキングピン原則と対置。
6. **第7章**：ペニスを「一見生理」から「シグナル／トレードオフ」へ再フレーミングで全書を収束。
