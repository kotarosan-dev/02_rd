# Structure Map: 『火星の人［新版］上』

## 基本メタ

- 書名: `火星の人［新版］上`
- 著者: `アンディ・ウィアー`
- 判定ジャンル: `B`
- サブタイプ: `SFサバイバル / 工学スリラー / 群像オペレーション`
- 読み筋: `遭難 -> 自給 -> 発見 -> 通信回復 -> 救援失敗`

---

## 三幕構成（上巻内）

### 第一幕: 遭難と自給体制の立ち上げ

- 起点: ソル6の砂嵐でワトニーが死亡扱いになる
- 当面の敵: 通信断絶、有限食料、孤独
- 主要手段: 棚卸し、農地化、水生成
- 第一幕の到達点: 「四年は無理だが、延命はできる」という現実的希望

### 第二幕: 地球の発見と火星の拡張

- 起点: ミンディが基地画像の異常から生存を見抜く
- 当面の敵: 通信手段の不在、距離、時間差
- 主要手段: ローバー改造、`RTG`、`パスファインダー`
- 第二幕の到達点: 地球との通信再建

### 第三幕: 救出の準備と再破局

- 起点: ワトニーとNASAの実務連絡開始
- 当面の敵: 食料タイムリミット、設備疲労、巨大組織の時間不足
- 主要手段: `アイリス` 補給、`MDV` 改造、ハブ運用継続
- 終点: ハブ爆発と `アイリス` 打ち上げ失敗

## 因果連鎖マップ

### 連鎖1: 生存

1. アンテナが刺さる
2. スーツに穴が開く
3. 倒れた姿勢と出血で一時的に密閉
4. 自動生命維持が残圧を維持
5. 警報で意識回復
6. ハブへ帰還

### 連鎖2: 食料

1. 一人なので糧食は延びる
2. しかし `アレス4` まで届かない
3. 植物学知識で農地化を発想
4. 土壌細菌と人糞で土を作る
5. 水が不足
6. 水生成計画へ移行

### 連鎖3: 水

1. `MDV` にヒドラジンが残る
2. `MAV` 燃料プラントで酸素を作れる
3. 水素燃焼で水を得る
4. 制御ミスで水素蓄積
5. 爆発
6. それでも農場は維持される

### 連鎖4: 発見

1. ヴェンカトが再観測を要求
2. ミンディが画像の違和感を読む
3. 生存確信
4. 公表
5. 救出計画の政治・技術ラインが起動

### 連鎖5: 通信回復

1. `アレス4` まで直接行くには遠すぎる
2. ワトニーが `パスファインダー` を思い出す
3. ローバーを長距離仕様へ改造
4. 探査機回収
5. JPLが旧システムを再構築
6. ASCII とハックで実用通信成立

### 連鎖6: 再破局

1. 農場成功でソル900近くまで延命可能に見える
2. ハブのエアロック材 `AL102` に疲労蓄積
3. ソル119で破断
4. ハブ減圧、農場全滅
5. 食料期限がソル584へ縮む
6. 補給便 `アイリス` に依存
7. `アイリス` も失敗

## 人物アーク

### マーク・ワトニー

- 出発点: 軽口を叩く有能な下っ端クルー
- 中盤: 火星唯一の農夫兼修理工
- 後半: NASAと交渉する現地責任者
- 上巻終点: 希望を取り戻しかけた直後に、再び補給不全へ追い込まれる

### メリッサ・ルイス

- 出発点: 判断の速い船長
- 転換点: クルー全員の生還を優先してワトニー切り捨てを決断
- 後半: 生存を知っても純粋には喜べず、自責に飲まれる
- 上巻終点: 救う側に回る準備は整うが、罪悪感は未消化

### ヴェンカト・カプーア

- 出発点: 火星計画の官僚的責任者
- 転換点: 再観測への執着
- 後半: 実務・政治・技術を束ねる司令塔
- 上巻終点: `アイリス` 失敗で再びゼロ地点に引き戻される

### ミンディ・パーク

- 出発点: 夜勤の衛星担当
- 転換点: 生存を見抜く
- 後半: 軌道監視と解析の中心へ昇格
- 上巻終点: 彼女の発見がなければ生まれなかった救出計画自体が、次段階へ移る
