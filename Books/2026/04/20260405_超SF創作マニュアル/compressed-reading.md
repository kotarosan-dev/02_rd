## 読書ナビゲーション

この本は、単なる「SF入門」ではない。
SFを書くために必要な発想法、読むべき本、使うべき用語、知っておくべき歴史、入るべきコミュニティ、出すべき新人賞、受講後の動き方までを一冊で束ねた創作インフラ本である。
前半は「SFとは何か」を編集者の会話で掴ませる。
中盤は「何を読めばいいか」「どんな語彙で話すか」を最短距離で与える。
後半は「SF界はどう回ってきたか」「これからどこに出せばいいか」「講座というゲームをどう生きるか」を実務に接続する。
つまり本書の本当の主題は、SF作品そのものではなく、SFを書く人が立つべき座標軸である。
読む順番としては、最短で効くのは座談会1 → AIコラム → 必読書24 → 用語50 → おわりに。
業界地図を掴みたいなら、座談会2 → 新人賞一覧 → 完全攻略マニュアルの順がよい。
講座受講者なら、付録は「おまけ」ではなく、実戦上の取扱説明書として読むべき章である。
この圧縮読本では、本書を「創作の起動法」「判断法」「接続法」を供給する道具箱として再構成する。

## コアフレームワーク

### 1. SFの最小核

本書が何度も別の角度から言い換える結論は、SFはガジェット名ではなく、思考の運動だということに尽きる。
ロボットが出てもSFでないことはある。
宇宙人が出なくてもSFであることはある。
重要なのは、最初の仮定が、その後の世界像と価値観をどう変質させるかである。
本書が強調するSFの核は、次の四点に集約できる。

```text
if を置く
→ 論理を転がす
→ 認識を異化する
→ センス・オブ・ワンダーを残す
```

- `if`: 「もし○○だったら」という小さな仮定の種。
- `論理`: 風が吹けば桶屋が儲かる式に、因果を遠くまで延ばす力。
- `異化`: それまでの「あたりまえ」が更新され、世界の見え方が変わること。
- `驚き`: ただ説明できるだけでなく、畏怖や知的興奮が残ること。

この四点のうち、本書は特に二番目と三番目を重視する。
「すごい設定」より「転がる論理」。
「珍しい道具」より「価値観の更新」。
ここを外すと、SFっぽい小道具を貼っただけの作品になる。

### 2. 蚊取り線香モデル

本書の象徴的な比喩が「蚊取り線香」である。
ミステリもSFも論理を扱うが、その論理の向きが違う。
ミステリは外から中心へ、可能性を削って一つの解へ収束する。
SFは中心から外へ、可能性を広げながら世界を変質させる。
この比喩が便利なのは、SFを「宇宙船の有無」ではなく「展開方向」で説明できる点にある。

| 項目 | ミステリ | SF |
|---|---|---|
| 論理の方向 | 収束 | 拡張 |
| 基本運動 | ふるい落とし | 展開と増殖 |
| 読後感 | 謎が解ける | 世界が広がる |
| 中心的快楽 | 解明 | 異化と飛躍 |
| 代表比喩 | 外から内へ | 内から外へ |

この比喩を創作時に使うと、プロット診断がしやすい。
自分の物語が「解答に向かって閉じている」のか。
それとも「仮定から別世界へ開いている」のか。
SFが途中で常識へ着地し、「結局ふつうの価値観でした」と終わると失速する。
本書が「やっぱり友情は大事」で終わるなと警告するのはそのためだ。

### 3. ガジェット中心設計

本書でもっとも実作向きなのは、「ガジェットの気持ちになれ」という助言である。
普通は「この物語にとってこのガジェットは何か」と考える。
だがSFでは逆向きに、「このガジェットにとってこの物語は何か」と問う。
これは発想の主語をずらす技法であり、SFをキャラ中心プロットから切り離す。
ガジェットやアイデアが先にあり、その論理が人物と社会を押していく構図が見えてくる。

| 問い | 弱い設計 | 強い設計 |
|---|---|---|
| 出発点 | キャラの都合 | アイデアの必然 |
| 道具の扱い | 飾り・小道具 | 世界を変える原理 |
| 世界設定 | 背景美術 | 因果の発生源 |
| キャラ | 先に固定 | 後から最適化 |
| 結果 | SF風ドラマ | SF固有のドラマ |

本書は、ファンタジーが象徴性から考えがちなのに対し、SFは機能から考えると述べる。
ドラゴンなら「崇高の象徴」より「乗って飛ぶと速い」が先に来る。
この雑さは、低級という意味ではない。
機能が先に立つからこそ、そこから社会制度や日常慣習まで連鎖的に変えられるのである。
創作上は、「その道具が実在したら生活のどこが変わるか」を執拗に追うのがコツになる。

### 4. アイデア生成の五段変形

本書の着想法は、一見すると散発的に見えるが、まとめると五段階の変形法になっている。
まず異質な単語をぶつける。
次に修飾語で意味の角度をつける。
さらに別の語へスライドさせる。
そこに利害関係と対立を足す。
最後に物語として持続するだけの強度があるかを見る。

```text
単語を出す
→ 修飾する
→ 組み替える
→ 利害関係者を置く
→ 長編化できるか検査する
```

- 星新一型: 雑然とした二語をぶつけて、妙な核を作る。
- 江坂遊型: 名詞と修飾語をペア化し、組み換えで概念を育てる。
- 田丸雅智型: 名詞から連想語を広げて、別の名詞に接続する。
- 太田忠司型: 助詞や語順まで変え、シチュエーションを変形する。
- AI補助型: 未解決問題や専門知を掘り、長編に耐える素材を探す。

この本の重要点は、アイデアは「ひらめき待ち」ではなく、組み合わせ可能な作業だと示すことだ。
発想を神秘化しない。
だが同時に、方法論を機械的に適用すれば勝てるとも言わない。
方法を身体化した先で、自分のノリを作れという立場である。

### 5. 物語化の判断木

本書が『ジュラシック・パーク』を例に見せるのは、if から長編へ移るときの判断木である。
ただ「恐竜が蘇った」では短い。
そこから、いいこと、悪いこと、得する人、損する人、悪役、制度、続編の余地へと広げる。
この手順は、SFアイデアをエピソードからドラマに変える最短ルートとして使える。

| 問い | 目的 |
|---|---|
| もし○○だったら? | 種を置く |
| 何が良くなる? | ユートピア面を掘る |
| 何が悪くなる? | ディストピア面を掘る |
| 誰が得する? | 利害の軸を作る |
| 誰が損する? | 反対者を作る |
| 誰が悪用する? | 衝突を発生させる |
| 続けて何が起こる? | スケールを拡張する |

この表は、SFだけでなく企画会議でも使える。
本書が創作マニュアルでありながら技術書的なのは、こうした判断軸がそのまま再利用可能だからだ。
また、本書はロボット三原則を「設定しただけで事件が生まれる大発明」と見ている。
よいSFガジェットとは、世界観の飾りではなく、反復的に物語を生む生成装置なのだ。

### 6. 創作者の成長ループ

本書の後半に通底するのは、「読む → 分解する → 書く → 出す → 続ける」というループである。
ここには才能神話より継続神話がある。
まず100冊読む。
気に入った作品をリバースエンジニアリングする。
短編だけでなく長編が書ける身体を作る。
講評や読者の反応を受ける。
そして一発で決めようとせず、作者として見られることに耐える。

```text
読む
→ 仕組みを抜き出す
→ 書き終える
→ 公開して読まれる
→ 次を書く
```

とりわけ本書は「最後まで書き終える」ことを重く見る。
途中で自作の弱点に気づくこと。
終わりから逆算して前段に何が要るか見えること。
他人に読んでもらう資格がそこで初めて生まれること。
この三点を、単なる根性論ではなく、構造的な学習条件として捉えている。

### 7. 業界接続の六経路

本書は作法書であると同時に、SF界への接続マップでもある。
書いた後の行き先を具体化している点が重要だ。

| 経路 | 入口 | 役割 |
|---|---|---|
| 商業新人賞 | ハヤカワSFコンテスト等 | プロへの正面ルート |
| 短編賞 | 創元SF短編賞等 | 密度を磨くルート |
| 投稿サイト | カクヨム等 | 読者接続と継続運用 |
| 同人誌 | 文フリ、Booth | 自律的発表基盤 |
| 創作講座 | ゲンロンSF創作講座 | 集中的訓練環境 |
| ファンダム / 読書会 | SF大会、ウェブ媒体 | ネットワーク形成 |

本書が繰り返すのは、「どれが正解か」ではなく「自分が何を目指すか」で選べということだ。
専業か兼業か。
趣味かプロ志向か。
同人中心か商業志向か。
ここを曖昧にすると、講座や賞が与えるゲーム的達成に振り回される。

### 8. パターン / アンチパターン

本書全体から抽出できる推奨パターンと回避パターンを対で置く。

| パターン（推奨） | アンチパターン（回避） | 理由 |
|---|---|---|
| if から始める | 題材ラベルから始める | SF性が浅くなる |
| 論理を転がす | 設定を説明するだけ | 物語化しない |
| ガジェット視点で考える | キャラ都合だけで閉じる | SF固有性が消える |
| 100冊読む | 既知の数冊だけで書く | 文脈の狭さが出る |
| 分解して真似る | 雰囲気だけ真似る | 再現性がない |
| 長編力を鍛える | 短編の巧さに逃げる | 職能が伸びにくい |
| 読まれる場に出す | 私的保存で終える | 反応が得られない |
| 継続を前提にする | 一発受賞をゴールにする | 作者として見られない |

## 章別エッセンス

### 監修者より

**核心命題**: 本書は、ゲンロン大森望SF創作講座の受講生が繰り返し直面してきた「SFって何ですか」「どうすればSFになりますか」に対する補助教材として作られた。編集者や講師が長年共有してきたハイコンテクストな前提を、一般読者にも届く形へ圧縮することが狙いである。
**事例サマリー**: 2016年4月開始の講座が第9期に入ったこと、2017年に『SFの書き方』が早川書房から出て2025年に中国語版まで刊行されたことが、講座の蓄積の大きさを示す。
**行動指針**: 自分がSFを知らないと感じるなら、それを恥じる必要はない。基礎知識を一冊で仕入れ、そこから読む・書くに接続すればよい。
この序文の仕事は、読者を気後れから救うことにある。
講座受講生の多くは、実はSFを大量には読んでいない。
それでも講座は続いてきた。
だから必要なのは、門前払いではなく、入門の足場である。
大森望は本書を「現代SFの基礎知識」的な本として位置づける。
同時に、最後の一週間で自分がかなり手を入れたと書き、編集物としての責任の所在も引き受ける。
この距離感は本書全体の調子を決めている。
内輪の蓄積を外へ開くが、ファンダムの具体性は捨てない。
監修者よりは、単なる挨拶ではなく、本書がなぜ必要になったかの問題設定である。

### ゲンロン大森望SF創作講座とは

**核心命題**: 講座は、梗概1200字と実作16000字を往復しながら、月1回の提出と講評で書く力を鍛える場である。単なる講義型ではなく、提出と講評のサイクルを軸にした実戦型の訓練装置だ。
**事例サマリー**: 講義では全員が梗概を出し、上位3名が選出され、次回までに16000字の実作を書く。期末には40000字短編でゲンロンSF新人賞選考会に進む。
**行動指針**: 受講するなら、講義を聞くだけでなく、提出のサイクルに身体を合わせることが重要になる。
この節は、講座を抽象的な「学校」ではなく、締切と公開講評の機械として示す。
作家や編集者が毎回ゲスト講師として登壇する。
開講から約10年で多くの作家がデビューしたことも書かれる。
つまり、これは学習の場であると同時に、発掘装置でもある。
本書後半の「ゲーム性」の議論を理解する前提もここにある。
選出、実作、最終課題という節目が明確だからこそ、人は攻略を考え始める。
しかし講座の価値は、勝敗以上に、月次の反復にある。
毎月アイデアを作り、毎月読まれる。
この強制力が、普通なら先延ばしされる創作を前に進める。

### はじめに「SF作家になりたいあなたへ」

**核心命題**: まず必要なのは、「自分はSFを知らないかもしれない」という感覚を認めることと、それでも学べる道筋があると理解することだ。本書の入口は、知識の誇示ではなく、コンテクストの共有である。
**事例サマリー**: 第1期第1回講義は深夜1時過ぎまで続いたが、多くの新規受講生には内容がハイコンテクストすぎた。池澤春菜監修『現代SF小説ガイドブック』にも触れつつ、総覧的ガイド不足が指摘される。
**行動指針**: 自分が知らない前提を埋めるため、まず全体地図を持て。名作、用語、歴史、ファンダムをばらばらではなく同時に掴むべきだ。
ここで小浜は、自分たちの「普通」が普通ではなかったと認める。
これは教育設計上とても重要な反省である。
教える側の内部常識を可視化しなければ、新人は議論に入れない。
本書が座談会・推薦図書・用語集・歴史対談を一冊に入れたのは、その穴を一気に埋めるためだ。
また、SFアレルギーが減った時代になった一方で、過去の文脈には不案内な人が多いとも述べる。
接点は増えたが、系譜は見えにくい。
このねじれが、本書の存在理由である。
つまり本書は「入口が増えた時代の、地図の欠如」に応える本なのだ。

### 座談会1「SFのロジックは蚊取り線香だ」

**核心命題**: SFは「if の文学」であり、その本質は小さな仮定から論理が転がっていき、認識的異化とセンス・オブ・ワンダーを引き起こすことにある。ギミックの細かさは本質ではない。
**事例サマリー**: 井手は「if」「論理の転がり」「センス・オブ・ワンダー」の三点を最小条件として挙げる。小浜は「この世界の常識に着地させるな」と言い、法月綸太郎の名探偵論を引いて「ガジェットの気持ちになれ」と助言する。
**行動指針**: 自作の核に、仮定から別価値観が出てくる運動があるかをまず点検せよ。
この座談会は本書の頭脳であり、繰り返し読む価値がある。
特に、「SFはなんでもあり」という乱暴な自由論ではなく、「if の手続きさえ丁寧なら結果的に何をやってもいい」という条件付き自由論なのが重要だ。
そこには手続きを通じて飛躍を正当化する姿勢がある。
また、ミステリとの違いを蚊取り線香で説明した比喩は、創作理論として非常に強い。
自分のプロットが収束型か拡張型かを一目で診断できるからだ。
加えて、現代SFが個人とAIの関係性、脳科学、近接未来へ寄りがちで、大きな物語が減っているという時代診断も入る。
この観察により、本書は古典礼賛だけでなく、現代の狭まりも見ていることがわかる。
最後に出る「羽目を外せ」「価値観を揺るがすびっくりどっきり」という助言は、倫理的無難さに閉じがちな新人への喝である。

### コラム1「生成AIでSFアイデアを得てみる」

**核心命題**: AIは完成品を出してもらう相手ではなく、強度ある素材を発掘するための検索・調査補助として使うのがよい。特に「未解決問題」を掘らせると、発展余地のある種が得やすい。
**事例サマリー**: 読書猿『アイデア大全』のような一般的発想法に触れつつ、AI活用では学問、専門技術、調査報道など知の蓄積領域を下敷きにすることが勧められる。
**行動指針**: AIに物語を丸投げせず、未解決問題・論点・研究の穴を探させ、その先の展開は自分で作る。
このコラムは短いが、2025年時点の創作実務に直結する。
AIを使うなではなく、使いどころを限定する。
素材探索と要約には有効。
しかし物語の展開は、自分の論理の悪ノリで伸ばすべきだという含意がある。
本書全体の思想とも矛盾しない。
if を置く主体は最後まで人間に残る。
また、長編を支えるためには短い奇抜さではなく、知識の厚みが必要とも言う。
この点で、AIは表面の奇抜さを増やす道具ではなく、背景知を圧縮する道具として扱われる。
創作AI論のなかでは、かなり節度のある立場である。

### 基礎知識1「作家になるにはこれを読め! 必読書24」

**核心命題**: SFを書きたいなら、まずジャンルSFが長年培ってきた王道の面白さを身体に入れる必要がある。日本12冊、海外12冊という絞り方は、入門者にとって十分に小さく、同時に十分に広い。
**事例サマリー**: 国内では山田正紀『神狩り』、小松左京『ゴルディアスの結び目』、眉村卓『消滅の光輪』、新井素子『ひとめあなたに…』、筒井康隆『虚航船団』、神林長平『あなたの魂に安らぎあれ』、瀬名秀明『BRAIN VALLEY』、野尻抱介『太陽の簒奪者』、飛浩隆『象られた力』、伊藤計劃『ハーモニー』、上田早夕里『華竜の宮』、酉島伝法『皆勤の徒』が挙がる。
**行動指針**: 好みだけで閉じず、異なるサブジャンルを横断して、何が自分の血になるかを見極めよ。
この章の価値は、ただの名作案内ではなく、サブジャンルの代表を一作家一作に絞った編集判断にある。
つまり、深く沼らせる前に、地図の全体像を優先している。
また、各作品紹介が短いながらも、固有の面白さを一行で立ち上げる練習にもなっている。
たとえば『神狩り』なら「神を狩れるのか」。
『華竜の宮』なら海洋世界の技術・社会制度・文化の変容。
『三体』なら文明規模の侵略の引き金。
推薦リストは、そのまま梗概の圧縮訓練の見本としても読める。
この章を読むと、自分が何に惹かれるかが露わになる。
つまり推薦図書は、読むためだけでなく、自分の志向を診断する鏡でもある。

### 基礎知識2「今日から使えるSF基礎用語50」

**核心命題**: SFを書くには、作品を読むだけでなく、その作品を語り分ける語彙が必要である。語彙は単なるラベルではなく、世界観、技法、歴史、ファンダムを結ぶハンドルだ。
**事例サマリー**: ハードSF、本格SF、メタフィクション、伝奇SF、ジェンダーSF、言語SF、時間SF、ミリタリーSF、スペースオペラ、ニューウェーブ、サイバーパンク、スチームパンク、ディストピアSF、シンギュラリティ、ロボット、ポストヒューマン、テラフォーミング、軌道エレベータ、恒星間宇宙航行、メタバース、マルチバースなど、概念と作品例が結びつけられる。
**行動指針**: 用語を暗記するのではなく、どの作品がどの概念の典型かをセットで覚えよ。
この章の技術書的価値は高い。
なぜなら、用語の説明が定義だけでなく、「その言葉がどの文脈で便利か」まで含んでいるからだ。
たとえば時間SFなら、タイムトラベル、タイムスリップ、タイムリープ、タイムループの差が整理される。
ロボットなら、アンドロイド、レプリカント、三原則、フランケンシュタイン・コンプレックスまで視野に入る。
さらに、この章は用語集でありながら、作品発想のネタ帳でもある。
用語一つごとに、制度、倫理、身体、時間、政治のどれが揺れるかが見えてくるからだ。
逆に言えば、語彙を持たないまま書くと、自分のやっていることが既存のどこに接続するか判別できない。
新人にとって、これは「知識の武装」というより「会話への参入券」である。

### 座談会2「プロもアマチュアもファンダムだった」

**核心命題**: 日本SFは作品だけでできたのではなく、翻訳者、編集者、同人誌、専門誌、コンベンション、大学SF研、ワールドコン、ウェブ媒体が重なって形成された。SFを書くとは、作品を書くことと同時に、どのネットワークとつながるかでもある。
**事例サマリー**: 矢野徹の1953年渡米、柴野拓美の『宇宙塵』、福島正実の『SFマガジン』、ハヤカワSF文庫1970年創刊、日本SF大会、星雲賞1970年創設、伊藤計劃以後、Nippon2007、VG+、Kaguya Planet、ゲンロンSF創作講座2016年開講などが節目として描かれる。
**行動指針**: 作品の系譜だけでなく、媒体とコミュニティの系譜を理解し、自分がどの流れの上に立つのかを意識せよ。
この章の面白さは、文学史が人間関係史として見えてくる点にある。
アメリカから本を送ってもらう。
神保町の古書店で果たし状が飛ぶ。
同人誌のゴシップ欄が若手を引き寄せる。
大会の運営思想がコミケに間接的につながる。
つまり日本SF史は、制度史であると同時に社交史である。
また、90年代の「冬の時代」を、新人賞休止・専門誌減少・ネット移行で説明する視点も重要だ。
ジャンルの衰退は作品の質だけでは起こらず、供給装置とコミュニケーション装置が痩せることで起こる。
だから再生も、作品単体ではなく、講座、ウェブ媒体、読書会、同人誌の再編によって生じる。
この視点を持つと、自分がやるべきことが小説一本の執筆に閉じなくなる。

### コラム2「これはSFではない」

**核心命題**: 「これはSFではない」という言葉は、排除の呪文としてだけでなく、判断基準がどこにあるのかを露出させる契機として受け止めるべきである。対話を通じて、自分の物差しと他人の物差しのズレを学べる。
**事例サマリー**: 小浜は「SFがどういうものか知りたいなら100冊読め」と再び述べたうえで、人によって「面白い」「かっこいい」「気持ちいい」の基準が違うことを強調する。
**行動指針**: 否定されたときは、落ち込む前に、相手が何を尺度にしているかを聞け。
このコラムは、ジャンル論争を人格攻撃にしないためのマナー集でもある。
好みの違いと判断基準の違いは重なるが、同じではない。
人は読書経験の蓄積から物差しを作る。
だから否定は、相手の蓄積の表明でもある。
もちろん作者側からすれば「こんなにSFなのに」と思う。
しかしそのすれ違いからしか学べないことがある。
本書の態度は、境界線をなくそうではない。
境界線の引き方を議論可能なものにしよう、である。
この節があることで、本書全体が教条主義に傾くのを防いでいる。

### おわりに「小説を書き上げ、書き続けるために」

**核心命題**: 才能や理論以前に、書き終える回数を増やすことが創作者の基礎体力になる。初心者、中級者、上級者で読むべき作法書は変わるが、共通するのは「最後まで書いて、次を書く」ことだ。
**事例サマリー**: 初心者向けに森沢明夫『プロだけが知っている小説の書き方』、榎本秋編著『100のあらすじでわかるストーリー構成術』、さそうあきら『マンガ脚本概論』、中級者向けに三宅隆太『スクリプトドクターの脚本教室』、上級者向けに橋本陽介『物語論 基礎と応用』などが挙げられる。
**行動指針**: 現在の自分の段階に合う本を選び、最短で一作書き上げるループに戻れ。
この章は、本書のなかで最も「継続」に寄った章である。
ここではSF性より、職能としての書く力が重視される。
ログライン。
三幕構成。
描写の精度。
長編を書き切る体力。
これらはジャンルを超えるが、SF作家にも不可欠だとされる。
特に印象的なのは、「説明より描写」と市場の需要を結びつけて語る部分だ。
また、生成AIに感想を言わせて慰めを得て執筆速度が上がった例まで拾い、継続に資するなら手段を選びすぎるなという現実感もある。
最終的には、複雑化する未来社会のなかで、どんな価値観と倫理観を創造するかが問われると締める。
ここで技術と責任がつながる。

### 新人賞一覧

**核心命題**: 発表先は抽象的な「どこか」ではなく、枚数、媒体、賞金、刊行形態、締切によって具体的に選ぶべきだ。賞は夢ではなく仕様である。
**事例サマリー**: 創元SF短編賞は40-80枚、ハヤカワSFコンテストは100-800枚、日経「星新一賞」は10000字以内、かぐやSFコンテストは2000-4000字など、条件が明記される。隣接では日本ファンタジーノベル大賞、横溝正史ミステリ&ホラー大賞、『このミステリーがすごい!』大賞なども紹介される。
**行動指針**: 自分の得意な長さ、目標とする媒体、商業化の速度を見て応募先を選べ。
この一覧の実務価値は高い。
創作論を読んで昂ぶっても、出す場所が定まらなければ作品は止まる。
逆に募集条件が決まると、書くべき長さと締切感覚が具体化する。
また、SFに限らず隣接ジャンルへも門戸を開いている点が重要だ。
本書は、純粋SF主義で進路を狭めていない。
いまの作家は、SF賞だけでなくホラー、ミステリ、ファンタジー、ライトノベル、ウェブ媒体を横断して出ていく。
作品の性質と自分の戦略に応じてルートを選ぶという、かなり現実的なガイドである。

### 付録「SF創作講座完全攻略マニュアル」

**核心命題**: 創作講座は教育の場であると同時に、公開比較・得点・選出・新人賞を備えたゲーム的システムである。その構造を理解し、利用しつつ、しかしゲームに自分の表現を食い尽くさせないことが重要だ。
**事例サマリー**: 河野咲子は、梗概1200字、実作16000字、期末40000字、ウェブ公開、得点一覧表、選出の仕組みを説明し、「選出を目指す」「守りに入らない」「ゲームの外の問いも持つ」ことを勧める。
**行動指針**: 講座を攻略するなら、見たことのないSF的アイデア、フェティッシュな見せ場、字数に見合った起伏を梗概段階から示せ。
この付録は、本書中もっともメタであり、同時にもっとも現代的である。
作品は一律に公開され、受講生どうしが相対化される。
読者はアリーナを見下ろす観客になる。
その構造を河野はきわめて明晰に言語化する。
また、ゴゴゴ会、ダールグレンラジオ、裏SF創作講座など、公式と非公式の中間にある支援装置まで記録しているのが面白い。
つまり講座の価値は、授業だけでなく、その周辺に形成される準公共圏にある。
さらに、講座の「勝利条件」はゲンロンSF新人賞受賞だが、受賞しなくても活躍する卒業生は多いと明言する。
ここでゲームを絶対化しない。
そのバランス感覚が、この付録を単なる攻略記事以上のものにしている。

## 原文に近い核（引用）

> 「SFは『もし○○だったら』というifの文学です。」
>  
> 小さな仮定から世界を変質させるという、本書全体の出発点を最も短く言い切った一文。定義というより、作動原理の提示になっている。

> 「SFもミステリも蚊取り線香のようなもので、SFは渦巻きの中央から外に出ていくが、ミステリは外から中央を目指す」
>  
> ジャンル差をキャラクターや舞台装置ではなく、論理の方向として説明する比喩。創作時の診断ツールとして非常に強い。

> 「このガジェットにとってこの物語はなんなのか」
>  
> 発想の主語を反転させることで、SFアイデアを単なる飾りにしないための核心フレーズ。人物中心発想から世界原理中心発想へ視点をずらす。

> 「まず100冊読みなさい」
>  
> 読書量のノルマというより、好き嫌いの軸と判断基準を獲得する最低ラインの提示。読まないまま書くことへの最も率直な警告である。

## 批判的視座

- **古典中心の入口設計**: 必読書24は非常によく絞られているが、入口が「王道」に寄るぶん、周縁的・非英語圏・現代ウェブ発の実験作はやや後景化する。
- **編集者中心の価値観**: 長編重視、商業新人賞重視の姿勢は明快だが、短編連作や非商業継続を主戦場にする書き手には圧が強い。
- **講座モデルの強さと偏り**: ゲーム的な仕組みは鍛錬に効く一方、比較と公開に強く疲弊するタイプの書き手をこぼす可能性がある。
- **ファンダム史の厚みと地域格差**: 歴史記述は豊かだが、東京・関西中心の視界がやや強く、地方の継続的実践は断片的にしか見えない。
- **AI論の射程の短さ**: 2025年時点では妥当な整理だが、AIが創作補助のどの工程を変えるかの議論はまだ緒に就いたばかりである。
- **ジェンダー / ダイバーシティの位置づけ**: 用語集で触れられるが、創作実践上どこで倫理判断に接続するかの議論はもっと掘れる。

## 用語・前提の整理

- **認識的異化**: 世界の見え方を更新し、「知る前には戻れない」感覚を作ること。
- **センス・オブ・ワンダー**: 未知や巨大さへの畏怖と好奇心が混ざった知的驚き。
- **ガジェット**: 単なる小道具ではなく、物語を発火させる設定原理。
- **リバースエンジニアリング**: 既存作を分解し、登場人物配置や場面機能を読み解く学習法。
- **ファンダム**: ファンの集合ではなく、雑誌、同人誌、大会、ネットワークを含む文化的基盤。
- **サーコン**: serious and constructive の略で、真面目で建設的なコンベンション志向。
- **ファニッシュ**: fannish の意で、交流や遊びの比重が高いファン活動志向。
- **Jコレ**: 早川書房の日本人SF書き下ろし叢書で、2000年代再編の象徴。
- **選出**: 講座で梗概が評価され、次回の実作を重点的に読まれる資格。
- **攻略**: 勝ち方の工夫だけでなく、システムとの距離の取り方まで含めて考えること。

## 実践チェックリスト

□ 作品の出発点を「設定名」ではなく「もし○○だったら」で言える。
□ その if から起きる二次的・三次的な変化を最低3段階書き出している。
□ その変化で得する人と損する人を分けて考えている。
□ 悪用する主体や制度の歪みまで想定している。
□ ガジェットを飾りではなく、物語の発火装置として扱っている。
□ 現実の常識に安直に着地させていない。
□ 読者の認識を更新する驚きがある。
□ 少なくとも30冊、できれば100冊の読書計画を持っている。
□ 好きな作品を一作、場面単位で分解している。
□ 短編だけでなく長編の設計経験を積もうとしている。
□ 自分の段階に合う作法書を一冊選んで実践している。
□ 説明文ではなく、描写で感情や行動を見せる意識がある。
□ 梗概の段階で見せ場、起伏、独自性を明示できている。
□ AIは完成品生成より、未解決問題や資料探索に使っている。
□ 用語を作品例付きで覚え、会話に使えるようにしている。
□ 同人誌、投稿サイト、新人賞など発表経路を複数比較している。
□ 締切、枚数、媒体条件を見て応募先を選んでいる。
□ 読まれることを前提に作品を出している。
□ 受賞をゴールではなく、書き続ける出発点と考えている。
□ 自分がなぜSFを書くのかを、ゲームの外側の問いとして持っている。

## よくある落とし穴

1. **ギミック過信**: 設定が珍しければSFになると思い込む。論理の転がりがなければ死ぬ。
2. **常識への回収**: 異化を起こしたのに、最後だけ現実の道徳で閉じてしまう。
3. **キャラ先行の閉塞**: キャラの感情線だけで話を回し、ガジェットが背景化する。
4. **読書不足の自己流**: 既存文脈を知らないまま新規性を主張し、既視感を避けられない。
5. **方法論の丸飲み**: 単語法や三幕構成を使うだけで面白くなると思う。
6. **短編巧者の停滞**: 磨いた短編だけを繰り返し、長編体力を育てない。
7. **賞の神話化**: 一度の受賞で作家人生が決まると思い込み、継続の視点を失う。
8. **公開回避**: 書くが出さない。反応が返らず、改善点が外部化されない。
9. **AI丸投げ**: 発想補助ではなく代筆期待になり、自分の論理運動が空洞化する。
10. **ゲームへの同化**: 講座で勝つことだけが目的になり、書く理由が消える。
