# Structure Map

## 書誌

- 書名: `超SF創作マニュアル`
- 監修: `大森望`
- 主執筆: `小浜徹也` / `井手聡司`
- 協力執筆: `河野咲子` / `東浩紀`
- 位置づけ: `創作マニュアル` + `ジャンルガイド` + `業界地図` + `講座実務書`
- 判定ジャンル: `C/G`
- 構造タイプ: `概念編 → 資料編 → 歴史編 → 実務編`

---

## 全体構造

```text
監修者より
→ 講座の前提共有
→ SFの定義と発想法
→ AI補助の短い実務
→ 必読書リスト
→ 基礎用語辞典
→ 日本SF史 / ファンダム史
→ 境界線をめぐる補論
→ 書き続けるための作法書案内
→ 新人賞一覧
→ 講座攻略 / 周辺コミュニティ記録
```

この本は章ごとに役割がはっきり分かれている。
前半は「SFとは何か」の定義と発想法を与える。
中盤は「何を読めばよいか」「どう語ればよいか」を補う。
後半は「どんな歴史と場があり、どこに接続するか」を示す。
最後の付録は、講座受講者向けの局所的な実務書でありながら、現代の創作共同体論にもなっている。

## 主要フレームワーク詳細

### A. SF生成フロー

```text
仮定を置く
→ 良い変化 / 悪い変化を洗う
→ 得する主体 / 損する主体を置く
→ 悪用する主体を置く
→ 社会制度まで連鎖させる
→ 認識が戻れなくなる地点まで押す
```

### B. 学習フロー

```text
読む
→ 好き嫌いを自覚する
→ 既存作を分解する
→ 自作で試す
→ 最後まで書く
→ 公開して読まれる
→ 次作で補正する
```

### C. 講座プレイフロー

```text
梗概1200字提出
→ 選出 / 非選出
→ 実作16000字提出
→ 講評と採点
→ 累積得点
→ 最終課題40000字
→ ゲンロンSF新人賞選考
```

### D. 業界接続フロー

```text
講座 / 同人 / 投稿サイト / 読書会
→ 新人賞 / 雑誌掲載 / 電子出版 / 文フリ
→ 商業刊行 / 継続的コミュニティ参加
```

## 受講者向けプレイブック

### 講座で評価されやすい梗概の条件

1. 見たことのないSF的アイデアがある。
2. 感情、景色、質感、キャラクターなどのフェティッシュな魅力がある。
3. 短い字数でも起伏が見える。
4. 実作に展開したときの伸びしろがある。
5. 平易で、講師が短時間で把握できる。

### 講座で消耗しやすいポイント

1. 公開比較に過度に飲まれる。
2. 得点表だけで自己評価する。
3. 選出されない月に実作を書かない。
4. 勝ち筋だけを追い、固有の変さを捨てる。
5. 講座後の継続設計を持たない。

### 講座外で活きる周辺資産

## 圧縮の結論

この本の構造は、`定義`、`読書`、`語彙`、`歴史`、`制度`、`攻略`の六層でできている。
中心にあるのは、SFを「未知の道具の一覧」ではなく、「仮定から世界を変質させる手続き」として教えることだ。
その手続きを一人で身につけるのは難しいから、必読書、用語、ファンダム、新人賞、講座という外部装置が配置される。
本書は結局のところ、「どう書くか」だけでなく、「どこで学び、どこへ接続し、どう続けるか」の本である。
