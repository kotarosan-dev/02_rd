## 転機の連鎖（因果の骨）

```
兄の精神病関心 → 神経解剖学の訓練
    → 1996/12/10 左半球出血（未診AVM）
        → 自己観察しつつ救助（職場電話）
            → MGH 救命・診断
                → GG介護下で基礎から再学習
                    → 開頭・血腫除去
                        → 8年の完全回復と公演・執筆
                            → 「右脳の平和」＋神経科学の橋渡し
```
