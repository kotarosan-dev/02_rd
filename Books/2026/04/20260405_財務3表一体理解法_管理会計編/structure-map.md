## 財務3表との対応

### PLで見るもの

- 原価計算
- 売上総利益
- 貢献利益
- 固定費
- 差異分析
- 利益対策ロジックツリー

### BSで見るもの

- 製品在庫と仕掛品
- 不要資産
- 投資対象
- 借入金と資本
- 運転資金
- 企業価値の静態評価

### CSで見るもの

- 営業CF
- 投資CF
- 財務CF
- 売掛回収タイミング
- 在庫滞留
- 買掛支払タイミング

## 重要な因果連鎖

```text
原価の見誤り
-> 粗利の誤認
-> 販売判断の誤り
-> 売れれば売れるほど赤字
```

```text
予算不在
-> 差異分析不能
-> 軌道修正不能
-> 未来へ手が打てない
```

```text
利益だけを見る
-> 売掛・在庫の膨張を見落とす
-> 営業CF悪化
-> 黒字倒産リスク
```

```text
投資を利益で評価する
-> 長期案件の価値を誤る
-> DCFやIRRを使わない
-> 経営判断が短期化する
```

## 読み手別の使い方

### 現場マネジャー

- 第2章で粗利の見方を掴む。
- 第3章で予算と差異分析の運用方法を掴む。
- 第4章で売掛・在庫・買掛の改善余地を見る。

### 経営企画

- 第1章で全体像を言語化する。
- 第3章で予算策定フローを整備する。
- 第5章で投資評価の共通物差しを持つ。

### 経営者

- 第4章で利益と現金のズレを確認する。
- 第5章で投資判断を利益中心から脱する。
- 附章で再生時の優先順位を確認する。

## 一文要約の束

- 管理会計は、管理のための会計ではなく、存続のための会計である。
- 原価計算は、計算技法の森ではなく、利益の所在を明らかにする地図である。
- 予算は、前年の写しではなく、未来へ資源を動員する方向づけである。
- 差異分析は、過去を責めるためではなく、次の手を打つためにある。
- 利益は会計判断を含む数字であり、現金は事実である。
- 投資評価は、利益ではなくキャッシュフローで行う。
- 再生局面では、順番を誤ると努力が空転する。
