## 時代的背景と現代的意義

### 原著刊行時（1974年）の文脈
- ベトナム戦争終結直前（1975年）
- ウォーターゲート事件（1972-74）
- 第一次オイルショック（1973年）
- 公民権運動の成果と反動（1964年公民権法以降）
- 第二波フェミニズム隆盛期
- セサル・チャベスの農場労働者組合（UFW）活動期（1960s-70s）
- 製造業における組合労働の最盛期（以降、空洞化へ）

### 2020年代に読む意義
| 1970年代の問題 | 2020年代での継続・拡大 |
|--------------|---------------------|
| 自動化への不安（ベリル） | AI・ロボットによる職業消滅 |
| 感情労働の商品化 | ギグエコノミー・サービス業の感情管理 |
| 農場労働者の搾取 | 食料サプライチェーンの透明性問題 |
| 黒人労働者の差別 | BLM運動・職場の人種格差 |
| 「良い仕事」とは何か | 意味のある仕事への探求（MBB人気） |
| 宣伝の道徳的問題 | SNSマーケティング・インフルエンサー倫理 |
| 炭鉱の環境破壊 | 石炭廃業・正義ある移行（Just Transition） |
