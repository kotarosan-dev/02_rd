# 構造マップ：「恥」に操られる私たち

## 全体アーキテクチャ

```
全体テーゼ：「恥」は個人の感情ではなく、産業・制度・プラットフォームが組織的に利用する搾取システム（シェイム・マシン）である
       ↓
  ┌────────────────────────────────────────────────────────┐
  │  【理論的枠組み】         【実証：各領域の事例】     【変革の手段と限界】  │
  │  恥の三段階モデル    →    産業型・制度型・              弱者からの反撃      │
  │  シェイム・マシン分類      プラットフォーム型             個人レベルの実践    │
  └────────────────────────────────────────────────────────┘
```

## マクロ構造：章の配置ロジック

```
はじめに
│  └─ プエブロの道化（健全な恥の理想型）vs「飛び出た腕の肉」（破壊的な恥）
│      → 本書全体の問いを設定：恥はいつ機能し、いつ搾取の道具になるか
│
第1〜4章：個人の身体をめぐるシェイム・マシン（産業型）
│  ├─ 第1章：ダイエット産業（肥満への恥×リピート失敗ビジネスモデル）
│  ├─ 第2章：依存症産業（クラックvs.オピオイドの人種格差）
│  ├─ 第3章：貧困産業（「福祉女王」神話と申請障壁）
│  └─ 第4章：美容・フェミニン産業（女性の身体への恥の歴史的設計）
│
第5〜6章：社会空間でのシェイム・マシン（プラットフォーム型・集合型）
│  ├─ 第5章：ネットワーク化された恥（アルゴリズムと怒りの増幅）
│  └─ 第6章：ホワイト・シェイムと歴史修正主義（人種と集合的恥）
│
第7〜8章：社会の周縁にいる人々への恥（制度型）
│  ├─ 第7章：拒絶と否認（インセル・ひきこもり・極端主義）
│  └─ 第8章：公共の利益（パンデミック・ワクチン・「健全な恥」の条件）
│
第9章：反転——弱者が強者を叩く恥の可能性と限界
│  └─ #EndSARS, ガンディー, #MeToo, ゲブルvs.グーグル
│
第10章：著者の個人的実践（理論の体現）
│  └─ 肥満症治療手術の選択→保険承認の戦い→手術後の逆説
│
終章：何をすべきか
   └─ 社会処方箋（政策）＋個人処方箋（判断基準）
```

## 概念関係マップ

```
        恥の感情（普遍的な人間経験）
             ↓
    ┌────────┴─────────┐
    │                         │
  機能的な恥               病理的な恥
  （共同体の調整）           （搾取の道具）
    │                         │
  プエブロの道化           シェイム・マシン
    │                    ┌────┼────┐
    │                  産業型  制度型  プラットフォーム型
    │                    │       │        │
    │               ダイエット  生活保護  Facebook
    │               美容産業   刑事司法  Twitter
    │               依存症産業  移民管理  Google
    │
    └──→ 健全な恥の条件：
           1. 共通の価値観の共有
           2. 否定できない事実の記録
           3. 対象者に変化を起こせる力がある

         ↓ 成立する場合
    弱者が強者を叩く恥（Punching Up）
    → ガンディー・キング・#MeToo・#EndSARS・ゲブル
    
         ↓ 成立しない場合
    単なるいじめ or 逆効果（ハシド派コミュニティへの強制）
```

## 「恥の三段階」の全章への適用

| 対象 | 第一段階（傷） | 第二段階（否定） | 第三段階（受容） |
|------|-----------|------------|------------|
| 肥満者 | ダイエット失敗への自己批判 | 「自分は意志が弱くない」「産業が悪い」 | 科学に基づく手術の選択（著者） |
| 依存症者 | 社会からの断罪・逮捕 | 「本人の責任だ」という否定が蔓延 | 構造的原因を認め支援に転換 |
| 貧困者 | 申請プロセスでの屈辱 | 「努力が足りない」という自己責任論 | ベーシックインカムによる尊厳回復 |
| #MeToo被害者 | 沈黙を強いられた数十年 | 「自分が招いた」という自責 | 声を上げることへの移行 |
| 白人の多数派 | 人種的特権への指摘 | 防衛・怒り・「逆差別」の主張 | 具体的な変化のための行動 |
| インセル | 性的・社会的拒絶の傷 | 女性・社会への怒りと憎悪 | 本書では解決策が示されていない（藤里町モデルがヒント） |

### 弱者から強者を叩く運動の比較

| 運動 | 時代・場所 | ターゲット | 主な手段 | 成功度 | 特記事項 |
|------|---------|---------|---------|------|--------|
| ガンディーの塩の行進 | 1930年インド | 大英帝国 | 非暴力行進＋国際報道 | 高（SARS解散に相当する結果） | 達成まで17年かかった |
| #EndSARS | 2017〜2021年ナイジェリア | 腐敗警察SARS | SNS＋街頭デモ | 中（SARS解散）| 56名が死亡 |
| #MeToo | 2017年〜米国 | ハリウッド・各業界権力者 | SNS告発＋メディア | 高（200人超が辞職） | バックラッシュあり |
| パークランドの学生（Never Again MSD） | 2018年米国 | 銃擁護派政治家 | SNS＋ボイコット呼びかけ | 中（パブリックスが献金停止） | FOXニュース広告撤退 |
| ゲブルvs.グーグル | 2020〜2021年 | Google/Alphabet | 内部告発＋SNS公表 | 部分的（謝罪を引き出す） | マーガレット・ミッチェルも解雇 |

## 時代背景・参照文献の配置

本書が参照する主な事件・事例の時系列：

- **1930年**：ガンディーの塩の行進
- **1932年**：タスキギー実験開始（黒人男性への非倫理的梅毒実験）
- **1976年**：レーガンの「福祉女王」演説
- **1981年〜**：クラック・エピデミック（都市部黒人コミュニティ直撃）
- **1988年**：クレイマーのファウチへの公開書簡（エイズ活動）
- **1996年〜**：パーデュー・ファーマのオキシコンチン販売開始
- **2003年**：ストライザンド効果の発生
- **2014年**：エリオット・ロジャーによる大量殺人（インセル問題の顕在化）
- **2017年**：#EndSARS運動開始（ナイジェリア）
- **2017年**：#MeToo運動爆発（ワインスタイン暴露）
- **2018年**：パークランド高校銃乱射事件・Never Again MSD
- **2018年**：ゲブルのグーグル採用
- **2019年**：著者の肥満症治療手術
- **2020年**：COVID-19パンデミック・#EndSARS全国化・ゲブル論文問題
- **2021年**：ゲブル解雇・#MeToo発生から4年の振り返り
