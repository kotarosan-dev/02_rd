## 各章の論証フロー

### 第2章の論証フロー
```
前提: MSX2のスプライト制限 → 通常の軍事アクションは作れない
    ↓
逆転発想: 「撃つ」のではなく「見つからない」ことを目標に
    ↓
発明: ステルスゲームジャンルの誕生
    ↓
主張: ハードウェア制約 → 創造的革新（逆説的関係）
    ↓
含意: テーマ（暴力回避）とメカニクスが一体化したプログレッシブ設計の原点
```

### 第4章の論証フロー（MGS2）
```
前提: プレイヤーはMGS1の主人公スネークを期待している
    ↓
設計: 主人公を雷電に変更（意図的なミスディレクション）
    ↓
プレイヤー反応: 「裏切られた」という感情
    ↓
著者の論証: これは偶然のミスではなく意図的な設計
    ↓
テーマとの接続: 「情報社会でアイデンティティを持つことの困難」を
            ゲームプレイ自体がメタに体現
    ↓
結論: メカニクス（主人公の剥奪）＝物語のテーマ（主体性の喪失）
```

### 第5章の論証フロー（デスストランディング）
```
表面的批判: 「ウォーキングシミュレーター」（退屈、ゲームではない）
    ↓
著者の読み直し: 「配達」という行為がテーマを手続き的に論証している
    ↓
ソーシャル・ストランド・システム: 非同期の他者支援が「孤立の時代の連帯」を表現
    ↓
BBのメカニクス: 親子的紐帯の設計 → 失われた人間的つながりの回復
    ↓
文脈: 2016〜2019年のポピュリズム台頭・分断社会への応答
    ↓
結論: 「退屈」なゲームプレイが最もプログレッシブなメッセージを担う
```

## 本書の限界と反論のマップ

```
批判① ローカライズ・翻訳の問題
→ 著者は英語版研究が中心、日本語版固有の要素（声優演技、文化的文脈）は範囲外

批判② セクシュアリティ表現との矛盾
→ クワイエット（MGS5）等の性的描写はプログレッシブ主張と明らかに矛盾
→ 著者は認めるが解決策は示さない

批判③ 反戦と兵士美化の二重性（ムーア論文）
→ 小島は反戦を説きながら敵キャラクターを英雄視する
→ 日本の国粋主義的解釈に利用される可能性
→ 本書では「意図的な曖昧さ」として留保

批判④ 礼賛的傾向
→ 「プログレッシブ」という定義の弛緩が論証の一貫性を損なう
→ MGS4の過剰なカットシーンへの批判を相対化しすぎる
```
