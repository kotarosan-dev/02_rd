## 論理の流れ

```text
数字に出会う
  -> まず疑う
  -> 無作為標本を集める
  -> 標本をグラフで読む
  -> 標本数 / 形 / 位置 / 広がりを押さえる
  -> 隠れた変数を疑う
  -> 統計量とパラメータを区別する
  -> 標本平均の分布を想像する
  -> CLTで正規分布近似する
  -> 信頼区間を作る
  -> 仮説検定で帰無仮説と比べる
  -> p値で「偶然でも起こりうるか」を判断する
  -> 条件が崩れたら分布や手法をひねる
```

## 圧縮方針メモ

- 主判定は C。理由は、論証の厳密さそのものよりも、統計的思考の型を大衆向けに反復して教える設計が中心だから。
- ただし CLT、信頼区間、仮説検定、p値の説明は D 的な概念精度が必要なので、そこは簡略化しすぎず保持。
- 漫画の細かいギャグや比喩の枝葉は削り、誤謬回避と直感形成に効く場面だけ残した。
- 数式詳細は Appendix に委ね、本体では「どの概念が何のために必要か」を優先保持した。
