# 脳の外で考える（Extended Mind Thinking）

アニー・マーフィー・ポール『脳の外で考える――最新科学でわかった思考力を研ぎ澄ます技法』に基づく、認知拡張フレームワーク。

## 概要

脳だけに頼った「脳に縛られた思考」から脱却し、身体・環境・他者という「脳の外」のリソースを活用して思考を拡張する方法論。

## 使用場面（When to Use）

- 創造的なアイデアが出ないとき
- 学習効率を上げたいとき
- 複雑な問題を整理したいとき
- チームの知恵を引き出したいとき
- 集中力が続かないとき
- 意思決定に迷っているとき

## フレームワーク1：身体で思考する

### 原則

「私たちは小学校に入ると、じっと座り、静かに学び、一生懸命に考えるよう教えられます」——この「脳中心主義」が潜在能力を制限している。

### 技法1：内受容感覚の活用

体内から湧き上がる感覚（心臓の鼓動、胃の収縮など）は意思決定の重要な情報源。

```
実践方法:
1. 内受容感覚ジャーナルをつける
   - 重要な決断の前後で体の感覚を記録
   - 「胸がざわざわする」「胃が重い」など

2. ボディスキャン
   - 頭から足先まで順に注意を向ける
   - 各部位の感覚を言語化する

3. 感情にラベルをつける
   - 曖昧な身体感覚を感情語で表現
   - 「この感覚は不安だ」「期待感だ」
```

### 技法2：運動と思考の統合

歩きながら考えるとアイデアが湧くのは科学的に正しい。

```
実践方法:
1. 考え事は歩きながら
   - 終業後ではなく、仕事のスキマ時間に
   - 村上春樹の「空白のランニング」

2. スタンディングデスクの活用
   - 立っている方が認知機能向上
   - 座りっぱなしを避ける

3. 緊張をワクワクに変換
   - 「緊張している」→「ワクワクしている」
   - 身体状態の再解釈で成果向上
```

### 技法3：ジェスチャーの活用

「私たちは口ではなく、手で会話している」

```
実践方法:
1. 説明するとき手を使う
   - 抽象的な概念を空間に配置
   - 自分の理解も深まる

2. オンラインでも手を見せる
   - カメラに手の動きが映る位置で
   - 聞き手の理解が向上

3. 学習時のジェスチャー
   - 外国語学習で動作と結びつける
   - 記憶定着率が向上
```

## フレームワーク2：環境で思考する

### 技法4：自然環境の活用

自然の中で過ごすと、消耗した注意力が回復する（注意回復理論）。

```
実践方法:
1. 休憩は自然のある場所で
   - 公園、庭、窓際の植物でも効果あり
   - 短時間でも効果がある

2. 自然の画像でも一定の効果
   - デスクトップ壁紙を自然の風景に
   - フラクタル構造が人を落ち着かせる

3. 植物を置く
   - デスクに植物があるだけで集中力向上
```

### 技法5：空間のデザイン

「天井が高い場所にいると、人は思考が広がる」

```
空間の効果:
- 天井が高い → 創造的・抽象的思考
- 天井が低い → 細部への集中

実践方法:
1. 「自分の場所」をつくる
   - 自分の領域だと感じられる環境
   - パフォーマンス向上

2. プライバシーの確保
   - オープンオフィスでは会話が減る
   - 創造性にはプライバシーが必要

3. 目的に応じた場所選び
   - ブレスト → 開放的な空間
   - 集中作業 → 閉じた空間
```

### 技法6：アイデアの空間化

頭の中の考えを物理的に配置・操作することで新たな洞察が生まれる。

```
実践方法:
1. 大きなディスプレイを使う
   - 小さなディスプレイは知的能力を枯渇させる
   - パフォーマンスが向上

2. 付箋とホワイトボード
   - 考えを書き出して配置
   - 移動・並べ替えで関係性を発見

3. イラストに描き出す
   - 言葉だけでなくビジュアルで
   - 学びが深まる

4. 詳細な日誌をつける
   - ダーウィンの例
   - 記録が思考を外部化
```

## フレームワーク3：人と思考する

### 原則

「人間は社会的に思考しているときが一番深く考えている」

### 技法7：模倣から学ぶ

「ジョブズだって、最初は模倣からスタートした」

```
実践方法:
1. 専門家の動きを観察
   - 視線、手の動き、身体の使い方
   - 言葉だけでなく行動を模倣

2. 専門家の指導法
   - 手順を丁寧に分解して教える
   - 特徴を大げさに伝える
   - 初心者の理解が促進される

3. 素早い二番手戦略
   - ビジネスでは模倣者が成功することも
   - ZARAの例
```

### 技法8：仲間との協働

「仲間と議論することで知性は深まる」

```
実践方法:
1. 他者に教える
   - 教えることでより深く学ぶ
   - 「人に説明するつもりで」学習

2. 二人で問題に取り組む
   - 一人より優れた解決策が生まれる
   - 互いの盲点を補完

3. ティーンエイジャーへの指導
   - 社会的文脈を活用
   - 仲間との学びが効果的
```

### 技法9：集団の知恵を引き出す

適切に構成されたグループは、最も優秀な個人を上回る。

```
条件:
- 多様性（同質的すぎない）
- 心理的安全性（異論を歓迎）
- 発言機会の平等（一人が独占しない）

実践方法:
1. 会議では全員に発言機会を
2. 異論を明示的に歓迎
3. 多様な背景のメンバー構成
4. ロールプレイで立場を入れ替え
```

## 実践チェックリスト

### 日常の思考改善

- [ ] 重要な決断前に身体感覚をチェックしているか？
- [ ] 行き詰まったら歩いているか？
- [ ] 説明するとき手を使っているか？

### 環境の最適化

- [ ] 自然を取り入れているか？
- [ ] 作業に適した空間を選んでいるか？
- [ ] 考えを外部化（書き出し、可視化）しているか？

### 社会的思考

- [ ] 人に教えることで学んでいるか？
- [ ] 議論の場で全員が発言しているか？
- [ ] 専門家の行動を観察・模倣しているか？

## 注意点

- 「脳の外で考える」が常に優れているわけではない
- 深い内省も必要な場合がある
- 文化によってジェスチャーの適切さは異なる
- 職場環境を変えられない場合は、できる範囲から始める
- 引用される研究には再現性の問題があるものも含まれる
