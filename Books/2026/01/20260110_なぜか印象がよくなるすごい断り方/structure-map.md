# 構造マップ：『なぜか印象がよくなるすごい断り方』

## 章一覧と一行要約

| 章・部 | 役割 | 一行サマリー |
|--------|------|----------------|
| 電子書籍注記 | メタ | 縦書き・私的利用範囲の注意。 |
| イントロ | 導入 | 1980年代、近藤真彦へ楽曲提供依頼を矢沢永吉が電話で断った逸話が「印象のいい断り」の見本になる。 |
| はじめに | 問題提起 | 「断っても印象がいい」とは断られた後により信頼・再依頼したくなる状態；クレーム研修15年・ブックオフ監禁経験も含む著者の動機。 |
| 第1章 | マインドセット | 断れないのは性格ではなく癖；血を吐いて倒れた著者体験；マスターとの対立と和解；ゴットマン、20代は断らず蓄える時期もあり得る。 |
| 第2章 | 技法 | 3K（感謝→結論→感謝）、名前・3回感謝、理由は具体かつ自分側、早く短く、感情と事実の交互、声・表情；矢沢事例の分解。 |
| 第3章 | 応用 | 手ごわい相手は戦わず相談；正論・「常識的には」NG；6W3HとWhy；代替案・仮定質問；姑への野菜、交換条件、巻き込み、真っ白告白、横並び・パーソナルスペース。 |
| 第4章 | 判断 | 人生の優先順位・心地よさ・好き嫌い；人の評価は当てにならない；保留・3秒沈黙・「断れる人」の役；テレビ断ち10年など著者の実践。 |
| おわりに | 結び | 創業パートナー急死と「断れなかった」後悔が執筆動機；2020年1月・津田卓也。 |
| 著者プロフィール | 補足 | 株式会社キューブルーツ代表、クレーム研修、2019年時点受講約11万人・年間200登壇など。 |

## 転換点（論旨・物語の折れ目）

| 位置 | 転換の内容 |
|------|------------|
| はじめに→第1章 | エピソードから「断り＝信頼」の命题へ；読者の思い込みを外す宣言。 |
| 第1章中盤 | ブックオフでの過労→吐血入院→退職決意（命と断りの接続）。 |
| 第1章後半 | 日本料理店マスターへの「ノー」→翌日謝罪と役割変化（関係深化の実例）。 |
| 第1章終盤 | 「断らない時期」の肯定と、悩んでいるなら今が学ぶ時、のバランス。 |
| 第2章冒頭 | 野球の「基本フォーム」比喩で7ルールの位置づけ。 |
| 第2章末 | 矢沢エピソードを7ルールで再読し、直接電話の効果まで言及。 |
| 第3章冒頭 | 「変化球」比喩；ケース3-1/3-2で基本が通じない場面の提示。 |
| 第3章 | 「戦わない」から質問・仮定質問・交換条件・第三者巻き込みへ段階的エスカレーション。 |
| 第4章冒頭 | 「選球眼」比喩；断る/断らないの判断へシフト。 |
| おわりに | 同僚の死と後悔→本書の倫理的動機の開示。 |
