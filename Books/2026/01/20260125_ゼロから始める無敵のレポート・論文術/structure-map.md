# 構造マップ：『ゼロから始める無敵のレポート・論文術』

## 全体像

本書は大きく5段階で進みます。

1. **テーマを決める**: 第1章・第2章
2. **資料を集める**: 第3章
3. **論旨を外在化する**: 第4章・第5章・第6章
4. **構成と文章を整える**: 第7章・第8章
5. **論文として完成させる**: 第9章

第10章は補講で、通常のレポート・卒論の主線からは外れた文学論の応用編です。

## 核心命題マップ

| 論点 | 対応章 | 本書の答え |
|------|--------|------------|
| 何を書くか | 第1章・第2章 | 自分の興味があり、しかも小さいテーマ |
| 何で支えるか | 第3章 | 多様な資料と先行研究 |
| どう考えるか | 第4章 | 紙に出して考える |
| どう面白くするか | 第5章 | 笑いどころと突っ込みどころ |
| どう書き始めるか | 第6章 | 話す→設計図を作る→書く |
| どう組み立てるか | 第7章 | 序章・本論・結論を意識した構成 |
| 何で失点するか | 第8章 | 長段落、時系列混乱、重複 |
| どう完成させるか | 第9章 | 注・文献目録・推敲 |
