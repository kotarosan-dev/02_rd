# 構造マップ：『野中モモの「ZINE」 小さなわたしのメディアを作る』

> ジャンル判定: F（エッセイ・随筆集）  
> 構造原理: 個人的メディア史 + 作り手インタビュー + 場の設計論 + 小出版文化論

---

## 全体像

本書は一冊まるごとで、ZINEを「物」ではなく「回路」として描いている。

1. **はじめに**で、ZINEとは何か、その歴史とDIY精神を定義する。
2. **1章**で、著者自身の個人的メディア史をたどり、なぜ自主出版に励まされてきたかを示す。
3. **2章**で、複数のジンスターの実践を並べ、ZINEがどんな温度の表現になりうるかを可視化する。
4. **3章**で、冊子単体ではなく、集い・イベント・取扱店・海外シーンまで含む「場」の問題へ広げる。
5. **おわりに**で、小さな声の累積が社会の空気を変えるという思想に収束する。

つまり本書の構造は、**私が作る** → **仲間が作る** → **場ができる** → **社会に作用する**、という拡張そのものになっている。

## 因果連鎖

1. **大きなメディアでは拾われない経験や感情がある。**  
2. その経験を抱えた人は、壁新聞・同人誌・コピー誌・個人サイトのような小さい媒体に向かう。  
3. **出してみると反応が返る。**  
4. 反応は「受け手でいるだけではない」感覚を生み、さらに発信を続ける理由になる。  
5. 同じことをしている人たちが見えてくると、個人の表現は交換・委託・イベントへ広がる。  
6. その広がりを支えるには、棚・取扱店・集い・フェアのような**場の設計**が必要になる。  
7. 場ができると、小さな私的な声が、やがて社会や政治の感覚に接続される。

## 主要な共鳴関係

- 著者自身の「学級新聞」と、こやまさんの「にこにこ自分新聞」は、ZINE以前の衝動がすでに存在していたことを照らし合う。
- 佐藤麗生の「過去の自分と未来の友達に向けて作る」という姿勢は、著者が一章で語る「伝えてみないと何もはじまらない」と同じ線上にある。
- B.G.U.のバイリンガル・フリージンは、著者の個人的メディア史を、より明示的な社会運動へ接続した実例として機能する。
- ZINESTER NIGHT PARTYや各地の集いは、二章で見た「個の表現」が三章で「場の設計」へ拡張した姿である。
- 海外座談会は、日本のZINE文化を例外ではなく、世界のDIY出版の一変種として位置づけ直す。

## 何を削ってはいけないか

1. ZINEが「おしゃれな冊子」ではなく、DIY精神を受け継ぐ小さなメディアだという定義。
2. 著者の個人的メディア史が、紙とネットの対立ではなく往復運動として描かれていること。
3. 二章の作り手たちが、完成度よりも適温の表現を重視していること。
4. 三章で、ZINEが場づくりと不可分だと明示されること。
5. 最後に、個人的なことと社会的なことがつながるという着地。

## 捨てても骨格が残る部分

1. 個々の店名、イベント名、海外スタジオ名などの細部。
2. 著者の音楽・雑誌・映画の具体的嗜好の列挙。
3. 各インタビューの細かな技法談義。
4. イベント開催情報の詳細な一覧。

ただし、これらを削ると原著の魅力である「文化の現場感」は大きく薄まる。
