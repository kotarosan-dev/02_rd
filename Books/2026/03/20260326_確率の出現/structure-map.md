# 構造マップ：『確率の出現』

## 全体構造

本書は19章構成だが、論理的には次の3部に分かれる。

| 部 | 範囲 | 役割 | 主な到達点 |
|----|------|------|------------|
| I. 前史 | 第1章-第5章 | 確率がまだ存在しない時代に、何が前提として準備されたかを示す。 | `臆見` から `証拠`、`しるし` への転換。 |
| II. 出現 | 第6章-第13章 | 賭博・意思決定・法・統計・年金が接続され、確率が実践概念になる。 | 偶然の計算が判断技法と制度設計へ拡張される。 |
| III. 定式化 | 第14章-第19章 | 二元性が哲学的に定式化され、帰納問題へ接続される。 | 等可能性、帰納論理、大数法則、ヒューム。 |

## 論証チェーン

### チェーン1: 権威から証拠へ

1. `臆見` は、論証的知識に届かないが、権威に支えられた判断だった。  
2. `証拠` は、証言だけでなく事物の側に内在するものとして再編された。  
3. `しるし` の実践が、その再編を医学・自然学の中で支えた。  
4. これにより「もっともらしさ」は、人ではなく物に根拠づけられるようになった。  
5. その時点で、頻度や徴候を「確率的」と読む知的条件が整う。

### チェーン2: 偶然計算から意思決定へ

1. 賭博計算は分配問題や公平価格を扱った。  
2. パスカルの賭けは、その構造を宗教的選択に転用した。  
3. ポール・ロワイヤルは、奇蹟・歴史・危険判断を同じ形式で扱った。  
4. 確率はゲームの数学であると同時に、合理的選択の技法になった。

### チェーン3: 頻度から制度へ

1. 死亡表は都市人口の規則性を可視化した。  
2. 政治算術は、その規則性を国家判断の資源とした。  
3. 年金計算は、頻度を価格・財政・保険設計へ組み込んだ。  
4. 確率は観念ではなく、制度を動かす技術になる。

### チェーン4: 二元性から帰納問題へ

1. 確率は最初から信念の度合いと頻度傾向の双方を担った。  
2. 等可能性はこの二つをつなぐ便利な媒介になった。  
3. ベルヌーイの大数法則は頻度と未知確率の関係を理論化した。  
4. その一方で、原因の必然的認識は後退した。  
5. そこでヒュームの帰納懐疑が成立する。

## 先行研究との対話

| 相手 | 本書の立場 |
|------|------------|
| 数学中心の確率史 | 不十分。確率は数学だけでなく証拠・法・統計の交点で出現する。 |
| 連続的発展史観 | 留保。ハッキングは一七世紀半ばの断絶を強く主張する。 |
| 主観説 / 頻度説の単純対立 | 否定。両者は誕生時から絡み合っている。 |
| ヒュームを超歴史的問題として読む伝統 | 修正。帰納問題にも生成条件がある。 |
