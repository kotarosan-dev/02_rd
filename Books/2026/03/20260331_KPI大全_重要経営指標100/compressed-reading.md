## 読書ナビゲーション

本書は、1990年代以降に一般化した **KPI（Key Performance Indicators）** を、**「目標達成の度合いを確認する指標」** として再定義し、狭義の KGI（最終ゴールに近い売上・利益など）も広義の KPI に含めて扱う立場を取る。はじめにでは、レンタルDVD店の「1回当たり貸出日数」、大学の「企業からの学生評価」など、業界固有指標は無限にあるが、ここで示す **基礎100** を押さえれば自社向けに拡張できる、というメッセージが置かれる。

**PART1（第1〜2章）** は「KPIとは何か」「なぜ効くか」「何が歪むか」を、**PDCA・比較思考・バランススコアカード（BSC）** で整理し、続く **CASE1：A光学工業** と **CASE2：コクーン「学ボウズ」** で、営業プロセスの可視化と Web サブスクのファネル・CPA・LTV を物語として見せる。

**PART2（第3〜6章）** は **001〜100** を領域別に百科形式で解説する。**第3章** がマーケ・セールス＋Web（001〜035）、**第4章** がオペレーション・イノベーション（036〜054）、**第5章** が組織・人的資本（055〜072）、**第6章** が会計・ファイナンス（073〜100）。各節末の「関連KPI」と各章末「用語解説」が横リンクになっており、**辞書的参照**と**通読による経営学の地図**の両方を想定している。

## コアフレームワーク

### 全体フロー：測定 → 比較 → PDCA → 再設計

```
戦略・KGI/目標
      ↓
KPIの選定（誰が・何を・どの定義で）
      ↓
モニタリング（時系列・部署間・業界比較）
      ↓
Check / Action（介入・目標修正・定義の見直し）
      ↓
評価・報酬との接続（必要に応じ分離：問題発見用は細かく、評価用は絞る）
```

- **測定されないものは管理しにくい** が、100%精度より **速報性**（例：月次売上・利益は翌月5営業日以内に98%程度の速報、という実務感）を優先する場面がある。
- **PDCAの弱点**は多くの組織で **C/A が弱い**こと。日本電産の例では、週次目標を日次でリカッチアップし、遅れを翌日に持ち越さない運用が対比として示される。

### 比較の型（分析の基本）

| 比較軸 | 用途 |
|--------|------|
| 時系列 | 昨対・直近トレンド・季節性の除去 |
| 部署間 | 標準化された定義でのベンチマーク |
| 業界・他社 | 外部ベンチ（取得可能性に注意） |
| 対目標 | 予算・ローリング計画との差異分析 |

### KPIのブレークダウン：足し算・掛け算型とファネル型

**足し算・掛け算型**（売上の分解）の典型：

- 売上高 ≈ 延べ顧客数 × 客単価、あるいは ユニークユーザー × 平均来店回数 × バスケット など、業態に応じた積分解。
- 「塊」の売上だけ見るより、**どの因子が悪いか**を特定しやすい。

**ファネル型**（認知→試用→継続 等）：

- 例：認知率 → 使用経験率 → 過去1年使用率 の各ステップ間の **コンバージョン率** を別々に見る。
- 認知率と最終使用率が同じでも、**中間コンバージョンが低い箇所**がボトルネック（プロモ・配架・製品競争力など仮説が分岐）。

### バランススコアカード（BSC）の四視点

| 視点 | 例示KPI（書中図のイメージ） |
|------|-----------------------------|
| 財務 | ROIC、EBITDA、売上総利益成長率、コアカテゴリー売上高 |
| 顧客 | 新規顧客獲得コスト、ブランドイメージ、顧客満足度、再訪率 |
| 内部プロセス | 在庫額、1人当たり付加価値 |
| 学習と成長 | 従業員満足度、研修参加者数、昇格人数比率 |

BSCは **20個前後のKPI** で戦略実行を接続し、**戦略マップ**で因果を明示する思想が触れられる（詳細は原著）。

### インセンティブ設計と「バランス」

| 罠 | 例 | 対処の方向性 |
|----|-----|----------------|
| 単一指標の最適化 | サッカーで「パス成功率」だけ → リスクパスを避ける | 複数KPIの組み合わせ（得点関連・期待値） |
| 医療での選別 | 手術成功率のみ → 難症例を避ける | 難易度調整・多面的評価 |
| 営業の歪み | 訪問件数偏重 → 無駄訪問 | プロセスKPIと成果KPIのバランス、確度管理 |

**評価に直結するKPIは多くても20個以下**が普通、としつつ **キーエンス** は例外的に営業1人あたり3桁のKPIもある、と書かれる（再現性は組織能力次第）。

### 制約理論とオペレーション：スループット

**スループット**（TOC）：  
\[
\text{スループット} = \text{売上高} - \text{真の変動費}
\]  
（真の変動費＝原材料・輸送など純粋な変動費）

- 製品A：売価1000円、スループット500円、1日60個 → **日次 30,000円**  
- 製品B：売価2000円、スループット1200円、1日20個 → **日次 24,000円**  
→ 個別の限界利益率だけでなく **スピード×スループット** で優先順位が変わりうる、という数値例が本文にある。

## 章別エッセンス

### はじめに

**核心命題**: KPIは一時的なバズワードではなく、経営・現場の意思決定に浸透した **インフラ語彙** である。本書は100個の **意味・使い方・誰が見るべきか** を載せ、辞書的にも通読的にも使える。  
**事例サマリー**: 著者が1990年前後は「指標」「比率」呼称が主流だったと振り返る。**東洋経済新報社・斎藤宏軌** 氏への謝辞。グロービスでの **カルチャーサーベイ（2年に1度）** が、定性的テーマの時系列・部署比較の例として後の章と接続する。  
**行動指針**: 「基礎＝Essentials」として、KPIを通じて経営全体の洞察を深める。

### 第1章：KPIとは何か?

**核心命題**: KPIは **目標達成の進捗を測る重要指標** であり、財務だけでなく顧客満足・不良品率など非財務も含む。戦略・立場によって **重み付けは変わる**。狭義では KGI と区別されるが、本書は **広義の KPI** で統一して議論する。  
**事例サマリー**: **小学校**の学級崩壊率・教師残業・メンタル不調、**プロ野球**の OPS・WAR、**2020年コロナ** での感染者数・経済指標のトレードオフ議論など、営利以外への援用例。  
**行動指針**: **時系列・部署間・他社比較** を標準装備にする（詳細は『定量分析の教科書 ビジネス数学力養成講座』へ誘導）。

**KPIの効用（章内小見出し）**

1. **可視化と管理** — 属人化した評価（「5点満点＋曖昧コメント」）より、定義された指標が引き継ぎに効く。  
2. **PDCA** — Plan-Do はどこもやるが **Check-Action** が弱いとコンティンジェンシー頻発。**日本電産** の日次リカバリー文化が対比。  
3. **意識集約とモチベーション** — 戦略転換（例：粗利重視）には **評価・報酬とKPIの整合** が必要。走り高跳びのバー高さの比喩で、**努力で届く目標**の設定論。あえて高目標でイノベーションを促す手法も言及。  
4. **BSC** — 四視点と戦略マップの考え方（ロバート・キャプランら）。

**KPI経営の課題（章内小見出し）**

1. **バランス** — 単一指標の弊害（パス成功率、手術成功率、訪問件数など）。営業なら受注額に加え **価格維持率・既存売上・新規開拓・NPS・RFP数** など複合。  
2. **ブレークダウン** — 問題発見のためには **粒度の細かいKPI** を日常計測し、評価用とは切り分ける発想。  
3. **正確さと鮮度** — 政府統計のラグ注意。企業速報は **多少の誤差と引き換えに速さ**。  
4. **KPI至上主義の回避** — 理念・ビジョンは数字にならないことが多く、KPIは **手段**。新聞の部数偏重、テレビの視聴率偏重、減量しすぎなどの比喩。**内発的動機・対話** と併用しないと組織が痩せる。

### 第2章：KPIの実例ストーリー

**核心命題**: ケースは **「見える化・標準化」＋ファネル／コンピテンシー** が、競合（Bオプト）や新規事業（サブスク）でどう効くかを示す。

**CASE1：A光学工業（顕微鏡ニッチ、国内直販）**

- **課題**: 体育会系営業、**最終受注額のみ評価**、四半期達成後の **ターゲットビハインド**、Bオプト（ベンチャー）のコンサル型営業に劣勢。  
- **介入**: コンサルタント **佐藤幸太郎** が **営業フロー図** と各段階の **成果物・コンバージョン率** を月次で可視化。**RFP** 枯渇が未達要因（リードの質・DMU接近の仮説）。  
- **コンピテンシー標準化**: 高成績者から **市場把握力・論理思考・事前準備・課題洞察力・製品知識活用・コンサルコミュニケーション・大胆なクロージング** 等を抽出しスコアレンジ化。中途採用面談・育成に利用。  
- **数値結果**: 約1年半で **営業生産性+20%**（本文記載）。社長 **山田順一郎** のモノローグで締め。  
**行動指針**: プロセスKPIを評価に **20〜25%** 反映しつつ、「訪問乱発」を防ぐ議論が取締役会レベルで行われる描写。

**CASE2：コクーン「学ボウズ」（2017〜、年間9000円サブスク）**

- **投資判断**: **DCF・NPV プラス**でGO。App Store **テラ銭15%** 簡略化、継続平均3年など前提。  
- **2020年6月ミーティング**: ウェブ訪問→会員登録→有料→継続の **ファネル**。**UU・有償登録・継続率** は目標超えも、**次工程コンバージョン率の低下**が論点。**自然流入UU** 増に対しランディングページ改善提案。**CPA** は媒体別（A〜D）で管理、**全体CPA 約12,500円**、計画 **10,000円** に接近。**LTV** が想定より低いリスクも指摘。  
- **登場人物**: PL **下田優里**、上司 **北川裕**、マーケ **大沢武**、広告 **花岡次郎**、交流サイト **大川瑞希**。  
**行動指針**: 週次＋月次で **KPIを見ながら会話する** 文化が、精度と納得感を上げる、という北川の所感。

### 第3章：マーケティング・セールスのKPI（001〜035）

**核心命題**: 売上・シェア・認知から、店舗・営業生産性、リード〜成約、Web・SNSまで、**顧客接点のフルパイプライン**を数値化する。

**001〜010（マーケ基盤）**

| No | 指標 | 計算・定義の要点（原著ベース） |
|----|------|--------------------------------|
| 001 | 売上高 | 対価の合計。キャッシュは売掛回収まで。ブレークダウン例：新規/既存、チャネル別、延べ×単価。 |
| 002 | 市場シェア | 自社数量÷市場全体数量、または **自社売上÷市場売上**。ランチェスターで約 **4%超**で1位安定の議論。 |
| 003 | 顧客内シェア | **顧客内の自社売上÷当該カテゴリの顧客総購入額**（ウォレットシェア）。東大病院50%の設定例。 |
| 004〜007 | 認知率・使用経験率・市場カバレッジ・配架率 | ファネルとチャネル。**配架率**＝置いてある店数÷買われるべき店数。 |
| 008 | 既存顧客維持率 | **今年リピート顧客数÷昨年顧客数**。NPSや満足度と併用。 |
| 009 | NPS | 0〜10の推奨意向。**9〜10→+1、7〜8→0、6以下→−1** に変換し平均。**+が合格、0.3以上は優秀** などの目安。国民性で **日本人は8点** になりがちで読み替え検討。操作（営業が顧客に依頼）への警告。 |
| 010 | 客単価 | バスケット戦略とセット。 |

**011〜024（セールス・リード）**

- **011 坪当たり売上高** — 小売の出店・業態比較。**成城石井** 等ベンチマークの話。狭すぎる売場の長期不満にも注意。  
- **012 営業1人当たり売上高** — **売上高÷営業担当者数**。粗利率・人件費との整合例（本文の計算式イメージ）。  
- **013 新規顧客数** — 定義の線引き（子会社化、研究室単位等）。**MQL/SQL/RFP** からの逆算。  
- **014 問合せ数** — チャネル別。電話番号非表示で機会損失、等。  
- **015〜017 MQL / SQL / RFP** — マーケから営業への **受け渡し品質** が全体効率を決める。RFPは **当て馬** に注意。  
- **018 成約率** — 分母（SQLかRFPか）の定義必須。難易度で歪む。  
- **019 受注期間** — **キーエンス** が短い有名例（DMU直撃等）。  
- **020 コンピテンシー** — ヘイグループ起源。**項目バランス**が重要。  
- **021 価格維持率（値引き率）** — ディスカウント圧力の監視。  
- **022 CPA** — **広告費÷獲得数**（定義は登録か有料かで要統一）。  
- **023 継続率 / 024 流入数** — ケース（コクーン）と接続。

**025〜035（Webマーケ）**

- **平均セッション時間** — GAでは離脱ページの時間が載らない点の注意。**5〜10分** が企業サイトの目安例。  
- **直帰率** — ランディングの魅力度。**真の離脱**と**読了後離脱**の切り分けツール。  
- **CTR** — バナー **~0.2%**、リスティング **2〜3%** 程度の相場感。  
- **CVR** — ランディング・フォーム・商材力の合成。指名ワード/一般ワード分割。  
- **UU** — **UU ≤ セッション ≤ PV** の関係。マルチデバイスで過大カウント。  
- **DL数・有償転換率** — フリーミアムで **約5%有償** で成功例、という俗説。**DAZN** 無料期間、**DeNA**・**LINEスタンプ**、**Amazonプライム** の先行投資。  
- **翌日再訪問率** — 習慣化。**初日10分未満・投稿なし** は翌日来ない仮説。  
- **送客数** — **グロービス「知見録」→学び放題／大学院** の社内例。  
- **フォロワー数・シェア数** — **前澤友作** 約700万フォロワー、**スタバ・ローソン** 公式規模。**エンゲージメント率** 0.5〜2% 例。炎上シェアは別カウント推奨。

### 第4章：オペレーション・イノベーションのKPI（036〜054）

**核心命題**: 現場の **QCD**、ボトルネック、イノベーション投入と損失の管理。

**036 スループット** — 上記式。**ボトルネック工程**が全体スループットを規定。  
**037 稼働率** — **実稼働時間÷フル稼働時間**。**パーク24** の「ちょうど良い稼働率」。100%固定は顧客体験を損ねうる。コロナで **ホテル・航空** が直撃された文脈。  
**038 納期順守率** — **期限内件数÷全件**。**楽観バイアス**、**デスマーチ**・過労との両立問題。  
**039 提供スピード** — **日本電産** の試作速度が商談を前に進める例。  
**040 不良品率** — **不良品数÷全生産数**。  
**041〜045** — カイゼン提案、トラブル、ヒヤリハット、予算超過率、ロボット化率。  
**046〜054** — 商品ロス、物流費率、研究開発費率、特許・新製品・開発案件、社外シーズ比率、開発期間、新製品売上比率 など、**イノベーションの入力と出力**のバランス。

### 第5章：組織のKPI（055〜072）

**核心命題**: 生産性、コスト構造、多様性、ガバナンス、**SDGs** までを「組織」KPIで捉える。

- **055〜057** — **従業員1人当たり売上高・人件費・労働分配率** で付加価値と分配の整合。  
- **058〜064** — 年齢構成、離職率、本社費率、正社員比率、中途採用、**階層数**（意思決定スピード）。  
- **065〜072** — 満足度、カルチャーサーベイ、人材開発投資/人、女性比率、有給消化、内定辞退率、社外取締役比率、SDGs貢献。

### 第6章：会計・ファイナンスのKPI（073〜100）

**核心命題**: P/L・B/S・CF・資本コスト・企業価値を **同じ物語**で読む。

**収益性（073〜077）** — 売上総利益率、営業利益率、経常利益率、税引後利益率、包括利益。  
**078 ROA** — **経常利益÷総資産** 等、**分子が経常か純利益かで要注意**。**カカクコム** 2019 東証1部で **5.8%** など実例。現金過多でROAが低く見える、簿価土地で逆、の注意。  
**079 ROE** — **純利益÷自己資本**。**分解**: (純利益÷総資産)×(総資産÷自己資本)。レバレッジで上げられるため **ROAとセット**。**日本の上場企業ROE 10%超** を合格感として言及。  
**080 自己資本比率** — **高すぎ＝エクイティコスト負担**、**低すぎ＝倒産リスク** のトレードオフ。**ソフトバンクG** の高負債例。  
**081 当座比率** — **（売掛＋現金等に近い流動）÷（買掛＋短期借入）** のイメージ（酸性テスト）。**90〜100%以上** 目安、**70%以下** は不安、など。  
**082 在庫回転期間** — **棚卸資産÷(売上原価/365)**。回転率＝売上原価÷棚卸資産。  
**083 CCC** — **棚卸資産回転期間 ＋ 売上債権回転期間 − 支払債務回転期間**（原著の式）。**Collect early, Pay late**。**Amazonマーケットプレイス** は **CCCマイナス** の例。  
**084〜086** — 固定費、**限界利益率＝(売上−変動費)÷売上**、**損益分岐点売上高＝固定費÷限界利益率**。**目標利益Z** のとき **目標売上＝(固定費+Z)÷限界利益率**。  
**087〜088** — 製造原価、**EBITDA**（減価償却前営業利益ベースのキャッシュプロキシ）。  
**089 FCF** — ケースでは **NPV** とセットでDCF。**089〜093** — WACC、NPV、**ROIC**、**EVA**（投下資本コスト超過分）。  
**094〜100** — 実効税率、EPS系、成長率、配当、海外比率、現金、**バーンレート** など、投資家・資金繰り視点で締める。

## よくある落とし穴

1. **単一指標の最適化** — 現場がゲームする（パス成功率、訪問件数のみ等）。  
2. **定義のすり替え** — 四半期ごとに「新規」の定義が変わりトレンドが壊れる。  
3. **速報と確定の混同** — 意思決定が古い確定値のみに基づく。  
4. **GAの直帰率・滞在時間の誤読** — 離脱ページの時間が0に見える。  
5. **UUの過大** — マルチデバイスで同一人物が複数UU。  
6. **CPAとLTVの切り離し** — 獲得単価だけ下げて質を落とす。  
7. **在庫圧縮の押し売り** — 顧客満足を損ねる。  
8. **納期100%信仰** — バッファ過多または過労のどちらかを生む。  
9. **ROE単独崇拝** — レバレッジで見かけだけ改善。  
10. **CCC改善の取引先不信** — 支払遅延が関係性を壊す。  
11. **KPIと理念の逆転** — 数字が目的化し、中期のブランドを毀損。  
12. **評価だけ数字** — 内発的動機と対話が枯渇する。
