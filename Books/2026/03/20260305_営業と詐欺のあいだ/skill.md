# 営業vs詐欺 見分けフレームワーク

## 概要

バイヤーとして数千人の営業マンと渡り合った著者が整理した「お金を奪う側の心理テクニック」一覧。同じ手法が営業にも詐欺にも使われるという逆説を理解し、「意識的に選ぶ力」を養う実践フレームワーク。

## 使用場面（When to Use）

- 商品・サービスの購買判断をするとき
- 営業担当として相手との交渉設計をするとき
- 組織内の説得工作・情報操作を疑うとき
- 投資・不動産・保険などの高額商品説明を受けるとき

## 6つの影響力の武器（チャルディーニ準拠）

営業・詐欺共通で使われる「人を動かす心理原則」：

| 原則 | 営業での使い方 | 詐欺・悪用での使い方 |
|------|--------------|-------------------|
| **返報性** | 無料サンプル・試供品提供 | お試し商品→強制購入義務 |
| **希少性** | 「残り3点限り」 | 架空の限定・期限 |
| **権威** | 専門家推薦・実績提示 | 偽の資格・肩書き |
| **社会的証明** | 顧客レビュー・実績数 | サクラレビュー・サクラ客 |
| **好意** | 共通点の強調・褒め言葉 | ミラーリング・過度な親密化 |
| **一貫性** | 小さなYES→大きなYES | 足の踏み入れ作戦 |

## 詐欺手口の類型

### コールドリーディング
- バーナム効果（誰にでも当てはまる言葉を個人化）
- 反応を観察して情報収集し「言い当て」を演出
- 占い・霊媒・悪質コンサルで多用

### モニター商法
- 「無料モニター」で商品送付 → 解約困難な割賦契約
- レポート料が割賦金を上回るという虚偽説明

### マルチレベルマーケティング（MLM）
- 友人・知人を勧誘するビジネス構造
- 「ダウンライン」が拡大しなければ収益ゼロ
- 「仮想の不労所得」で感情を操作

### カルト洗脳プロセス
1. 孤独・不安な状態への接触
2. 段階的な親密化・コミュニティ形成
3. 思考停止と服従の強化
4. 金銭・人間関係の依存化

## 防衛フレームワーク（買い手5原則）

```
1. 何を買いたいかを事前に明確にする
   → 「今ほしいもの」ではなく「必要なもの」を先に定義

2. セールスマンと商品を分離して評価する
   → 好感度・プレッシャー・関係性を判断から外す

3. 支払える金額・条件を事前に設定し曲げない
   → 交渉に入る前に「自分のルール」を持つ

4. 即決圧力には「24時間考える」と言う
   → 希少性・緊急性の演出への最良の対抗策

5. 「なぜ今この人が私にこれを売ろうとしているのか」を問う
   → 相手のインセンティブを逆読みする習慣
```

## 危険信号チェックリスト

以下が複数重なった場合は要注意：
- [ ] 「今だけ・今日だけ」の期限プレッシャー
- [ ] 「あなただけに特別に」という選別感
- [ ] 「絶対に」「必ず」という確定的表現
- [ ] 無料プレゼント→後で義務が発生
- [ ] 紹介者が知人・信頼できる人である
- [ ] 断ると感情的になる・執拗に再アプローチ
- [ ] 詳細な契約書・書面の確認を急かされる

## 営業（仕事）視点での応用

- 上記テクニックを理解した上で「相手の価値になるもの」を軸に設計する
- 「社会をより良くしたい意思」を根拠にした信頼構築が長期的に有効
- 顧客の意思決定プロセスを操作するより、情報提供で支援する姿勢が結果的に成約率を高める
