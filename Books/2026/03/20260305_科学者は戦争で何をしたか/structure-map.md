# 構造マップ：『科学者は戦争で何をしたか』益川敏英

| 部・章 | 一行テーゼ | 重要度 |
|--------|------------|--------|
| はじめに（記念講演・空襲体験） | ノーベル記念講演で戦争を語ることは、諸刃の科学の自覚と「語れる最後の世代」の責務である。 | ★★★ |
| はじめに（坂田・特定秘密） | 科学者は「人間として人類を愛する」以前に研究に没頭しがちであり、秘密法制と軍拡は科学のオープン性を脅かす。 | ★★★ |
| 第1章 諸刃の科学 | ノーベル級の発見は福祉と大量殺戮の両刃であり、キュリー的警告とハーバー的愛国開発が歴史を分岐させた。 | ★★★ |
| 第2章 戦時中の科学者 | 戦争下では「科学に国境はない」は停止し、マンハッタン計画は動員・排除・贖罪の連鎖を生んだ。 | ★★★ |
| 第2章（戦後〜ジェーソン） | ラッセル＝アインシュタイン宣言・パグウォッシュは為政者に届きにくく、ジェーソンはマンハッタンと同型の秘密動員だった。 | ★★★ |
| 第3章 選択と集中 | 科学の巨大化・分業で研究者も全体像が見えず、短期成果主義と「選択と集中」が基礎研究と倫理を圧迫する。 | ★★★ |
| 第3章（STAP・オープン） | STAP問題は予算政治の帰結でもあり、成果のオープン化と報奨の設計が利権化を抑える。 | ★★ |
| 第4章 軍事研究の現在 | 日本学術会議の非戦誓いは薄れ、防衛資金の「スカウト」型軍学協同が間接動員となりうる。 | ★★★ |
| 第4章（デュアルユース等） | デュアルユースは原理的に全面禁止と両立しにくく、東大方針論争と名古屋平和憲章が対照的緊張を示す。 | ★★★ |
| 第4章（朝永の知恵） | 朝永振一郎の「形式協力・核心不出し」は、不可能条項だけでは足りない現場の抵抗の型を示す。 | ★★ |
| 第5章 政治と歯止め | 2015年安保法制・集団的自衛権解釈は九条の歯止めを侵食し、軍備増（オスプレイ等）と「使えない」平和の知恵が対比される。 | ★★★ |
| 第5章（九条科学者の会） | 「九条科学者の会」発起と労組・デモ主体の衰退が、批判的政治の地形を説明する。 | ★★ |
| 第6章 原子力の縮図 | 原発は政官産の安全軽視の縮図であり、坂田の審査批判が311で現実化した一方、廃炉・廃棄物には研究継続が必要という二面性。 | ★★★ |
| 第6章（核脅威） | 大国の核は「学習」で硬直化しつつ、途上国・独裁下の核が益川の最大懸念となる。 | ★★ |
| 第7章 戦争をなくすには | 200年スケールの進歩視座と「不満を埋める装置」、政治の過剰が起爆剤になるという民主主義の自己修復仮説。 | ★★★ |
| あとがき | 「科学者の戦争責任」押しつけを嫌い、生活者としての科学者育成と市民接点が本線。 | ★★★ |

## 論証の主軸（俯瞰）

```
戦争体験・記念講演の正当化
    → 諸刃の科学（ノーベル系譜）
    → 動員の歴史（WW1–2–冷戦–ベトナム）
    → 現代（巨大科学・資金・デュアルユース・憲法）
    → 原子力＝縮図（安全・核拡散）
    → 長期楽観＋今やるべき市民結合
```
