# Structure Map：『デジタルマーケティング戦略』

## 全体像

- 構成: `序章 + 第I部5章 + 第II部7章 + 終章`
- 総セクション数: `14`
- 全体主張: `デジタル社会で変わった消費者と関係性を起点に、4P・組織・空間・倫理まで再設計せよ`
- 読み方: `土台理解 -> 基盤整備 -> 4P再設計 -> 組織/空間 -> 倫理`

---

## パート別要約

### 序章

- 役割: 日本企業のデジタル遅れと、デジタル社会の到来を示して問題設定を行う
- 重要度: `高`
- 一言でいうと: `なぜ今、マーケティング全体の再設計が必要なのか`

### 第I部 デジタル・コンシューマーと顧客対応

- 役割: デジタル社会・消費者・関係性・データ基盤・CRM の基礎を固める
- 重要度: `最重要`
- 一言でいうと: `土台を理解しないと第II部の施策論が活きない`

### 第II部 デジタルマーケティング

- 役割: ブランド、製品、コミュニケーション、チャネル、価格、組織、OMO/メタバースへ展開する
- 重要度: `最重要`
- 一言でいうと: `デジタル関係性を4Pと組織に翻訳するパート`

### 終章

- 役割: 倫理・自己決定権・アルゴリズム問題を入れて全体を締める
- 重要度: `高`
- 一言でいうと: `できることと、やってよいことは別`

---

## 章別マップ

### 序章 デジタルマーケティングのために

- 主要テーマ: デジタル社会、スマホ、情報通信、市場環境、日本の遅れ
- コア命題: `デジタル化は一部業務のIT化ではなく、企業と消費者の関係性の再編である`
- 重要度: `高`
- 戻る優先度: `A`
- 読み返すべき理由: 問題設定と全体の危機感がここに集約される

### 第1章 デジタルマーケティングとは何か

- 主要テーマ: デジタル社会、デジタル・コンシューマー、デジタル関係性、デジタルマーケティング定義
- コア命題: `測定可能・リアルタイム・常時双方向・常時接続が全章の共通基盤`
- 重要度: `最重要`
- 戻る優先度: `S`
- 読み返すべき理由: 本書全体の用語定義と論理の起点

### 第2章 デジタル・コンシューマーと消費者行動

- 主要テーマ: 高知識消費者、ユビキタス消費者、クリエイティブ消費者
- コア命題: `消費者は情報弱者ではなく、比較・発信・共創する主体へ変化した`
- 重要度: `高`
- 戻る優先度: `A`
- 読み返すべき理由: 顧客理解の前提を更新する章

### 第3章 コネクテッド・コンシューマーと企業-顧客関係

- 主要テーマ: ヒト・モノ・企業・地域・社会との接続、IoT、SNS、Web3、スマートシティ
- コア命題: `購買瞬間ではなく、つながっている時間全体がマーケティング対象になる`
- 重要度: `中高`
- 戻る優先度: `B`
- 読み返すべき理由: 接点設計を広く捉えるための章

### 第4章 デジタルマーケティングのデータ活用プラットフォーム

- 主要テーマ: オンプレ/クラウド、API、Cookie、DMP、CDP、BI、MA
- コア命題: `収集・統合・分析・実行の循環を作らないと、データ活用は施策へ落ちない`
- 重要度: `最重要`
- 戻る優先度: `S`
- 読み返すべき理由: 実務で最も詰まりやすい基盤論

### 第5章 顧客関係管理

- 主要テーマ: CRM、ロイヤルティ、エンゲージメント、カスタマーサクセス、ジャーニー
- コア命題: `CRM は顧客データの保管ではなく、関係性を維持・深化させる運用思想`
- 重要度: `最重要`
- 戻る優先度: `S`
- 読み返すべき理由: データ基盤と顧客価値をつなぐ中心章

### 第6章 デジタル時代のブランド戦略

- 主要テーマ: データドリブン・ブランディング、カスタマー・コンピタンス、進化ブランディング
- コア命題: `ブランドは企業が固定的に定義するものから、顧客と共に更新するものへ変わる`
- 重要度: `高`
- 戻る優先度: `A`
- 読み返すべき理由: 顧客参加とデータの統合をブランド論で示す

### 第7章 製品戦略のデジタル化

- 主要テーマ: 情報財、サブスクリプション、PSS、UX、IoT
- コア命題: `製品はモノではなく、継続的に経験価値を届けるインターフェイスになる`
- 重要度: `高`
- 戻る優先度: `A`
- 読み返すべき理由: 売切り発想から体験発想への転換がまとまっている

### 第8章 デジタルコミュニケーション戦略

- 主要テーマ: ペイド/オウンド/アーンド/シェアード、コンテンツ、KPI、評価
- コア命題: `媒体別最適ではなく、顧客ジャーニー別にメディアを統合せよ`
- 重要度: `高`
- 戻る優先度: `A`
- 読み返すべき理由: オウンドメディア中心の整理が実務転用しやすい

### 第9章 流通・チャネルのデジタル化

- 主要テーマ: マルチ/クロス/オムニチャネル、ユニファイド・コマース、物流、ID統合
- コア命題: `オムニチャネルは販促施策ではなく、在庫・物流・データを束ねる経営課題`
- 重要度: `最重要`
- 戻る優先度: `S`
- 読み返すべき理由: EC と店舗の統合を表面的理解で終わらせない

### 第10章 プライシングとペイメント

- 主要テーマ: ダイナミック・プライシング、フリーミアム、サブスク、キャッシュレス、決済データ
- コア命題: `価格は状況応答的に、決済は関係性接点として設計する`
- 重要度: `高`
- 戻る優先度: `A`
- 読み返すべき理由: 価格と決済を UX/データ観点でまとめている

### 第11章 顧客中心のデジタルマーケティング組織

- 主要テーマ: 顧客志向構造、流動組織、ダイナミック・ケイパビリティ、ネットワーク型チーム
- コア命題: `デジタル時代の競争力は、固定的に強い組織より変化し続けられる組織に宿る`
- 重要度: `最重要`
- 戻る優先度: `S`
- 読み返すべき理由: 実行失敗の原因を組織側から捉え直せる

### 第12章 OMO とメタバース・マーケティング

- 主要テーマ: O2O、OMO、ミラー空間、ファンタジー空間、メタバース
- コア命題: `顧客体験はリアルだけでなく、ミラー空間と仮想空間にまたがって設計される`
- 重要度: `中高`
- 戻る優先度: `B`
- 読み返すべき理由: 将来拡張の方向を把握するのに有効

### 終章 デジタルマーケティングと倫理

- 主要テーマ: フィルター・バブル、プライバシー、自己決定権、アルゴリズム・バイアス
- コア命題: `関係性が強くなるほど、支援と操作の境界を問う必要がある`
- 重要度: `高`
- 戻る優先度: `A`
- 読み返すべき理由: 全体を倫理で締めて、実務の暴走を防ぐ

## 因果連鎖

1. `デジタル社会の進展`
2. `消費者の高知識化・ユビキタス化・クリエイティブ化`
3. `企業-顧客関係の常時接続化`
4. `データ基盤とCRMの重要性上昇`
5. `4P の再設計`
6. `組織・空間・決済まで含めた全社最適化`
7. `倫理と自己決定権の再設定`

この順番を崩すと、本書は単なるデジタル施策の寄せ集めに見えてしまう。著者の本当の構図は、`基盤 -> 戦略 -> 組織/空間 -> 規範` である。
