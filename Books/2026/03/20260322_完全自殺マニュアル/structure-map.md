# 構造マップ：『ぼくたちの「完全自殺マニュアル」』

奥付・目次・本文見出しに基づくインデックス。重要度は圧縮読本制作時の「全体理解への寄与」による主観付け（★1＝補助、★3＝核心）。

| ブロック | 一行サマリー | 重要度 |
|----------|----------------|--------|
| はじめに | 編者が本書の目的（手紙の収集、前著との関係、問題意識）と「何を考えるべきか」を宣言する | ★★★ |
| 第1章 騒動 | 発売〜取次・広告規制〜週刊誌・海外報道・遺体発見報道・ワイドショー・申立書など時系列ドキュメント＋編者ルポ・事例コメント | ★★★ |
| 第2章 読者手紙 | 536通からの選別掲載；年齢帯別；山田花子の父の書簡ほか、生存・共感・告白の複声 | ★★★ |
| 第2章末「分析データ」 | 手紙の量的・分類的まとめ（本文参照） | ★★ |
| 第3章 語られたか | 複数メディア・著者の文章・対談（鶴見vs柳美里等）による作品の意味づけの競合 | ★★★ |
| おわりに | メディア反論の放棄；本書＝媒体；読者手紙優位の宣言 | ★★★ |
| 奥付・書誌 | 1994年初版、太田出版、ISBN表記、引用禁じますの注意 | ★ |

## 第1章内の機能分担（読みの補助）

| 機能 | 内容の型 |
|------|-----------|
| 年表 | 日付＋出来事＋出典メディア |
| 制度・業界 | 取次、広告、販売規制のコメント引用 |
| 編者介入 | ケース紹介と短い「コメント」欄 |
| 抗議文書 | 発刊中止申立てなどの公文書調テキスト |

## 第3章内のメディアタイプ（目次ベース）

- 音楽／カルチャー誌（例：CROSSBEAT、DIME）  
- 漫画・男性誌系コラム（クレア、ポパイ）  
- 週刊誌（文春、現代、朝日、SPA!）  
- 批評・文芸誌系の長文  
- 対談（鶴見済 vs 柳美里）
